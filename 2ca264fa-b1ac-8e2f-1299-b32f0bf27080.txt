　　第425章
<br/>　　樊梨花一袭红袍，座下胭脂马跑到程大雷身边时仍打着响鼻。樊梨花两腮见红，鼻翼轻轻起伏。
<br/>　　程大雷这才响起，这姑娘也是战场一员悍将。本是拿刀的手，最近逼得她在蛤蟆城随苏樱一起绣花，当真有些委屈她了。
<br/>　　“走啦，走啦！”
<br/>　　程大雷喝了两声，座下墨玉照月兽发动起来，在荒原上荡起一阵黑色烟尘。
<br/>　　同时，蛤蟆城五路大军也以极高的行军速度，扑向凉州牵牛谷。牵牛谷距离蛤蟆城大概有三百里，就算以骑兵的速度，也至少需要一天一夜。而步兵的速度更慢，至少需要三天。
<br/>　　而且，赶到牵牛谷时必须保持能够战斗的体力。真如果扑过去时累得气喘吁吁，筋疲力尽，直接相当于给人家送人头，去了等于没去。
<br/>　　在五路大军中，最快的自然是赵子龙的轻骑兵部队，但即便赵子龙，也并没有因好战而冲昏头脑，也保持着三十里一歇，等恢复体力后，再进行下一次奔袭。
<br/>　　草原上的戎族，常常一人携带两匹马，一匹骑乘，一匹驮着干粮兵器。等一匹马力竭时，再换乘另外一匹马，如此保持行军速度。所以，戎族能够百战百胜，并不是没有道理。
<br/>　　以现在蛤蟆城的财力，还没能力给三千人每人准备两匹马。
<br/>　　赵子龙的轻骑兵队伍赶到牵牛关至少要到明天黄昏，这一天一夜，就是高飞虎需要捱过的，所以情况并不是很乐观。
<br/>　　“快些，快些，再快些！”张肥骑着马奔驰在队伍中，手中的皮鞭时不时落在某人头顶：“再不快些，到时候你们连骨头渣都吃不到。”
<br/>　　在秦蛮、关鱼、高飞豹的队伍中，这一幕时不时发生。对于这些行为，程大雷也没有办法。
<br/>　　军，无威不立。在这个时代，与士卒好好说话，谈人生谈理想，几乎没多大用处。必须用严酷的军法控制，所以，军中体罚士卒的事情时不时发生。
<br/>　　不过，程大雷手下兵混子老油条不能说完全没有，但极其少见。单从行军速度看，一忍三年，程大雷的确练了一支精兵。
<br/>　　而程大雷所料得不错，这段时间的高飞虎的确很难熬。
<br/>　　他们从江南千里过来，路上吃了太多苦头。人困马乏，在遭遇凉州兵时打了一仗，损兵折将严重。现在他们躲进了牵牛谷，士气低迷，连高飞虎本人都受了箭伤。
<br/>　　高飞虎在江南拉拢起一支人马，这三年日子过得不错。江南是个好地方，但好地方容易消磨人的志气，这三年他没打什么仗，统兵打仗的能力比日日接受训练的高飞虎都不如。
<br/>　　他们已在牵牛谷困了三天，已快要陷入弹尽粮绝的绝地。
<br/>　　夜里，还要提防凉州兵的攻击，高飞虎根本没有时间休息，在大帐内，与黄三元以及手下几名首领议事。
<br/>　　二首领齐少建，原先是绿林道上的悍匪，杀人如麻，在江湖上广有恶名。
<br/>　　三首领郭不戳，是一个水贼，为人生得瘦小，眼睛却像黄鼠狼一样瘆人。
<br/>　　……
<br/>　　战败之后，大家心情都不是很好，大帐内压抑着一种沉重的气氛。黄三元算见过大场面的人，眼前的胜负对他而言其实不算什么。但其他首领，却有些沉不住气了。
<br/>　　高飞虎哈哈笑了一声，道：“诸位稍安勿躁，徐松已经突围去大当家那里求援，算时间，这一两日就该到了。等大当家到了，眼前的问题也就不算什么问题了。”
<br/>　　对于程大雷的名字，齐少建和郭不戳等人都不算陌生。因为高飞虎经常说起，说程大雷如何如何了得。
<br/>　　不过，名声这种事，听得多了容易产生逆反心理。况且，这次是刚刚兵败，对高飞虎的不信任，转而就变成对程大雷的不信任。
<br/>　　“老大，程当家来了，真的能解眼前之围么？”齐少建忍不住问：“围着咱们的可是凉州兵。”
<br/>　　帝国的军队，总的来说，是从西到东，从北到南越来越弱。而西北凉州兵，向来有虎狼之师的称号。当然，他们也是被戎族打得最惨的一支军队。
<br/>　　“凉州兵又如何？”高飞虎道：“我来问你，是凉州兵强，还是戎族的骑兵强？”
<br/>　　“这个自然是戎族……”
<br/>　　“可我们大当家连戎族都打得跑，难道会畏惧区区几个凉州兵。”高飞虎不屑道。
<br/>　　这件时，高飞虎经常说起，大家耳朵都快磨出茧子。高飞虎是十分尊重程大雷的，甚至有几分崇拜，不然，怎么会一出事，就想着来凉州投奔他。
<br/>　　可大家听得实在太多了，多到已经有些反感。
<br/>　　“老大，靠人人走，靠墙墙倒，生死攸关的事情，咱们还得靠咱们自己啊。”郭不戳。
<br/>　　高飞虎脸上笑容瞬间消失，眉头紧紧皱了起来。
<br/>　　“你这话中什么意思？”
<br/>　　“老大……”郭不戳一怯：“我只是说我们不能把希望全部寄托在别人身上。”
<br/>　　“住口！”高飞虎突然发怒：“我和大当家是一起从青牛山杀出来的，十万戎族都被我们杀了，我们是生死弟兄，什么别人。”
<br/>　　高飞虎吹捧程大雷，其实也有吹捧自己的事情。青牛山一战，他也参与了，一辈子只这一件露脸的事情，所以时常提起。
<br/>　　“老大，你莫生气，只是，大当家真能击溃凉州兵么？”郭不戳赶紧赔罪，他倒是没有质疑高飞虎的勇气，只不过眼前的情况，真的很难让他乐观起来。
<br/>　　传言说程大雷在凉州如何如何了得，关于他的传说，江南那边也是沸沸扬扬。但传说终究只是传说，究竟是名不虚传，还是盛名之下其实难副，也只有见过程大雷本人才知道。
<br/>　　“你给我把心放在肚子里。”高飞虎重重的冷哼一声：“蛤蟆寨上下，都是我磕头换命的兄弟，知道我高飞虎出事，他们谁都坐不住。肯定是第一时间赶过来。”
<br/>　　高飞虎说这话时，心中其实没多大底气。他和程大雷曾经并肩作战，但要说他在蛤蟆寨的人缘，其实远远谈不上好。
<br/>(本章完)
