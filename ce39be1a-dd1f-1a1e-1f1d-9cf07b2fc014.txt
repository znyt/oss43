        &lt;h3&gt;第三卷第二百二十三章  寒伯安在美洲  （寒冰玉出事）&lt;/h3&gt;
<br/>
<br/>        寒伯安在欧洲的事情还没有解决，就看到了惊天噩耗。
<br/>
<br/>        寒冰玉和落雪葬身火海。
<br/>
<br/>        新闻里都是寒冰玉和落雪葬身火海的消息，寒伯安没有心思再管别的事情，拿起车钥匙抓住李长卿，开车赶往出事地点。
<br/>
<br/>        落家分公司的经理悲疼的介绍着寒冰玉和落雪的情况，并且深感自责，没有照顾好寒冰玉和落雪。
<br/>
<br/>        寒伯安并不相信他的兄弟会那么容易葬身火海，除非是有人为之，除非是寒冰玉和落雪没有任何防范，而能让寒冰玉和落雪没有防范的人除了落家人还有谁？
<br/>
<br/>        寒伯安心里充满了悲愤，如果寒冰玉真的是葬身火海，他将会不惜一切代价找到凶手，让他们给他弟弟和落雪陪葬，他深信，落家也会这样做的。
<br/>
<br/>        他给落雪的父亲打了一个电话：“落伯伯，您肯定是看到新闻了。”
<br/>
<br/>        “伯安，我看到了，我准备启程去美洲。”落伯伯声音沙哑。
<br/>
<br/>        “您先不要过来，我在美洲，我先看看情况，有什么事情我会及时给您汇报。另外，家里的事情您多担待些，告诉我父亲让他最近一定要保持中立，切不可轻举妄动。”
<br/>
<br/>        “好！”
<br/>
<br/>        他们几家关系好的都知道，落白雪只有落雪一个女儿，而落白雪将她的女儿雪藏了很多年，突然让她的女儿接手落家的事业，在短短的几年里落雪已经完全超乎了人们的想象，她所做的事情一件比一件刷新了人们对她的认知，表面上看，她美丽无双，与世无争，其实其凌厉的手段让他们这些男人都望尘莫及，不出手则以，一出手便是让人家破，连翻盘的机会都没有，比喻说陈家事件，短短的两天，一个商业帝国陨落，只留下一个空壳，而她还美其名曰救陈家如水火之中，陈家是有苦难言还得对落雪感激涕零。
<br/>
<br/>        不过树大招风，落雪的成长也会诱发一系列的反应，有人嫉恨也是正常的，尤其是外界传言落雪和吴家的联姻让大家对局势感觉扑朔迷离，至于落家内部有没有内斗也很难说清楚。
<br/>
<br/>        弟弟寒冰玉忠心耿耿的跟在落雪的身边，是落雪的知心好友也是落雪的守护神更是深深的爱着落雪，愿意为落雪做任何事请，但是他还有另外的一种身份，一个职能部门的领导人，这个身份只有极少数人知道，包括他的父亲都不知道，所以弟弟经常告诉他要保持中立。
<br/>
<br/>        是谁如此大胆敢动寒冰玉和落雪？
<br/>
<br/>        寒伯安赶到现场时，现场的火情已经得到控制，有关方面正在调查情况，看到的只是一堆灰烬，寒伯安悲痛欲绝。
<br/>
<br/>        他要开始行动了，树欲静而风不止啊！
<br/>
<br/>        “总裁，你看......”李长卿欲言又止。
<br/>
<br/>        “你说你的意思。”
<br/>
<br/>        “你弟弟是著名的散打王子，其敏捷的身手和明锐的判断力是没有几个人能匹配得上的，何况他还在特殊部门呆过，对危险的感知度超乎常人，所以我不相信他会出事情。”
<br/>
<br/>        “我也是这么想的，只是现场这个样子，你看.......”
<br/>
<br/>        “待会人都走了，我去整点东西带回去，做个dna检测就好。”李长卿轻声说。
<br/>
<br/>        “那我们先去别的地方，你联系一下斐南，让他调查一下最近这边的情况，看看有哪些人和落氏分部领导人来往密切，给我盯紧了。”
<br/>
<br/>        “好！你先回分部，我来做这些事情，另外你最好也带上保镖。”
<br/>
<br/>        寒伯安心情忐忑的回到了分部，迎来了一位不速之客。
<br/>
<br/>        “请问，你是寒总吗？”
<br/>
<br/>        “我是！”
<br/>
<br/>        “这个是我的名片。”来人拿出名片。
<br/>
<br/>        “我其实是甸城谷家的人，我是谷家老三，沧岚雪，在三年前来到美洲，负责美洲一切事宜。”
<br/>
<br/>        “你是谷家老三为什么不姓谷，反而姓沧？”
<br/>
<br/>        “哈哈哈，寒总不知道的事情很多，我们谷家其实都是异姓兄弟，谷老大是我们老大，然后是秦寿，老三是我。不瞒你们说，我们都是陆家和吴家的人，我们所做的一切都是他们授意的。”
<br/>
<br/>        “沧总告诉我这些是什么意思？”
<br/>
<br/>        “吴家和陆家有什么样错综复杂的关系自然不需要我细述，寒家能立足社会必然有过人之处，我们最近需要你们承运一批货物，下面的人说，寒家拒收，我就来拜访一下寒总。”
<br/>
<br/>        “沧总这是来要挟我？寒家从一开始就不会承运一些不合规矩的东西，或者是有损我国利益的东西，如果沧总没有这两条，我们都是会承运的。”寒伯安冷冷的看着来人。
<br/>
<br/>        “我们并没有要你们承运不合规矩的东西，只是中间有点小东西需要一起带回去或者是带出来。”这些年海陆空是越来越严厉了，有些东西走正常渠道完全是不可能的，像寒家这样的老企业，蒙混过关的几率很大，他们有很多的货物需要出手，得找一家明面上的企业。
<br/>
<br/>        如果寒家能加入到他们的团队中来，无疑是如虎添翼，谷强在三年前搁浅了计划，着手搞信息化工程，谷老大便找准时机让他先期来开拓市场，打通两边的路子。
<br/>
<br/>        一明一暗，他们要确保万无一失。
<br/>
<br/>        沧岚雪认为，利益能打动一切，寒家不同意还是条件没有达到，如果条件能达到，谁会拒绝金钱的魅力？
<br/>
<br/>        “只要一切合规都好办！”寒伯安还是坚持着自己的意见。
<br/>
<br/>        “寒总，如果你们承运我们的东西，我会再给你一个寒家这么多的资产。”
<br/>
<br/>        “哈哈哈，沧总这是将我寒家都调查清楚了？连我寒家有多少资产都知道了？”
<br/>
<br/>        “想要合作，肯定是会知道对方能不能答应，也是要知道对方的优缺点。我们从来不打没有把握的账。”
<br/>
<br/>        “哈哈哈哈，我们寒家恰恰不缺钱。”
<br/>
<br/>        “是的，但是寒家缺人，你弟弟前不久葬身火海。不，是就在前几个小时。”
<br/>
<br/>        “这个是你们干的？”
<br/>
<br/>        “不不不，怎么会是我们呢，我们只是帮着找了点素材。我们现在也在帮寒家找证据！”
<br/>
<br/>        “你们到底想干什么？”
<br/>
<br/>        “我只是来找你们帮忙，签了这份合约，我们就是友好的合作关系了。”
<br/>
<br/>        “如果我不签呢？”
<br/>
<br/>        “你的前女友苓希小姐五年前成了我们公司里的得力干将，这会正在陪着你的太太安男女士讲故事。不信，你可以问问。”
