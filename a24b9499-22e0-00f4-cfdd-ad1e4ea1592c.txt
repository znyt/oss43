        她不知道自己哭了多久，倏然间，身子腾空升起，童筱颖只觉自己落入了一个温暖的怀抱中。
<br/>
<br/>        当她定睛看到是傅夜沉抱着自己的时候，禁不住抽泣得更加厉害了。
<br/>
<br/>        童筱颖哭得梨花带雨，双腿挣扎着要推开傅夜沉。
<br/>
<br/>        她的一双小手不停地捶着他健硕的胸膛，止不住地号啕大哭：“你放开我！我不需要你！你走开！我不需要你！我不需要你！”
<br/>
<br/>        “宝贝，别闹了。”傅夜沉温柔地哄道，被她一拳又一拳重重地捶打在胸口上，连眉头都不皱一下。
<br/>
<br/>        直到他将她抱回了车内，她仍旧不依不饶地挣扎，潸然泪下地呵斥：“你放开我！我不需要你！不需要你！不需要你！你走开！”
<br/>
<br/>        小爱很识趣地从童筱颖的怀中跳了下来，爬去了副驾驶座上。
<br/>
<br/>        而下一秒，她被傅夜沉推倒在后座上，他的身子猛地倾身压下，炙热的吻铺天盖地的席卷而来，时而温柔、时而怜惜地在她的唇上摩挲，缠绵悱恻。
<br/>
<br/>        “唔——”童筱颖受不了他这热情的法吻，挣扎着要推开他，可是双手却被他紧紧地扣向了头顶。
<br/>
<br/>        泪水的咸味漫上味蕾，唇瓣微离，四目灼灼地相视。
<br/>
<br/>        “筱颖……”
<br/>
<br/>        他支撑起身子，凝视着她明亮的眸子里噙着泪水，因他的一声叫唤，轰然决堤，莹润泪水顺着脸颊滑落，让他心疼不已。
<br/>
<br/>        傅夜沉慢慢地松开了童筱颖的双腕，极其耐心地试图解释：“她叫夏玄炘，是夏凌焰的堂妹，同时也是我领导的侄女。”
<br/>
<br/>        他不说还好，一开口，童筱颖的眼泪，骨碌地从眼角滑出，晶莹的泪珠落在他的手上，他的心不由得随之一颤。
<br/>
<br/>        童筱颖一边抽泣，一边继续捶打着他的胸膛，终于，她打累了，也哭累了，双手揪着他的衣襟，声音嘶哑而哽咽：“我没有利用你去报复昼景。”
<br/>
<br/>        “我知道。”见身下的女人安分了，傅夜沉微微挑眉，唇角止不住地上扬。
<br/>
<br/>        他单手托起她的脸蛋，又俯下身去，轻柔地吻干她眼角的泪痕。
<br/>
<br/>        昨天，他其实半信半疑，甚至陪着她在阿景面前演了一场戏。
<br/>
<br/>        那时，他彷徨过，也迷茫过，一边开车一边独自安静地思考了许久。
<br/>
<br/>        她最近每一次都会主动跟他解释，但这一次，他忽然间不需要她的解释了。
<br/>
<br/>        因为，他恍然大悟。
<br/>
<br/>        或许，她只是为了气走阿景，为了断了阿景对她的念想，也为了防止阿景嫉恨他，才对阿景说那些狠话。
<br/>
<br/>        只是事后，他确实有事不得已离开。
<br/>
<br/>        更多的，他还想确定自己在她心里，到底处于什么位置。
<br/>
<br/>        她不是为了报复阿景才决定要当好他的太太，那她是为了什么？
<br/>
<br/>        这些天，她为他的改变，他心有所感。
<br/>
<br/>        是因为她已经喜欢上他了吗？
<br/>
<br/>        童筱颖缓缓地拉起傅夜沉的手，贴在自己的脸颊上，轻轻地问道：“那大哥，可不可以，不要不理我？”
<br/>
<br/>        这是在跟他撒娇吗？
<br/>
<br/>        傅夜沉剑眉微挑，深深地凝视着她的眼睛，只觉自己宽厚的手掌下传来她小手那软软的触感。
<br/>
<br/>        他心里倏地滑过一阵暖流，那徘徊不定的心，仅仅因为她这一句示好的话而得到了肯定的答案。
<br/>
<br/>        “乖，闭上眼睛。”傅夜沉的另一只手，伸进了西装裤口袋里。
<br/>
<br/>        “诶？”童筱颖不明所以然地眨巴着双眼，但还是很听话地闭上了双眼。
<br/>
<br/>        隐约能感觉到他撩起她披散的长卷发，脖颈间划过一丝凉意，睁眼的时候，只见他手里托着一个铂金项链吊起来的天鹅形状的坠子。
<br/>
<br/>        天鹅的眼睛是黑钻，翅膀上由白色碎钻点缀，做工十分精致，璀璨夺目，一时间让她移不开眼。
<br/>
<br/>        “喜欢吗？”他温热的气息喷洒在她脸上，温柔地说道，“特意为你订制的。”
<br/>
<br/>        “喜欢大哥、谢谢你。”她的心里，此刻犹如蜜糖般甜得快要将她融化。
<br/>
<br/>        童筱颖故意将前面四个字连起来说，抬起头来时，在傅夜沉的薄唇上飞快地送上一吻。
<br/>
<br/>        然而，就在她即将退开之际，他却已经托起她的后脑，将她这难得主动的吻，慢慢地加深。
<br/>
<br/>        童筱颖睁大了眼睛，看着这个将她再次压回后座上热吻的男人，鼻息里全是属于他的味道，一颗心好像被抛到了云端，心跳越来越快。
<br/>
<br/>        隔着衣服，她能感觉到他的身体在发烫。
<br/>
<br/>        她完全没料到，自己这主动的一记浅吻，竟然会让这空气里散发出荷尔蒙的气息。
<br/>
<br/>        他的吻在不知不觉中结束，彼此的唇珠上泛着车窗外的流光溢彩。
