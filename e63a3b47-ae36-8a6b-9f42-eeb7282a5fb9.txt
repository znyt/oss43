　　第204章
<br/>　　夜来，徐神机、秦蛮、徐灵儿三人聚在一起，交流着得来的情报。
<br/>　　“最近长安就两件大事，一是武科举，二是杨龙停要来长安，这两件事自不必说。我还打听到一件事，戎族的和谈队伍要来了，这次是为把帝国某位公主娶走。另有一件事，帝国六王子在一年前失踪，至今下落不明。”
<br/>　　经过一年相处，三人已养成默契，每到一个地方，三人就会分开打探情报，看哪件事可能和程大雷有关系，抽丝剥茧，寻找其中线索。
<br/>　　“我在市集上打听到，长安城西有座葫芦山，山上盘踞着一伙山贼，有一千多人，官兵剿了几次，都没有打下来。其他，也就没打听到什么。”
<br/>　　“嗯，我打听到一件事。”徐灵儿道：“长安最大的青楼是满庭芳，满庭芳的花魁名叫一枝桃，一枝桃向来是不陪客的，多少人一掷千金都难见一枝桃一面。但为了这次武科举，一枝桃已经放出话来，谁若能夺得武状元，一枝桃愿意以身相许。”徐灵儿。
<br/>　　“满庭芳花魁，一枝桃？”徐神机若有所思。
<br/>　　“军师，您这身体已经不行了。”秦蛮。
<br/>　　“谁说我身体不行，我身体好得很。”徐神机勃然大怒，道：“什么跟什么啊，我只是在想这件事和大当家会不会有关系。”
<br/>　　“那有关系嚒？”秦蛮。
<br/>　　“好像没什么关系。”徐神机叹了一声，忽然想起什么，冲徐灵儿道：“青楼那种地方的消息，你是这么打听到的？”
<br/>　　“我进去了呀。我装作卖花的，他们就放我进去了。”
<br/>　　“你一个女孩子怎么能去那种地方！”徐神机大惊。
<br/>　　“为什么我不能去啊，里面的小姐姐个个都是人才，说话又好听，好喜欢在里面，爹爹不也是经常去么。”
<br/>　　“我什么时候去了，我那是为了工作，工作的事情……和你说了你也不明白。”
<br/>　　“军师，军师，言归正传，你看这些消息，和大当家有关么？”
<br/>　　“好像，也没什么关系。”徐神机想了想道：“当然，今天这只是第一天，长安人多事多，咱们再打听打听，如果实在得不到什么消息，我们就去葫芦山走一遭。”
<br/>　　“大当家会在葫芦山么？”
<br/>　　“有可能吧，大当家总不能忘了自己的老本行。”
<br/>　　谈罢，三人都有些心灰意冷，寻了一年，却无程大雷半点消息。长安已是大家最后的希望，如果再寻不到，大家实在已不知该去何处。
<br/>　　“天不早了，咱们先歇着吧。现在长安人满为患，到处都是赶考的武人，咱们只订到一间房。灵儿你睡床上，我和秦蛮打地铺。”
<br/>　　“不知到什么时候，咱们才能找到大当家。”灵儿叹了口气。
<br/>　　“要说那……蛤蟆……可真够了得的，生生就进了王府，王府可是有好多高手，竟一点察觉都没有。”
<br/>　　正在此时，隔壁房间传来说话声，隔着墙也听不太清楚，只模糊听到『蛤蟆』二字。
<br/>　　整整一年，三人都在寻找程大雷，如今对这两个字已经过敏。此时三人都是一机灵，腾地站起来，瞬间窜出房间。
<br/>　　砰！
<br/>　　房门被一脚踹开，屋里两个胖子正在喝茶，此时见三人穷凶极恶闯进来，顿时吓了一跳。
<br/>　　“你们……是什么人？”
<br/>　　还未反应过来，秦蛮已经用刀压住一人的脖子，另外一人刚想出声，徐灵儿的刀搁在他的背心。
<br/>　　“三位好汉，三位好汉，饶命呐。”
<br/>　　徐神机将门关好，笑眯眯的走过来，道：“二位稍安勿躁，我们只是想向二位打听一些事，除此之外，既不谋财，也不害命。”
<br/>　　“你们要银子我可以给，求求你们别杀我。”
<br/>　　“看二位紧张的，肯定是把我们当坏人了。”徐神机给二人倒满茶，道：“先喝杯茶压压惊，我就是想问问，刚才二位聊什么呢？”
<br/>　　“我们正说那名扬京城的大侠飞天蛤蟆。”
<br/>　　“飞天蛤蟆？”徐神机与秦蛮对视一眼，二人彼此点点头，徐神机在二人对面坐下，微微倾身：“详细说说。”
<br/>　　一提起这件事，两个胖子立刻兴奋起来，吐沫横飞，满面涨红。
<br/>　　“飞天蛤蟆可是了不起的大侠，专门劫富济贫，只对为富不仁的恶人下手。官兵抓了他多少次，连他的影子都没摸着，有次宫里的禁卫军都动手了，可是布下天罗地网，最后怎么样，愣是被他跑了。”
<br/>　　“我听说那蛤蟆大侠不是凡人，天生背后长着一对肉翅，扇一扇就飞上天，跺跺脚就几十丈，要不怎么叫飞天蛤蟆呢。”
<br/>　　“我觉得也是，如果一般人能逃脱宫里禁卫军的追捕么，肯定不是凡人。”
<br/>　　徐神机、秦蛮、徐灵儿彼此看看，心里都觉得这飞天蛤蟆的行事风格和程大雷很相似，事情至少有五分数了。
<br/>　　可关键是怎么找到这飞天蛤蟆，既然是大盗，自然是隐藏行踪，不会轻易被人发现。
<br/>　　“这蛤蟆大侠还有一个规矩，每次作案之前，都会给对方家里送上一张二指长的条子，声明某日某时某刻前去盗宝，蛤蟆大侠做了这么多起案子，是十七件，还是十八件，可从来没有一日耽误过时辰。”
<br/>　　“是十七件，三天前，周郎官就收到了蛤蟆大侠的条子，上面写明今夜子时三刻要取周郎官的官印，我听说，条子上还画着一只小蛤蟆，加上这一件就是十八件了。”
<br/>　　腾！
<br/>　　两个胖子正聊得起劲，浑然忘了脖子上还压着刀呢，突然间，三个匪人猛地蹿了出去，房门洞开。
<br/>　　两个胖子摸摸脑袋，完全不明白怎么回事。
<br/>　　“大哥，他们这是走了吧？”
<br/>　　“好像是走了。二弟，你觉得他们是干嘛的？”
<br/>　　“估计……”胖子顿了顿：“也是大侠呐。”
<br/>　　“什么侠？”
<br/>　　“蹿这么快，兔侠吧？”
<br/>　　突然，刚离开的三人又蹿了回来。
<br/>　　“大侠饶命，大侠饶命，我们什么都说了啊。”
<br/>　　“说！”秦蛮闷声低吼：“那周郎官家在哪里？”
<br/>
