　　在这山顶上，武者便是已经聚集的越来越多。
<br/>　　药王谷，天枢殿的武者，也都已经是纷纷到来，在山顶上，大约已经是有着二百多人。
<br/>　　楚风眠之前大约看过，在山脚下当初聚集的武者，只怕是有着接近千人之多，现在才仅仅二百多人上来，剩下的人，便是都无法通过这第一道考验。
<br/>　　不过目前人数还在增加，最后的人数，应该还是要远远比起这多。
<br/>　　“那是药王谷的药海，据说小小年纪，都已经是炼制出了天级灵丹！”
<br/>　　药王谷弟子经过一刻，一尊武者也是指着为首的年轻男子开口道。
<br/>　　这年轻男子长相淡雅，跟其他武者大部分都是香门第出身的。
<br/>　　药王谷位列七大宗门，最为著名的并非是实力，而是医术。
<br/>　　九域的无数炼丹宗门，几乎都是从药王谷中走出的，或者是跟着药王谷有着千丝万缕的关系。
<br/>　　可以说七大宗门之中，最为不能招惹的，就是这药王谷，谁若是杀了药王谷的弟子，就像是捅了一个马蜂窝，会遭到无数强者得报复。
<br/>　　“药王谷的弟子，这群天天炼丹的家伙，为什么要来归墟天？”
<br/>　　一名武者，不由的有些不解询问道。
<br/>　　“在这归墟天中，其他武者可都不会忌惮他们的身份，尤其是药王谷弟子，一个个身家富足，想要杀了他们的人可不少。”
<br/>　　“杀？你真以为药王谷的人，都是手无缚鸡之力的人？”
<br/>　　旁边一名武者，听到这话冷哼一声道。
<br/>　　“药王谷的弟子，除了一身医术外，一个个可都是精修毒术的大师，这药海能够炼制天级灵丹，自然也能够去炼制天级剧毒，你们谁有自信可以挡下天级剧毒？”
<br/>　　“天级剧毒。”
<br/>　　听到这武者的话，其他武者便是都闭上了嘴，无人还敢开口质疑。
<br/>　　毒师，在整个九域之中都是让所有武者最不愿意去得罪的存在。
<br/>　　因为一个强大的毒师，他所炼制的剧毒，释放的毒术，几乎都是在无形之中。
<br/>　　要是招惹上了一位强大的毒师，那便是不知道什么时候，就会中毒陨落。
<br/>　　医者毒更精，药王谷弟子的毒术，足以称得上是九域第一，真的贸然招惹他们，才是真正找死。
<br/>　　楚风眠自然也清楚这一点，当初七大宗门围攻剑道之主的时候，这药王谷可是出了大力。
<br/>　　当时药王谷的那些老古董，将他们远古时代传承的剧毒都给拿了出来，让剑道之主不小心中毒，这样才会被七大宗门围杀的。
<br/>　　楚风眠的眼神中显现出几分杀意。
<br/>　　在场所有七大宗门的弟子，谁也别想活着离开。
<br/>　　除了这些药王谷弟子外，天枢殿的弟子也都过来，对于天枢殿，众人所能够形容的两个字，便是神秘。
<br/>　　天枢殿的功法，乃是沟通天上星辰的力量，这在无数武者看来，都是虚无缥缈的传说罢了。
<br/>　　但是天枢殿正是依此，才从九域中真正崛起，以至于现在，无人敢去轻视他们。
<br/>　　尤其是大部分的武者，都不知道如何跟天枢殿弟子为敌，一个个便是极为忌惮，只能避开。
<br/>　　“天枢殿弟子中为首之人，名为星筠，当初在我被追杀的时候，曾经碰到过她，不过她并没有杀我，还给我了丹药，放走了我。”
<br/>　　看这天枢殿弟子经过，寒月漓突然开口道。
<br/>　　“若是碰到她，最好放她一条生路，毕竟我欠她这一个人情。”
<br/>　　“哦？”
<br/>　　听到寒月漓的话，楚风眠一眼望过去，便是看到天枢殿弟子中心，便是一名女子，这女子一身星袍，整个人看起来都是神秘无比，楚风眠都似乎看不清她的容颜。
<br/>　　以楚风眠的灵识，就算是让碎命之境的武者前来，楚风眠都可以直接看破。
<br/>　　但是这星筠，楚风眠却是无法看破，显然此人的体质极为特殊，令楚风眠都无法摸清楚底细。
<br/>　　“她既然帮过你一次，我也算是欠她一个人情，只要她不与我为敌，我可以不去杀她。”
<br/>　　楚风眠轻声说道。
<br/>　　楚风眠嗜杀，却也并非是滥杀无辜之人，这星筠救过寒月漓一次，楚风眠自然不可能会在杀她。
<br/>　　在楚风眠看向那星筠的一刻，这星筠似乎也是转过头来，看向了楚风眠方向看了一眼，随后便是带着天枢殿弟子，也占据了山顶上的一角。
<br/>　　七大宗门其四，已经全部到达。
<br/>　　只是剩下九魔城，天剑宗，圣王宗的弟子，却是没有任何一人，不见踪影。
<br/>　　“九魔城，天剑宗，圣王宗的弟子，怎么没有任何一人到来？”
<br/>　　一些武者，也是察觉到了这一点。
<br/>　　云羽天宫开启，绝对是天羽神域之中发生的最大之事，其中的无数武者，都是为了这云羽天宫的宝藏传承，才来到天羽神域之中的。
<br/>　　不可能会不来的才对。
<br/>　　突然无数人，都是想到了最近的一个传闻，九魔城，天剑宗，圣王宗弟子，都被一名武者所斩杀。
<br/>　　这个消息，就是从这两天才开始流传的，不过在武者武者看来，都是如同笑话一样。
<br/>　　在这天羽神域之中，便是以七大宗门的弟子为首，他们数量众多，实力强大。
<br/>　　别说是一个人，就算是任何一方势力，都无法跟七大宗门所抗衡。
<br/>　　可现在，这九魔城，天剑宗，圣王宗的弟子，居然真的没有出现，让他们不少人，都是想到了这个传闻。
<br/>　　不少武者，都是纷纷向着周围张望去，想要看一下，那传闻之中的武者，到底是谁。
<br/>　　虽有传闻，但是真正谁杀的，却是并没有人知晓。
<br/>　　其中几道视线，都是纷纷落入到了楚风眠的身上来。
<br/>　　楚风眠，这样一个生面孔，自然是值得他们怀疑。
<br/>　　但是当他们打量了一眼楚风眠的境界之后，便是也就没有在观察了。
<br/>　　生死境五重的境界。
<br/>　　在天羽神域之中，都只能排名末流。
<br/>　　这样的人，怎么可能以一己之力，毁灭三方势力。
<br/>　　众人都在互相打量，一个个怀疑，却并没有人真正开口议论，大部分人的注意力，都是集中在了那彩虹阶梯之上。
<br/>　　“七万年来，终于又有人，踏足这云羽天宫了。”
<br/>　　一声神秘的声音，突然从天上传来，这声音的出现，让在场无数人的脸色，都是惊讶无比。
