        “跟我见最后一面？原因呢？”何明旭不以为然地反问。
<br/>
<br/>        童筱颖抿了抿唇，回答道：“因为，你一直都是我最喜欢的男生。”
<br/>
<br/>        “这就是你要跟我说的话？”何明旭微眯起眼睛，狐疑地反问。
<br/>
<br/>        童筱颖微微摇了摇头，抬起手来，揉了揉眼睛，故作哭泣状地说道：“我家的事情，你一定从你父母那儿听说了吧！我爸妈负债，我哥哥坐牢。家里缺钱，所以，我决定出去卖。你知道的，我还有第一次。我不想把我的第一次，交给陌生男人。所以，我想把我的第一次给你。从今以后，我会把你深深地藏在我心里。”
<br/>
<br/>        “筱颖你……”何明旭欲言又止，忽然间心软地皱起了眉头。
<br/>
<br/>        筱颖的心思向来单纯，这点何明旭笃定。
<br/>
<br/>        只不过……
<br/>
<br/>        那晚后来到底发生了什么？
<br/>
<br/>        学校里封锁了所有的消息，就连网上都没有任何蛛丝马迹可寻。
<br/>
<br/>        其次，筱颖居然还能安然无恙地出现在这里，没有被警察捉去审问？！
<br/>
<br/>        种种疑惑，在何明旭的心头萦绕。
<br/>
<br/>        “阿旭，你不愿意吗？”童筱颖抬起双眸，充满希冀地凝视着何明旭的眼睛。
<br/>
<br/>        而其实，在她眸光深处，藏匿着对他的憎恨。
<br/>
<br/>        “哼！筱颖，你是想跟我玩什么花招？”何明旭忽然警惕地呵斥。
<br/>
<br/>        童筱颖一脸茫然地看着何明旭，瘪着嘴，无辜地反问：“阿旭，你是怎么呢？”
<br/>
<br/>        其实，此时此刻，她心里没底了。
<br/>
<br/>        她不知道何明旭会不会就此上当，计划好的台词，也已经偏离了轨线。
<br/>
<br/>        果然，计划赶不上变化。
<br/>
<br/>        何明旭看着童筱颖这张清纯的脸蛋，脑海里回想起了他们的高中时代。
<br/>
<br/>        从高一一直到高三，他们都是同桌。
<br/>
<br/>        那个时候，他的学习压力很大，她与他互勉，而且她还会做好吃的早餐从家里带过来给他吃。
<br/>
<br/>        是什么时候起，他看她的心思变了。
<br/>
<br/>        童筱颖见何明旭的思绪在开小差，为了防止他因为多想或是猜忌，而坏了自己的计划，她顾不了那么多，直接扑入了何明旭的怀中，紧紧地抱着他，哀求道：“阿旭，你就完成我的心愿，好吗？”
<br/>
<br/>        何明旭终究还是心软了，抬起双手，搂住了童筱颖的身子，嘴角不禁咧起一抹阴阴的笑意。
<br/>
<br/>        不如赌一把！
<br/>
<br/>        何止他何明旭在赌，童筱颖也在赌，就只看最后谁输谁赢了。
<br/>
<br/>        到了临海大学，就是他何明旭的地盘。
<br/>
<br/>        何明旭轻车熟路地带着童筱颖在临海大学附近的小宾馆里开了房。
<br/>
<br/>        像她童筱颖这种身份，第一次，也就只配这种小宾馆。
<br/>
<br/>        但不得不说，何明旭再次看到她童筱颖的时候，还是会怦然心动。
<br/>
<br/>        他不否认，她曾是他的初恋，但是他和她已经回不去了。
<br/>
<br/>        他不再是高中时期的那个书呆子，也不再像高中时期那么单纯。
<br/>
<br/>        两个人各怀心思……
<br/>
<br/>        既然何明旭害得马智杰被学校开除，那么她童筱颖也要让他何明旭尝尝被学校开除的滋味。
<br/>
<br/>        刷了房卡，一进门，何明旭就开始迫不及待地想要脱衣服。
<br/>
<br/>        童筱颖故作娇羞地说道：“我们可不可以先点东西吃，我……今天为了等你，连中饭都未吃……”
<br/>
<br/>        “好！”何明旭咧了咧嘴，从裤兜里掏出手机，打算点外卖。
<br/>
<br/>        “我想吃辣一点的饭菜！”童筱颖弱弱地补充道，“记得帮我点矿泉水。”
<br/>
<br/>        何明旭有些猴急，她说什么，他都说好。
<br/>
<br/>        “要不，我先去洗澡。”童筱颖娇羞地说道。
<br/>
<br/>        何明旭点了点头。
<br/>
<br/>        童筱颖提起自己的包包，进了浴室。
<br/>
<br/>        外卖什么时候到的，童筱颖就什么时候从浴室里出来。
<br/>
<br/>        对于一个女人洗澡要洗半个小时，何明旭似乎一点都不足为奇。
<br/>
<br/>        因为，他的女友林若琴也是这样子。

<br/>        童筱颖身上穿着松垮的浴袍，披散着湿漉漉的头发，光着脚丫子从浴室里走了出来。
<br/>
<br/>        何明旭正拿起筷子准备吃外卖的时候，童筱颖笑着娇嗔道：“快去洗洗手呀！饭前不洗手，会生病的噢！”
<br/>
<br/>        这是她读高中时，和他一起去食堂吃饭前常说的一句话。
<br/>
<br/>        何明旭并未多想，起身去了卫生间。
<br/>
<br/>        童筱颖见状，以最快的速度，从浴袍里掏出一支注射器，将里面混合了药粉的液体，注入到了其中一瓶矿泉水中的底层。
<br/>
<br/>        何明旭洗完手回来时，发现童筱颖已经开吃了，于是也跟着端起自己的那份也跟着吃了起来。
<br/>
<br/>        就在童筱颖感觉到何明旭也快辣到不行的时候，于是随手拿起其中一瓶矿泉水扭开来喝了一口后。
