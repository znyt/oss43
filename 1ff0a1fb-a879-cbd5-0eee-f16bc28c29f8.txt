　　第170章
<br/>　　篝火照亮山谷，也同样照亮程大雷的杀意。当那杀意流露出来时，不禁令所有人心中一寒。
<br/>　　徐神机、刘悲、秦蛮、高飞虎……包括苏樱等人都没有看穿程大雷，也没有看懂程大雷。
<br/>　　他从未想过逃，从来没有！
<br/>　　想到自己还劝他和自己一起去冀州避难，苏樱轻轻叹口气，原来自己从来没有看懂过他。
<br/>　　面对高飞虎的指责，程大雷放声大笑，笑得如此痛快淋漓。
<br/>　　众人安静下来，看着抓狂大笑的程大雷，如同看着一个疯子。
<br/>　　“不错，我从来没想过逃！”程大雷忽然收住笑声，狰狞道：“逃，那里逃，如丧家犬一样把后背卖给敌人，逃得过去么！”
<br/>　　锵！
<br/>　　匹夫剑出鞘，程大雷挥剑指着谷外。
<br/>　　“戎族已经到了，此地就是绝境，你们再无退路，除了舍身一战，别无选择！”
<br/>　　众人一片窃窃私语，现在想逃自然再无可能。可是守……
<br/>　　“对方可是戎族，咱们能打的不过两千人，能守得住么？”有人忍不住问出声来。
<br/>　　咚咚咚！
<br/>　　急促的马蹄声，有人从山寨大门跑到这里，气喘吁吁道：“杀，杀过来了！戎族杀到门口了！”
<br/>　　一片哗然，有小孩已经吓得惨叫出声。有人看着程大雷的目光充满埋怨，眼神似乎在说：看吧，看吧，我说逃命去吧，你非要作死！
<br/>　　“怕什么！不过一两千追兵而已！”程大雷道：“不是说戎族不可敌么，今日我就杀给你们看！”
<br/>　　众人随着程大雷上城墙，只见城下乌压压一片，戎族骑兵在马上用弓箭向上射。
<br/>　　程大雷挥剑将射到自己面前的一支箭挑开，由下往向射，射到城墙上已经乏力，根本没有任何杀伤力。
<br/>　　程大雷冷哼一声：“把重弩推上来，五十人一组，听我号令！”
<br/>　　上百架重弩搭在城头，对准城下攻城的骑兵。
<br/>　　程大雷举剑，落，整齐的弩箭射杀出去。
<br/>　　程大雷再举剑，再落，又是一排弩箭。
<br/>　　重弩的原型是系统奖励的，山寨联合后有木匠，想仿造也并不困难。修建山寨的时候，程大雷集中人力仿造了一批重弩。这些重弩威力极大，三百步内能穿马匹。
<br/>　　高飞虎忽然想起，程大雷这几天让大家伐木运进山寨，说要制造马车，现在才想明白，他那里是要制造马车，他是要储备弩箭。
<br/>　　这个人，原来一开始就想好了！
<br/>　　随着程大雷命令，一排排弩箭射杀出去，山谷门前十分狭窄，挤在下面的戎族骑兵根本逃无可逃。
<br/>　　他们如大浪一般重逢，弩箭把他们的重逢击退，然后又像大浪般涌上来，再次被弩箭击退。
<br/>　　杀戮，如此干净简单。
<br/>　　山寨诸多人都挤在城头，看着这一幕，脸上露出思索模样。
<br/>　　戎族不可敌，他们身强体壮，个个有万夫不当之勇……这些话口耳相传，慢慢就沉进众人心底。所以大家见到戎族的第一个念头不是战，而是逃。
<br/>　　每个人都是这么个想法，所以戎族才能一日破幽州兵，半日破黑石城。薛半川能挡杨龙停两个月，真就挡不了戎族半天么？怕只怕，还是未战先怯的心理多些。
<br/>　　而今天，程大雷却让大家看到，戎族并非三头六臂，刀枪不入，他们不是怪物，也是娘生父母养，一样是人。痛了会叫，死了会怕，会痛会死，就可以被击败。
<br/>　　“来，来，兄弟让我放一箭试试。”
<br/>　　“排队，后面排队去！”
<br/>　　“咱兄弟可不错，你不能这样对大哥，让我放一箭怎么了。”
<br/>　　“都停手！”程大雷忽然道：“下面已经没多少人了，不要浪费弩箭。”
<br/>　　程大雷回转身，一张脸从未有如此大的杀气：“下面如今只剩下下残兵胜勇，蛤蟆寨的好男儿，可愿意和我冲出去，把这帮畜生赶尽杀绝！”
<br/>　　“走！”众人齐呼。
<br/>　　程大雷收剑换斧，带着一队人往城下去。苏樱站在一旁，轻声嘱咐道：“你小心些，我等你回来。”
<br/>　　“女人，女人怎么在城门上！”程大雷看了他一眼：“从今日起，女人不许上城门。”
<br/>　　“我……”苏樱。
<br/>　　小蝶站在自家小姐身后：“他这么能和小姐这样说话，他……”
<br/>　　“好了，不要说了，我们回去吧，不要挡他做事！”
<br/>　　城门打开，程大雷带着队伍冲了出去。
<br/>　　山寨两千汉子冲杀出去，而戎族幸存者不过百余人，在两千凶神恶煞的汉子刀下，根本形不成有效的反抗。
<br/>　　这一战让大家看到，戎族也是可以被杀死的，当面对死亡时他们也是如此恐惧。那句话怎么说来着：杀人者原来如此懦弱！
<br/>　　战斗过后，程大雷带人清理战场，马匹盔甲兵器都要运回山寨。
<br/>　　“大当家，我点过了，咱们这次全歼敌人，杀敌一千一百二十名，咱们这边只有十几人受伤，大获全胜呐。”黄三元道。
<br/>　　的确算一场了不得的胜利，时间跨过三十年，空间放在整个帝国，这都算一场极其罕见的胜利。如果上报到朝廷，封个伯爵都不成问题。
<br/>　　然而大获全胜，有时候是好事，同时也是坏事。全歼敌人的含义是，没有一个戎族战士逃跑，而他们中箭的伤口也全在胸口，这意味着他们明知必死，也没有转过身逃命。
<br/>　　这是怎样凶悍的一匹敌人，将他们称为怪物其实并不过分。
<br/>　　这一战让大家明白，戎族并不是不可战胜。也让程大雷明白，想要战胜他们真的很难。
<br/>　　而今天，也仅仅是个开始。
<br/>　　黑石城，赤眉等着手下的回报，等待的时间略漫长了些。
<br/>　　“不过就是几十个人，早该杀死回来了，为何还要等到现在。”
<br/>　　“这些帝国人都是很好杀的！”托蛮笑道：“可能是路上耽搁时间了吧。”
<br/>　　抢掠帝国不是一次两次，大家早已习惯胜利，至于失败，今日死一队巡逻兵已经算是失败。
<br/>　　“咱们可要快些喔，别让我的那些哥哥弟弟抢了先。”赤眉歪在椅子上道了一声，忽然眉头皱起：“会不会他们中了埋伏，已经死了。”
<br/>　　那一夜，北蛮部大军出动，兵发青牛山。
