<br/>　　“血邪魔帝，化身为魔之时，曾经将净土宗的大半弟子，坑杀在这佛寺之中，这些怨灵，只怕都是那些净土宗弟子所化。”

<br/>　　血邪魔帝，可是在魔族历史之中最为耀眼的一位魔帝了。

<br/>　　一下子便是足以超越同境界的所有魔族，成为霸主，在魔族之中都会有着超然的地位。
<br/>　　正因如此。
<br/>　　哪怕是净土佛寺之中的危险，人尽皆知，这些在场的魔族在得到的消息之后，也都没有丝毫犹豫，所有人都敢来了。

<br/>　　夜魔王，血魔王，虚魔王，这三位魔王也是同样冲了过去。
<br/>　　这三尊魔王，本就都是九劫古帝巅峰的人物。

<br/>　　哪怕是蝎狮王，都要退避三人，一下子抢占到了最前方。
<br/>　　不过蝎狮王，也不甘落后，在他的身边，也同样有着一尊强横的魔族，显然是他拉拢而来的榜首。
<br/>　　也是向着净土佛寺冲了过去，争分夺秒。
<br/>　　“走！”
<br/>　　楚风眠也没有犹豫，身形猛然一变，直接化身为千丈魔龙的样子，龙威凝聚，向着净土佛寺冲了过去。
<br/>　　一路直行，任何敢挡在楚风眠面前的魔族，都被楚风眠一掌轰开，横行霸道无比。
<br/>　　将龙族的霸道，施展的淋漓尽致。
<br/>　　“该死，这头魔龙！”
<br/>　　几名被震开的魔族，都是口中吐出了一口鲜血，盯着横冲直撞的楚风眠，一脸怒意。
<br/>　　可更多的魔族，却是选择躲开，没有几个魔族愿意跟龙摩罗来做无用的争斗。
<br/>　　这净土佛寺之中，一眼看去便是知晓其中必然的危险无比，没有人愿意这个时候受伤。
<br/>　　楚风眠化身为魔龙，很快便是超越了不少的魔族，冲到了最前方。
<br/>　　那蝎狮王，也看到了楚风眠的到来，目光闪烁，却是没有阻拦，楚风眠而是选择了退避。
<br/>　　楚风眠一路横冲直撞，在他的面前，只剩下那夜魔王三人。

<br/>　　楚风眠张嘴长啸一声，龙吟咆哮，震的周围魔族都是瑟瑟发抖。
<br/>　　楚风眠施展出的龙吟，比起真正龙摩罗施展而出的龙吟，还要恐怖的多。

<br/>　　这还是在场魔族，几乎都是魔族之中的顶尖强者，换做是一些实力稍弱的魔族来，可能在这一声龙吟之下，都会会被活活震死。

<br/>　　楚风眠冷哼一声，也不废话，悍然出手。

<br/>　　血魔王看到楚风眠直接出手，脸色也是无比的愤怒，他也是一尊魔王，什么时候被人如此轻视，当即出手打出一道血光。

<br/>　　灵力爆炸，楚风眠的龙爪，横扫无敌，轰然落下的一刻，那血魔王打出的血光，被直接碾碎。
