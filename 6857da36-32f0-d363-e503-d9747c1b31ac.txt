　　在楚风眠的身躯中，一千二百九十六滴远古战龙精血，同时爆发而出。
<br/>　　瞬息之间，他的身躯便是化为了远古战龙的样子。
<br/>　　在这其中的一道道鳞片之上，却是蕴含有着几分特殊的血痕。
<br/>　　周围弥漫着无数血气，看起来简直便是一尊浴血战龙。
<br/>　　“轰隆！”
<br/>　　战龙之爪一出，这无数武技，在这战龙之爪面前，尽数破碎。
<br/>　　在这大殿中布置的无数禁制，也是在这一爪之下，都彻底毁灭。
<br/>　　“噗！噗！”
<br/>　　那虚瑾跟虚无奇二人，皆是在这战龙之爪的力量之下，直接都被打飞了出去。
<br/>　　口中吐出的无数鲜血，身躯砸在大殿上，才停了下来。
<br/>　　这战龙之爪力量，已经是大半打在了这大殿的禁制上，这二人如今才只是吐血重伤。
<br/>　　不然的话，以现在楚风眠的力量全开之下，一道战龙之爪，足以是将这二人，都给彻底灭杀。
<br/>　　“此人的力量，怎么会恐怖到如此程度？”
<br/>　　虚无奇无论如何也想不明白，楚风眠的实力怎么会达到如此恐怖的地步。
<br/>　　他自然看得出，楚风眠并没有隐藏境界，楚风眠的身上没有任何一点空间法则的存在，的确就是御风境武者。
<br/>　　甚至是还未踏入圣者的蝼蚁。
<br/>　　但是就是这样一尊蝼蚁，居然是一招化解了他的全部杀招，反倒是将他打重伤，吐血。
<br/>　　足以跨越两个大境界战斗，这已经不是天才可以形容了，这简直已经不是人了，是妖孽。
<br/>　　“血祭人间！”
<br/>　　虚无奇的眼神中，露出几分狠色。
<br/>　　突然开口，大吼一声，在这虚家之中，却是突然间传来无数惨叫声。
<br/>　　楚风眠一眼望去，这虚家中的无数弟子，皆是疯狂惨叫，然后他们的身躯，一下子全部都爆炸开来。
<br/>　　爆炸之中的无数血气，都是疯狂的飞了过来，全部被这虚无奇，统统吞噬。
<br/>　　“好狠的手段，居然是献祭族中无数族人，来获取力量。”
<br/>　　看着虚无奇，楚风眠眼神中都是露出几分寒意。
<br/>　　这虚无奇的手段，实在是狠毒。
<br/>　　这虚家的无数弟子，往日的修炼中，都是被种入了魔血。
<br/>　　如今虚无奇这一招，直接便是将他们所有虚家弟子身躯中的魔血，统统释放出来，将所有虚家弟子全部献祭。
<br/>　　吞噬了整个虚家弟子的生命，来恢复了他的力量。
<br/>　　这等手段，绝对称得上是狠毒。
<br/>　　“哼！大丈夫做事，向来不计手段，家族中的这些废物，死了就死了。”
<br/>　　虚无奇看着楚风眠，眼神中尽是贪婪道。
<br/>　　“小子，你的身上肯定有着大奇遇，杀了你，得到了你的奇遇，我们虚家还可以再次振兴。”
<br/>　　“十二血神！给我出来！”
<br/>　　虚无奇大喝一声，凝聚无数血气，在他的身边，纷纷凝聚出无数人影。
<br/>　　足足是十二尊血神，突然出现。
<br/>　　这每一尊血神的实力，都有着生死境一重的实力，在加上虚无奇的实力，本身便是已经踏入到了生死境的范畴。
<br/>　　一瞬之间，一下子在场便是出现了十三尊生死境强者。
<br/>　　“杀！”
<br/>　　这十二尊血神，出现的一刻，纷纷出手。
<br/>　　这十二尊血神，根数同源，同时出手，配合起来，亲密无间，比起十二尊生死境强者，一起出手，都要厉害许多。
<br/>　　“就算是你唤出十二尊血神来，也没有任何用处，这些血神，都是由最为精粹的血气凝聚完成的，正好可以融入到我的远古杀神之中去，为我增添一份力量！”
<br/>　　楚风眠大步踏出，一下子向着那十二尊血神，直接冲了过去。
<br/>　　他大手一挥，背后的远古杀神同样一股血气，猛然爆发出来，化为了一道血色大掌印。
<br/>　　一掌对着其中一尊血神，直接砸了过去。
<br/>　　这一尊血神，感觉到楚风眠出手，仿佛也是察觉的了危险，同样是一掌，打了出去，跟楚风眠相对。
<br/>　　“轰！”
<br/>　　这一尊血神的一掌，直接便是被化解，同时剩下的力量，也是向着那血神猛然砸了过去。
<br/>　　“砰！”
<br/>　　这一尊血神，在楚风眠巨大无比的力量面前，直接便是被炸开，化为了一团血雾，被楚风眠背后的远古杀神直接吞噬。
<br/>　　“爽！”
<br/>　　楚风眠长啸一声，这一尊血神被吞噬，简直就像是吞噬了一尊那虚家老祖一样的人物的全身血气。
<br/>　　如此精粹的血气，全部都是被吞噬到了远古杀神之中，令楚风眠的力量，都节节攀升起来，令他兴奋到了极点。
<br/>　　他的眼神，顿时又是落在了那剩下的十一尊血神身上。
<br/>　　只要是将这十一尊血神，全部吞噬殆尽的话，楚风眠的实力便是可以再进一步，距离突破神力境，指日可待。
<br/>　　“你！”
<br/>　　看着一尊血神如此轻而易举的便是已经被楚风眠毁灭，那虚无奇脸色变得僵硬无比。
<br/>　　这十二尊血神，可是他献祭了整个虚家无数弟子的，才得到的最后力量。
<br/>　　若是这十二尊血神，都不是楚风眠的对手的话，他可就真的完了。
<br/>　　“布阵！合击！血煞灭佛大阵！”
<br/>　　虚无奇大喝一声，顿时跟那十一尊血神，站在一起。
<br/>　　其中一尊血神已经被楚风眠灭杀，他便是也替代了一尊血神的位置，以十二尊生死境强者的力量，布置出了无上杀神。
<br/>　　虚无奇的身上，一口口精血，直接喷了出来，燃烧着其中的力量。
<br/>　　那十一尊血神，也是同时燃烧起来。
<br/>　　阵阵血气，化为了一道道飓风，猛然布阵，强大的力量，顿时向着楚风眠压迫而来。
<br/>　　“居然用出了血煞灭佛大阵。”
<br/>　　楚风眠看着空中凝聚而出的大阵，也是不由的默念一声，眼神中露出几分认真。
<br/>　　这血煞灭佛大阵，可是真正的杀招，远古宗门血煞门的绝技。
<br/>　　若是十二尊血神，同时出手的话，这一招的威力，足以是将生死境三重的强者，都可以灭杀。
<br/>　　十一尊血神，同时怒喝一声，纷纷燃烧起来，身躯猛然破碎。
<br/>　　所有血气，都是在空中，开始凝聚出来，化为了一条巨大的血气长河。
<br/>　　刹那之间，楚风眠的身躯，便是就被卷入到这血气长河之中。
