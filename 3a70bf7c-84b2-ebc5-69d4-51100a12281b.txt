　　入夜，萧逸停止了修炼。
<br/>　　透过窗沿，看着天际上的漫天繁星和月亮。
<br/>　　繁星、月亮，与地球的并没有什么两样。
<br/>　　唯一不同的，只是自己的灵魂。
<br/>　　或许，还有一样。萧逸看着推门而进，手里捧着饭菜的依依，笑了笑。
<br/>　　“少爷，吃饭了。”依依甜甜地说道。
<br/>　　“嗯。”萧逸点点头。
<br/>　　经过一天的修炼，体内真气越加浓郁，不过还未突破凡境三重。
<br/>　　武道修炼，非常艰难。
<br/>　　资质不佳的武者，终其一生都难以突破到后天。
<br/>　　哪怕是萧家，也有许多成年族人还处在凡境。
<br/>　　一旦突破到后天，就会自动晋升为家族执事，在家族中有一定的话语权，仅次于长老。
<br/>　　而执事，偌大个萧家也不过数十人罢了。
<br/>　　如果今天不是有那三颗淬体丹，恐怕萧逸晋升凡境二重还需要更多的时间。
<br/>　　虽然他拥有紫色武魂，但天地间的灵气毕竟是很稀薄的，远远比不上直接吸收丹药力量的速度。
<br/>　　“要是有更多的淬体丹就好了。”萧逸暗想着。
<br/>　　当然，也仅仅是想，淬体丹虽然只是一品丹药，但价格不便宜，足足十两一枚。
<br/>　　一般的普通家庭，三口之家，一个月的生活费都用不了五两银子。
<br/>　　哪怕是萧家这种大家族的子弟，一个月也就二十两零花钱罢了。
<br/>　　也就是说，以自己的钱，一个月都买不了两颗淬体丹。
<br/>　　“按这个速度下去，突破到凡境三重，起码需要四、五天。”萧逸有些苦恼。
<br/>　　如果他的想法被其他萧家子弟听到，一定会被大骂变态。
<br/>　　要知道，哪怕是萧家子弟，在凡境中要突破一重，都起码需要数月以上，乃至更久的时间。
<br/>　　就算是萧家有名的天才萧若寒，拥有强大的黄阶武魂，从凡境二重突破到三重，也足足用了一个月。
<br/>　　武魂之所以决定武者的未来，最大的体现就是吸收天地灵气的速度，从而影响了修炼快慢。
<br/>　　越是高级的武魂，修炼速度越快。
<br/>　　“依依，你干嘛？”萧逸忽然看到依依转身离去，喝住道。
<br/>　　“少爷，我回厨房吃饭阿。”依依怯生生地回答道。
<br/>　　以前的萧逸，从不允许依依同台吃饭。而且，厨房里只会剩下些最差的剩菜剩饭给依依。
<br/>　　依依总是吃不饱。
<br/>　　“坐下，一起吃。”萧逸道。
<br/>　　“可是，少爷你说过，依依长得丑，和你一起吃饭会影响你的心情，所以....”
<br/>　　“让你坐下就坐下，废什么话。”萧逸呵斥了一句。
<br/>　　依依怯生生地坐了下来，脸上尽是不安之色。
<br/>　　“夹菜吃。”萧逸见依依蒙头吃着白饭，说道。
<br/>　　“哦。”依依害怕地看了萧逸一眼，发现萧逸没有看他，这才小心翼翼地夹了点菜。
<br/>　　“也夹肉。”萧逸差点没气死，依依真地就只夹青菜。
<br/>　　依依虽然有些天然呆，但并不是迟钝。
<br/>　　相反，小时候的依依冰雪聪明，伶俐可人。
<br/>　　只不过，在萧逸面前，她总是小心翼翼罢了。
<br/>　　忽然，萧逸放下了手中的碗筷，淡淡地看了眼依依，道，“给你十秒钟，把剩下的饭菜吃光。”
<br/>　　“阿？”依依愣了愣，没多想，立刻狼吞虎咽起来。
<br/>　　十秒刚过，碗中的白饭已经清空，但桌上的菜还有不少。
<br/>　　“剩下的不要了，收拾一下回厨房吧。”萧逸的语气变得带有命令性。
<br/>　　“嗯。”依依口中还有一大口饭没能咽下，只能含糊地嗯了一声，而后快速收拾碗筷。
<br/>　　离开时，她有些伤心。
<br/>　　“被少爷赶出来了，是我影响了少爷的心情吗，少爷没吃多少口饭呢，少爷今晚会饿肚子的吧。”依依虽然心里有些难过，但还是第一时间想着萧逸的温饱。
<br/>　　....
<br/>　　房内，萧逸忽然站了起来，伸了个懒腰。
<br/>　　“这个世界的同行，还是第一次遇到阿，只不过，看起来手段很烂。”萧逸嘴角咧出一丝冷笑。
<br/>　　“出来吧，在房顶上看了这么久，不累吗？”萧逸似是在自言自语。
<br/>　　嗖的一声，房顶上一道人影跳了下来。是个中年男子，穿着一套夜行衣，脸上有一条狰狞的刀疤，满脸杀意。
<br/>　　“你发现了我？”中年男子疑惑地问道，有些惊讶。
<br/>　　萧逸没有回答，直接问道，“你是什么人？”
<br/>　　中年男子冷笑一声，道，“一个凡境二重的废物罢了，老子便大发慈悲告诉你，免得你死了都不知道死在何人手中，老子叫杨伏。”
<br/>　　“杨伏？”萧逸的记忆中有这个人的名字。
<br/>　　杨伏，绰号‘毒匕’，凡境九重武者，紫云城有名的杀手。
<br/>　　此人行踪诡秘，手段狠辣，为了钱，他可以为雇主暗杀任何人，手上性命累累。
<br/>　　“原来是毒匕，就你这隐匿手段，竟然还能成为有名的杀手，啧啧。”萧逸不屑地说道。
<br/>　　确实，在萧逸这个曾经的杀手界王者眼中，这个杨伏的隐匿手段极为粗糙低下。
<br/>　　吃饭之时，也是杨伏刚刚来到房顶埋伏时，萧逸就已经发现了他，这才找了个借口让依依离开。
<br/>　　换了以前，这种不入流的杀手，萧逸一巴掌就能弄死他。可现在，就很麻烦了。
<br/>　　“谁请你出的手？”萧逸问道。
<br/>　　“将死之人，何必多问。”杨伏冷笑一声。
<br/>　　“你不说我也知道，想要我命的人，无非就那么几个。而出得起钱请动你这个有名的杀手，就只有萧若寒了。”萧逸道。
<br/>　　杨伏皱了皱眉头，有些吃惊，道，“看来，萧家有名的废物并不像表面上那么简单阿。”
<br/>　　他虽然没有明说，但其实已经默认了。
<br/>　　“不过，也仅此而已了。收人钱财，替人消灾。既然收了钱，我就不能让你活着见到明天的天阳。”杨伏说罢，一步步朝萧逸走去。
<br/>　　他并不想节外生枝，毕竟这里是紫云城三大家族之一的萧家，家族里不乏先天境、后天境武者。
<br/>　　萧逸咪起了眼睛，看着逐渐靠近地杨伏，心里没有慌。
<br/>　　凡境二重与九重的差距极大。
<br/>　　凡境二重，体内那微末的真气，顶多够武者打出百来斤的力度。
<br/>　　但凡境九重，这可是只差一步就能迈入后天境的强者，轻松就能打出数千斤的力度，甚至更多。
<br/>　　换了别人，此刻肯定早已被吓得不知所措。
<br/>　　但萧逸见惯大风大浪，自然不会慌。
<br/>　　不能力敌，只能智取。凭自己的形意五绝还有战斗经验，加些手段，应该能暂且保命和拖延时间。
<br/>　　一旦这里的动静闹大，家族里的先天长老就能马上感知到并赶来。
<br/>　　“喝。”萧逸低喝了一声，摆出了战斗姿势。
<br/>　　“嗤。”杨伏不屑地嗤笑了一声，手中猛地出现了一把黑漆漆的匕首。
<br/>　　这就是他的武魂，‘毒匕’，等级为黄阶。
<br/>　　拥有器武魂的武者，能够消耗体内真气，将武魂的投影暂时凝聚出来，为几所用，发挥出极强的战力。
<br/>　　“杨伏，这里可是萧家，我乃是少家主，你杀了我，就不怕遭到萧家的报复？”萧逸冷声问着，眼中找寻着杨伏的破绽。
<br/>　　“报复？哈哈，你们这些大家族的子弟，生来就有家族庇护，有着大把的修炼资源，修炼得也快。”
<br/>　　“而老子这种普通武者，每天拼死拼活，甚至拿命去拼，才能得到修炼资源。”
<br/>　　“否则，以我的黄阶武魂，此刻早就突破到后天，甚至先天境了，哪至于还是凡境九重。”
<br/>　　“哼，你的命可值不少钱。拿了这次的报酬，我能买更多的丹药，很快就能突破后天。到时候逃之夭夭，你萧家也拿我没办法。”
<br/>　　杨伏早就心里扭曲，对萧逸这种家族子弟充满了仇恨和敌视。
<br/>　　“放心，我的毒匕，毒性很强。只要沾上一些，你会死得很快，不会痛很久的。”杨伏阴恻恻地笑着。
<br/>　　他已经来到萧逸一米之前，他一直都自信十足，要杀一个凡境二重，对他来说轻而易举。
<br/>　　“去死吧。”杨伏的匕首狠狠朝萧逸刺去。
<br/>　　萧逸目光一凛，身体诡异地扭动了一下，堪堪躲过了刺杀。
<br/>　　形意五绝之一，蛇形。
<br/>(本章完)
