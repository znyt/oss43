　　鬼王祭拜完毕，起身后见慕清霄怔怔出神，道：“慕公子，这石壁上所记载的就是天书，你可尽情阅览，秘洞中机关无数，万某就先告辞了。”
<br/>　　说罢，鬼王就不再停留，瞥了眼一旁的陆雪琪，见她手中的蓝色仙剑，脚步微微一顿，最后默不作声的离开秘洞，石门也渐渐关闭。
<br/>　　“叮，成功收集五卷天书，任务完成3/5”
<br/>　　脑海中响起系统的提示声，慕清霄微微一笑，并不准备离开，而是准备参悟一番，上次在万蝠古窟中参悟，收获不小，五卷天书，其中恐怕隐藏着不同的天地至理。
<br/>　　回头看向陆雪琪，温柔道：“雪琪，天书中隐藏着天地至理，对修行有莫大的帮助，我们一起参悟。”
<br/>　　闻言，陆雪琪看了看石碑上的金色大字，微微点头，于是，两人相依盘坐而下，掌心向天，心神守一。
<br/>　　半个时辰后，陆雪琪睁开美眸，神色有些古怪，她心神在石碑上停留许久，并未发现其中有何奥妙，可是看自己的丈夫却深陷其中......
<br/>　　随着时间的推移，慕清霄浑身一震，一股恐怖的气息在空中荡起一阵涟漪，向四周扩散，随后消失不见。
<br/>　　元婴四重巅峰！
<br/>　　睁开星辰一般的眼眸，慕清霄嘴角牵起一丝笑意，天书蕴含天地至理，绝对不是本位面的东西，将天书留在此处的人，修为又是何等恐怖。
<br/>　　感受一番自身的气息，慕清霄发现，修为再次得到了提升，而且锐利的气息也收敛了起来，收放自如，转过身来，四目相对。
<br/>　　“你的气息又有了变化，难道天书当真如此玄妙？”
<br/>　　陆雪琪苦笑一声，气息的变化不可能作假，美眸温柔的望着慕清霄，自己的丈夫想必是在天书中有所领悟，但她连参悟天书的资格都没有。
<br/>　　慕清霄微微点头，看来天书也不是所有人都能够参悟的，连元婴境界的道玄都无法参悟，筑基修为陆雪琪不能参悟，也是情理之中。
<br/>　　“别气馁，天书虽含有天地至理，但对于现在的你来说，并非好事，犹如拔苗助长，修为还需一步步提升才行，聚时，你自然会理解这些道理。”
<br/>　　慕清霄嘴上虽然这么说，但心里却有些想法，想要参悟天书，恐怕需要极高的悟性才行，他能够参悟天书，估计是因为他的资质已经不属于人类的层次。
<br/>　　“嗯，好。”
<br/>　　陆雪琪虽然有些失望，但听慕清霄如此安慰，心情也大好起来。
<br/>　　拍了拍白衫上的灰尘，慕清霄站起身来，道：“天书已经参悟，我们也离开吧。”
<br/>　　洞门开启，两人前脚刚踏出秘洞，一道紫色的倩影就走上前来，道：“慕公子，鬼王已经准备好晚宴，请公子与陆姑娘前往。”
<br/>　　“也好，带路吧。”
<br/>　　天书到手，慕清霄心情也是上佳，天色已经不早，而且鬼王已经设好晚宴，幽姬恐怕在这里等候多时，既然别人诚心招待，他们也不好拒绝，更没有在野外露宿的理由。
<br/>　　晚宴上自然客套话居多，青龙也不再给脸色，幽姬等人都颇为客气，慕清霄也是毫不吝啬的拿出美酒，一起畅饮。
<br/>　　见慕清霄一身剑意有所收敛，鬼王震惊不已，但也没有多言。
<br/>　　原本，在陆雪琪眼中，鬼王宗里都是可恶魔教妖人，如果不是慕清霄，她是万万不可能来此，也不敢来此，但经过半天的相处，她的心性也有所提升。
<br/>　　站在不同的角度，不同的阵营，所需要考虑与顾忌的也有所不同罢了，想到此处，陆雪琪的气息微微变化，距离金丹又近了一步，慕清霄眼中闪过一丝欣慰。
<br/>　　晚宴上，鬼王一杯美酒下肚，朗声大笑，道：“慕公子真是海量，今日天色已不早，我已经命手下准备好上等客房，若是不急，在门内暂住一宿也无妨。”
<br/>　　“如此，便多谢鬼王了。”
<br/>　　慕清霄也正有此意，现在还剩下最后的两卷天书，其中一卷在天音寺，与鬼王宗相隔甚远，也不急着一时，还有最后一卷在天帝宝库中。
<br/>　　根据原剧中记载，天帝宝库是在十年后才会开启，慕清霄并没有开启天地宝库的办法，恐怕也无法强行破开禁止，能做的唯有慢慢等待。
<br/>　　竖日清晨，慕清霄与陆雪琪一早就起床，简单的洗漱过后就告别鬼王宗。
<br/>　　鬼王站在山巅，望着慕清霄离去的方向，道：“你们觉得，此人如何？”
<br/>　　青龙不语，幽姬的声音响起，道：“深不可测，不可为敌，却不妨碍我们对付青云门。”
<br/>　　闻言，鬼王不再说话，只是神色深沉，不知道在思考何事。

<br/>　　天音寺位于须弥山上，天音寺与青云门、焚香谷为正派三大门派。
<br/>　　天音寺为佛门修真，修习佛门至高法门——大梵般若，依靠佛法普渡众生，门内僧人注重自身修行，以佛门功法强身健体，寂灭入定。
<br/>　　法术以防守见长，佛门真言中蕴涵的力量可以使人固本培元，不受外力所侵害，大慈悲的力量也可以救济世人，恢复气血。
<br/>　　当兽妖涂炭生灵的时候，他们更会挺身而出，以佛法降妖除魔。
<br/>　　悠悠晨钟，沉沉暮鼓，须弥山沐浴在缥缈云气之中，从初升的旭日到傍晚的残霞，天际风云变幻，白云苍狗滚滚而过，时光终究不曾为任何人而停留。
<br/>　　天音寺雄伟壮丽，雄峙于须弥山上，仿佛一位慈悲的巨人望着世间，无数的凡人在清晨从四面八方汇集而来，对着佛庙殿堂里的神像顶礼膜拜，诉说着自己或喜或悲的心愿，祈求着神明的保佑。
<br/>　　千万人来此，汇聚，千万人散去，离别，一日复一日，从来不曾改变，聚聚散散般的岁月，只有那庙中神佛金身神像，殿堂前不灭明灯，袅袅烟火，看尽了世事沧桑。
<br/>　　就在这时，一道剑光划破天际，最终落在须弥山脚下：“这里就是天音寺吗，真是雄伟，相比青云门通天峰也要豪华的多，无论是何处，佛门果然都是最有钱的。”
