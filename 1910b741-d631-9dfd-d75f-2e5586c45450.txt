<br/>    古仙历四千二百年八月初五，燕山一统十万大山的消息传出，震惊整个大陆。
<br/>    
<br/>        其实陈明所在的这一片大陆有些奇怪，因为这一片大陆并没有名字，而没有名字的原因，可能和古仙门的商议效率有一定的关系。
<br/>    
<br/>        众多修行者都在议论关于这个最近崛起的势力，这个新崛起的势力，足够让人给予他足够的关注，因为这至少意味着，有着一位新的大君出现，而陈明这个名字正式出现在所有修行者的视野之中，可谓是名传天下。
<br/>    
<br/>        “燕山，以前竟然从来没有听说过，看来，是出现了一位新的大君了。”
<br/>    
<br/>        “一位新君的崛起，一位大君的陨落，还真是风云变幻啊！”
<br/>    
<br/>        “霸主的更迭，向来都是如此！”
<br/>    
<br/>        “不过妖族而已，有什么值得注意的，妖族，在这片大陆之上，算什么东西！”
<br/>    
<br/>        古仙历四千二百年八月初八，时隔三天之后，魔宗大宗主无疆公开宣言道：“燕山，我取你一刀，你折我一刀，两不相欠！”
<br/>    
<br/>        如果说前面的消息不够震惊所有人，但是这一条消息，魔宗大宗主无疆的一句话，却是一石激起千层浪，“不是吧，这个燕山到底是什么样的存在，居然连魔宗大宗主都说出这样的话来！”
<br/>    
<br/>        “魔宗大宗主，这句话到底是什么意思？”
<br/>    
<br/>        “什么我取你一刀，你折我一刀，难不成燕山其实和魔宗有密切关系，燕山一统十万大山之后有着不可告人的交易？”
<br/>    
<br/>        “燕山，到底和魔宗是什么关系？”
<br/>    
<br/>        “燕山，这个最近崛起的大势力，到底是什么样的存在？”
<br/>    
<br/>        七天之后，古仙门召开群仙大会，这一次四大古仙门宣布的有两件事。
<br/>    
<br/>        第一件事，便是通报整个大陆修行界新晋大君，“我四大古仙门见证，在这暗无天日的时代，两位新的大君崛起，必将昭示我仙门正道气运昌隆！”、
<br/>    
<br/>        “两个月之前，魔宗运用传送阵，由魔宗十六宗主，解千愁，带领三十王级军团，伏杀我正道所有有望大君的天骄，我正道危机万分，适逢其时，燕山之主，陈明，力敌诸魔，誓死捍卫天地正道，斩诸王，屠邪魔，杀人证道，位临大君，万剑弑魔，斩杀解千愁，还我正道朗朗乾坤，当新君降临的时刻，山川和河流，微风和浮云，都应呼唤他的名，他是黑暗之中唯一的曙光，天地虽暗，一光破之，陈明，封号：曙光！是为曙光大君！”
<br/>    
<br/>        “同样在两个月之前，九幽寒泉之中，仙魔大战之际，群星仙门紫霞，与曙光大君共抗魔宗，斩魔证道，位临大君，当新君降临的时刻，太阴和太阳，群星与苍穹，都在呼唤她的名，她是无尽黑夜之中最亮的一抹光彩，她照耀了整个夜空，黎明破晓之际，日月齐辉，紫霞，封号：晨星！是为晨星大君！”
<br/>    
<br/>        “共呼君名！”
<br/>    
<br/>        “曙光！晨星！”
<br/>    
<br/>        “曙光！晨星！”
<br/>    
<br/>        “曙光！晨星！”
<br/>    
<br/>        “恭贺新君！”
<br/>    
<br/>        “大君降临，寿比千秋！”
<br/>    
<br/>        而古仙门的第二件事，便是宣布另外一个消息，其实古仙门不宣布，大陆之上的势力也有了一丝猜测，魔宗做了这么大的事，古仙门不可能没有反应，“既然魔宗想要看看我仙门正道的实力，就让他们仔细看看，自今日起，大君之下，诛魔令通行整个大陆，我仙门正道，共诛邪魔！”
<br/>    
<br/>        之前的战争还持续在入道境，而且并没有放在明面上，而现在，战争级别已经是直接提升到了王级了。
<br/>    
<br/>        大君毕竟是整个大陆之上最强大的战力，如果大君一旦动手，那么就是一场足以毁灭整个大陆的战争，即便是魔宗，听到正道的消息，也是有所克制，下令大君不可介入战场。
<br/>    
<br/>        大君参战，无论是胜是败，对于整个大陆都是一场灾难仙门正道和魔宗都达成了共识，他们现在还没有准备好发动这样的一场大战，他们需要更多的时间去酝酿，所以现在，战场就出现在了王级之中，这是一场诸王之战，会猎于野。
<br/>    
<br/>        而燕山，似乎也正式宣称，他们是仙门正道。
<br/>    
<br/>        毕竟燕山山主的封号，是古仙门裁决的。
<br/>    
<br/>        不过他们听到陈明和紫霞仙子的封号，总有一些猜测，“曙光，晨星，莫名感觉般配。晨星的光芒，不就是曙光吗？”
<br/>    
<br/>        “难道说，曙光大君和晨星大君有一腿？”
<br/>    
<br/>        “在九幽寒泉之中共同突破到大君，那么长的时间，还有一腿？七八腿都得有了吧？”
<br/>    
<br/>        “如果只有一腿，那么曙光大君得有多持久？”
<br/>    
<br/>        紫霞仙子现在正在妖城之中，听到关于自己和陈明的封号之时，莫名一怒，“这两个封号，到底是什么意思？”
<br/>    
<br/>        苏青阳道：“这你去问你们家群星老君啊！”
<br/>    
<br/>        雪自悟也是道：“别看我，你可能不知道，群星老君之前拟定的封号是什么。”
<br/>    
<br/>        紫霞仙子问道：“是什么？”
<br/>    
<br/>        雪自悟道：“群星老君之前是这么说的，你是群星仙门的，乃是群星之中最闪耀的一颗，所以你的封号是星娘，星辰的星，新娘的娘。而宗师突破的时候，有着群星降临，所以他的封号是星郎，星辰的星，新郎的郎。最后还是其他三位老君觉得太过了，所以改成了曙光晨星。”
<br/>    
<br/>        紫霞仙子怒道：“老君实在是太过分了！”
<br/>    
<br/>        卓青瑶听到这个封号的时候心中一惊，而且，现在外面的传言可是让她紧张不安，师父到底是怎么想的，这个时候，卓青瑶忽然想起了师父曾经给她上过一课，那一课的主要内容是：男人没有一个好东西，坐怀不乱是不存在的。
<br/>    
<br/>        于是卓青瑶很快登上燕山，想要去看看陈明听到他和紫霞仙子封号之后的变化，当卓青瑶来到陈明的身前之时，陈明此刻正在深思，卓青瑶小心地靠近陈明，以免打搅了陈明，想要偷听陈明说什么，好推测出陈明对于这两个封号的反应。
<br/>    
<br/>        正在这时，卓青瑶忽然看到陈明站了起来，“你说两不相欠，可我并没有这么说，也同样没有这么觉得，他日我与你对阵在两军之前，我会和你好好算算这笔账！”
