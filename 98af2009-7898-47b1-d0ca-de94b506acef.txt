　　第568章
<br/>　　“我与你往日无怨，近日无仇，夫子何必苦苦相逼。”
<br/>　　“你将公主万箭穿心，说什么无仇无怨，今日我绝不许你离开万卷山。”
<br/>　　二人已打在一起，程大雷一柄快剑，追星逐月，当真令人望着生畏。而夫子用的同样是快剑，快到令人发指的程度。谁能想到，这年迈苍苍的老者，竟有不输年轻人的精力和杀气。
<br/>　　夫子越打性子越起，似乎一堆老炭被重新点燃，发光发热。打法火爆，和往日慈眉善目的样子大不相同。
<br/>　　程大雷是越打越寒心，真不能小看了这个老怪物。公主……万箭穿心，莫非他说的是白衣老尼？但那老尼姑什么时候成公主了。
<br/>　　程大雷当真不知道十九国的旧事，估计满天下知道的人也没几个。但这些对程大雷来说并不重要，程大雷也并不关心。
<br/>　　他关心的还是如何脱身。
<br/>　　二人交手十余回合，程大雷渐渐摸清了夫子的攻击节奏，已在战斗中占得上风。夫子毕竟老了，他偏偏要和程大雷比快剑，这不是找死么。哪怕曾经你的剑当真很快，但现在已不是曾经，你的剑再快，能敌得过时光么。
<br/>　　再有十回合，程大雷有把握将夫子斩于剑下。
<br/>　　问题是，书院会不会给他十回合的时间？
<br/>　　答案是：不会。
<br/>　　张明理已带人到了，看到夫子亲自与程大雷动手，当场就是一激灵。夫子这般年纪，万一有个闪失，死在恶贼的剑下，对书院可是难以承受的损失。
<br/>　　“布阵。”
<br/>　　张明理立刻带人将程大雷包围，这可是难得的机会，毕竟，程大雷是个很狡猾的人。但现在他终于被困住了，张明理不觉得程大雷有冲破剑围的可能。
<br/>　　“夫子，您没事吧？”张明理关心的问。
<br/>　　夫子面目涨红，胡子都竖了起来：“什么事，我能有什么事。”
<br/>　　张明理一怔，夫子说话的语气，可和从前不同。
<br/>　　夫子气喘吁吁，打这一场，他骨子里的火爆脾气被激了出来。但他不得不承认，他的年纪已经大了，况且几十年不出剑，有些技艺已经熟悉。亏得张明理出现得及时，不然的话，十招之内，他绝对会死在程大雷剑下。
<br/>　　而此刻，书山剑阵已经合围，将程大雷、刘发财以及那头黑牛围在当中。
<br/>　　情况不是很少，书院虽然人手空虚，又被程大雷杀了不少。但是，依旧凑齐二十一人，将程大雷困在剑阵中。
<br/>　　很凌厉的剑阵，防守得当，配合默契，这二十一人都有不错的剑法。令程大雷腹背受敌，一时狼狈不堪。
<br/>　　破阵之法有二，一是一力降十会，任你千般变化，我自一巴掌拍碎；二就是知道剑阵的运转变化，提前站好位置，从内部打破。
<br/>　　这两种法子，程大雷一种都做不到。
<br/>　　终于，他的情况和燕不归一样糟糕了。
<br/>　　山门前，倒下很多具尸体，火把将四周照亮，地上的血是惨红的。宋寒之主持剑阵，每死一人，他胸中便添几分怒火。
<br/>　　死的人太多了，书院承受了它从未承受过的损失。
<br/>　　不过，燕不归已经撑不了太久了，他身上有细细密密的伤口，有些小伤口因为时间流逝已经凝结，但又因为战斗裂开，流出更多的血。
<br/>　　这头狮子的血，快要流干了。
<br/>　　他节省着体力，不再大喊大叫，动作也极力保持干净，以节省不必要的体力浪费。但他的眼睛依旧很亮，只要你注视他的眼睛，就会觉得他永远不会倒下，他会永远战斗下去。
<br/>　　但，这也仅仅是感觉上……
<br/>　　“让张六出来见我。”燕不归并非不知道死期将至，死他不怕，但他想在死亡前，为黑贞做些事。
<br/>　　该死的人太多了，自己能力气快用尽了。如此，也只能退而求其次，只把张六杀了。
<br/>　　宋寒之不知该如何回答，因为张六真的不在。否则，以张六的火爆性子，不可能忍受宋寒之在书山放狂。
<br/>　　“怎么，做缩头乌龟麽。”燕不归仰天大吼：“张六，出来。”
<br/>　　“那个……”宋寒之莫名有些尴尬：“他真不在。”
<br/>　　“书院准备将他藏多久，我要再杀多少人，他才肯出来。”燕不归气喘吁吁。
<br/>　　宋寒之真的有些头疼了，和戎族这种蛮子，当真是无法沟通呐。
<br/>　　他再次指挥剑阵，转防守为攻击，是时候结束了。
<br/>　　而程大雷目前的情况，也是同样糟糕。他不仅要想办法破阵，还得保护刘发财。哪怕就算冲出剑阵，还要面对夫子这老贼。
<br/>　　他像完全变了个人似的，握着剑，在剑阵外来回转圈，颇有种：你出来啊，你出来让我杀了你的感觉。
<br/>　　程大雷竟觉得，这剑阵是在保护自己。
<br/>　　但他已开始受伤，开始流血。而这表示，程大雷并没有在剑阵中自保的力量，当他开始受伤，他就会受更多的伤，当他开始流血，他就会一直将血流干。
<br/>　　或许，夫子没有亲手杀死程大雷的机会了，因为程大雷根本不能活着走出剑阵。
<br/>　　一盘棋局，已进行到最后，棋盘上的程大雷大势已去，只有一枚車，一座没有炮台的炮。任谁旁观这棋局，怕都会摇摇头，希望程大雷能够弃子认输，当然，他没有重来的机会。
<br/>　　而在角落里，还有一枚被所有人忽视的『卒』，从始至终，他只向前动了一步，然后就被所有忘记了。甚至程大雷都怀疑他是不是黑棋刷红了假扮的。
<br/>　　如此说来，下棋方面程大雷实在是个庸手，因为真正的高手，绝不会忽略任何一枚棋子。
<br/>　　山中小亭，楚云生一动不动。
<br/>　　万卷山乱糟糟，这里竟成了少有的不被打扰之所在。当然，也有乱糟糟的声音传来。
<br/>　　他知道燕不归快不行了，知道夫子已亲自出马，知道夫子亲自出马后，已经将程大雷困在。
<br/>　　情报称不上多，楚云生很认真想。如果他有些武艺，也想帮程大雷一帮，但他不过是枚微不足道的卒。
<br/>　　在合适的时候，出现在合适的地点，一枚小小的卒子也能成为翻盘的力量。
<br/>　　可合适的时机合适的地点，究竟在哪里。
<br/>(本章完)
