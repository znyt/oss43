　　此时，慕清霄身处一座赤红色墙壁通道中，脚下是不知用何种材料炼制而成的精石踏板，建造在岩浆上不化。
<br/>　　精石踏板的缝隙间，赤红色岩浆流动，空间中散发着高温，似乎就是一处溶洞中。
<br/>　　行走在精石踏板上，约莫走了一分钟左右，脚步在通道的尽头停住，抬头望去，溶洞上面刻着龙飞凤舞的四个古篆字体。
<br/>　　极乐世界！
<br/>　　似乎是不认为有谁能进来，极乐世界的入口并没有布置任何的阵法，天然的岩浆屏障，在火鬼王心中便是最有效的阵法。
<br/>　　走进极乐世界，其中的温度依旧非常的高，估计也只有火鬼，才会喜欢如此闷热无比的地方。
<br/>　　极乐世界似乎是建造成了一座地宫，站在门口，慕清霄就已经看到远处密密麻麻，不知通往何处的通道。
<br/>　　要知道，这仅仅是入口而已。
<br/>　　就在这时，两名穿着黑色盔甲的鬼兵出现在视线中，慕清霄脸上牵起一抹弧度，真是想什么就来什么。
<br/>　　看到慕清霄后，两名鬼兵觉得有些奇怪，相互对视一眼。
<br/>　　陡然，两名鬼兵一惊，方才还在数米开外的白衣身影，不知何时，已经出现在了他们的眼前，他们竟没有反应过来。
<br/>　　回过神来，映入眼中的是一双猩红妖异的瞳孔……
<br/>　　望着两名眼神变成空洞的鬼兵，慕清霄嘴角勾起一抹笑意，食指与中指与鬼兵额头接触。
<br/>　　旋即，后者的记忆就被读取了出来。
<br/>　　极乐世界，又称火鬼界，火鬼界建造在数千米的地底，东南西北尽被岩浆包围……
<br/>　　根据鬼兵脑中反馈的信息，极乐世界地域广阔，方圆百里，而火鬼王的鬼王宫，则建在了极乐世界的最中心。
<br/>　　收回手掌，除了火鬼王的信息，慕清霄可没兴趣了解鬼兵的过往。
<br/>　　血轮眼隐没，空间轻微波动，慕清霄的身影消失后，两名鬼兵的双眼就恢复一片清明。
<br/>　　先是迷茫，回过神来，两名鬼兵面面相觑。
<br/>　　“我们刚才……是想干嘛来着？”
<br/>　　“记不起来了，不过，今天貌似是我们的值班日，别偷懒了，若是让上面知道，吃不了兜着走，这可是份肥差。”
<br/>　　……
<br/>　　……
<br/>　　约莫半刻钟后，慕清霄已经来到了极乐世界的中心地域，由于分岔口太多，弯弯绕绕，所以用了不少的时间。
<br/>　　整个极乐世界，结构与蜂巢有些类似。
<br/>　　中心地域的生命波动很稀少，似乎这里都是火鬼王领域范围，慕清霄进入这里后，就在也没有遇到别的鬼修。
<br/>　　双手结印，空间结界悄无声息的扩展而出，数息就将整个中心领域都笼罩在其中。
<br/>　　神识释放而出，转眼就找到了火鬼王的居所‘鬼王宫’，慕清霄身躯跨过空间，下一秒就出现在了鬼王宫外。
<br/>　　说好听点是鬼王宫，说难听点就是宽敞一点的溶洞，布置也比其他地方豪华一些，且戒备森严。
<br/>　　溶洞外立着一块石碑，石碑上印着‘鬼王宫’三字。
<br/>　　在慕清霄的神识中，除了鬼王宫中有一名合道六重的鬼修外，其他地方都散布着总共十余名练虚期，百名化神期鬼修。
<br/>　　火鬼王在众多鬼王中算不得最弱，但也算不上最强，实力徘徊在中等层次，否则也不至于龟缩在着极乐世界中。
<br/>　　若非阎罗鬼王，火鬼王想必也不会如此安份。
<br/>　　慕清霄实力足以碾压整个极乐世界，但他却不想为了火灵珠，惹来鬼界高层的追杀，犯不着这样冒险。
<br/>　　并未惊动任何鬼修，慕清霄的身影已经出现在了鬼王宫中，朝着那名合道期的气息掠去。
<br/>　　进入鬼王宫后，慕清霄将空间结界缩小，哪怕其中发生惊天动地的大战，外界也不会有任何人能够察觉到一丝异状。
<br/>　　锁定空间后，慕清霄也不再隐匿身形。
<br/>　　这时，溶洞中传来一道妩媚的女子声音：“是本王的美容汤熬好了吗？站在外面干什么？还不快点给本王送进来。”
<br/>　　慕清霄剑眉一挑，推开木门，大摇大摆的走了进去。
<br/>　　刚进入溶洞中，映入慕清霄眼帘的是一名身穿火红长裙，头戴金钗凤冠，仰躺在拉着红色帘幕床榻上的俏丽女子。
<br/>　　女子虽看似柔弱，令人怜惜，却是一名货真价实的鬼王，合道六重。
<br/>　　或许是没有得到回复，火鬼王发现有些不对劲，抬起臻首，却看到一名身着白袍，剑眉星目，风姿绰约的青年站在门口。
<br/>　　坐起身来，火鬼王并未恼怒，看向慕清霄俊逸的脸庞充满了欣赏：“呀，这是那位公子，闯进了奴家的住处。”
<br/>　　慕清霄面色淡漠，道：“火灵珠，你的性命，你可选其一。”
<br/>　　话语刚落，火鬼王面露微笑的俏丽容颜顿时凝固，凤眼眯了眯，溶洞中响起冷冽的声音。
<br/>　　“你不是极乐世界的子民，你是如何躲开本王视线，进入这里的？”
<br/>　　火鬼王有些好奇，除了天然的岩浆屏障，她可是布下了空间结界，将整个极乐世界都笼罩在其中，只要有东西触碰结界，画面就会传入她的脑海中。
<br/>　　面前的青年，显然不是极乐世界中的居民，却诡异的出现在了她的面前。
<br/>　　“火灵珠，你的性命，我给你十息的时间做出决定。”对于火鬼王的询问，慕清霄没兴趣回答。
<br/>　　闻言，火鬼王凤眼中杀意凌然，道：“无论你是谁派来的，劝你在本王没有完全被激怒前，滚出极乐世界。”
<br/>　　慕清霄充耳不闻，道：“你还有五息时间。”
<br/>　　“你！”
<br/>　　“三息。”
<br/>　　火鬼王拖着火红长裙站起身来，凤眼冷冽如风，语气如若冰霜：“既然你执意找死，本王就成全你。”
<br/>　　说罢，火鬼王玉手一挥，一团青色火焰就狂涌而出，瞬间席卷慕清霄全身。
<br/>　　眨眼间，慕清霄的身躯就被诡异的青色火焰包裹，看上去就像一个人形火团。
<br/>　　“真是不知死活，你彻底惹怒了本王，待地狱冥火将你的身躯焚化，本王在拘禁你的灵魂，让你求生不得，求死不能。”
<br/>　　火鬼王俏丽的面孔已经变得狰狞无比，狠辣的话语回荡在溶洞之中。
<br/>　　……
<br/>　　……
