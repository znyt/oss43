　　这一道彩虹，从这云羽天宫之中，突然延伸出来，一只延伸到最下方的一座高山之上，逐渐凝实。
<br/>　　这彩虹凝聚，便是化为了白玉一般的样子，七彩白玉，层层叠叠直接化为了一处台阶阶梯。
<br/>　　顺着这阶梯，便是可以从山峰之上，直接前方最上方的云羽天宫。
<br/>　　“这是？云羽天宫开启了？”
<br/>　　楚风眠看着天空之上的异变，脸色大喜，这云羽天宫的存在，果然不可能仅仅是就这样让人看着的。
<br/>　　其中的机缘，不是没有，只是时候未到罢了。
<br/>　　现在这机缘，终于到来了。
<br/>　　楚风眠的眼神之中都露出了几分火热。
<br/>　　这云羽天宫的传承，楚风眠必须要得到。
<br/>　　曾经的云羽天宫，便是九域巅峰的宗门，其中到底是留下有着多少宝藏，几乎无人可知。
<br/>　　但是只要的能够得到其中一点，绝对都是莫大的机缘。
<br/>　　“云羽天宫开启了，我们快点过去吧。”
<br/>　　寒月漓也是抬起头，看着这天宫上的异变，大声喊道。
<br/>　　“好，我们走。”
<br/>　　楚风眠毫不犹豫，他的身形一动，直接向着那山峰，催动宝车，直接便是冲了过去。
<br/>　　同时楚风眠也感觉到，在这天羽神域之中无数的强者，甚至是一些隐藏的强者，都是纷纷感觉到了云羽天宫的变化，一个个全部都是苏醒了过来，也打算去争夺。
<br/>　　在这天羽神域之中的无数武者，他们的最终目的，便也正是这云羽天宫，谁要是能够得到这云羽天宫，就是真正的一步登天。
<br/>　　云羽天宫的传承，可以说比起现在七大宗门都远远无法相提并论。
<br/>　　楚风眠跟寒月漓，坐在战龙宝车之上，直接向着那山峰飞去，一路上，楚风眠也足以看到了不少武者，都是向着这山峰之中赶了过去。
<br/>　　可以说是整个天羽神域之中的武者，所有人都全部赶了过去。
<br/>　　“这山峰，应该正是天羽神域的中心所在。”
<br/>　　楚风眠一眼便是看得出来，这山峰，实际上就是整个天羽神域的中心，也是最为高大雄伟的一座山峰。
<br/>　　之前楚风眠刚刚来到天羽神域的时候，也曾经是观察过这一道山峰，但是那个时候这一道山峰上，还是一直弥漫这一种七彩霞光，似乎是一种古老的禁制，阻止任何人的进入。
<br/>　　楚风眠观察了一下，也明白这七彩霞光，远远是他足以对付的了的，所以当时便是没有理会这山峰。
<br/>　　现在随着彩虹霞光从天而降，化为彩虹阶梯，这山峰周围的七彩霞光便是也都纷纷破碎开来。
<br/>　　看来这云羽天宫的传承，如今是真正引动开启了。
<br/>　　“站住！”
<br/>　　“都给我滚开！谁敢拦我谁死！”
<br/>　　就在这山峰下方，无数武者，已经是战在一团，互相争抢这，谁也不愿意让其他人，登上这山峰。
<br/>　　下方已经是一片混乱，无数武者都如同蝗虫一般，疯狂的战斗这，一个个都想要抢先一步，进入这山峰之中。
<br/>　　七大宗门之中所剩的，药王谷，古月峡谷，荒古门，天枢殿的弟子，也全部都在这混战之中，谁都想要杀出一条血路，直接上前。
<br/>　　除此之外，还有什么玄天宗，炎阳门，甚至是北邙学院的弟子，都在其中混战。
<br/>　　足足是有着数百人之多，这些人都是不知道多少年之前，来到这天羽神域之中的武者。
<br/>　　都是在等待这云羽天宫开启的时机，现在这云羽天宫发生异变，突然开启，他们自然一个个都是疯狂无比，都想要抢先一步，进入其中。
<br/>　　“如此混乱，怎么办？”
<br/>　　寒月漓看到这一团混乱，不由的开口道。
<br/>　　以他们二人的实力，若是陷入到了这混乱之中，想要在脱身都不容易，虽然没有性命威胁，但是却也是容易被拖延时间。
<br/>　　若是拖延任何一点时间，便是有可能这云羽天宫的传承宝藏被其他人所得到。
<br/>　　尤其是现在楚风眠，已经是看到了不少武者，都是从这混乱中冲了出去，直接向着山峰上冲了上去。
<br/>　　“不管他们，直接冲过去，谁挡谁死！”
<br/>　　楚风眠眼神中露出几分杀意，冷喝一声，直接催动战龙宝车，猛然一声龙吟咆哮。
<br/>　　整个战龙宝车便是化为了一道遁光一般，直接冲着人群中冲了过去，周围的灵力猛然爆发，将周围的武者，全部纷纷镇开，化为了一条道路，横冲直撞之间便是冲了上去。
<br/>　　“该死！”
<br/>　　“不能让他过去！”
<br/>　　混战之中的不少武者，都是看出楚风眠的意图，一个个眼神杀机涌现，现在让任何一个人过去，他们得到这云羽天宫传承的机会就会少上几分。
<br/>　　瞬息之间，足足是有着数十道武技，便是同时向着战龙宝车轰击过来。
<br/>　　“千里冰城！”
<br/>　　从战龙宝车中，寒月漓的身影猛然一动，站立起来，她的剑锋一动，就在面前无数寒冰，化为了一道巨大无比的冰城。
<br/>　　无数武技轰击在了巨大冰城上，其中的力量都被抵挡大半，而剩下的那些灵力，则是打在战龙宝车罗伞的七彩霞光上，甚至都无法撼动这战龙宝车。
<br/>　　面对无数轰击，楚风眠的战龙宝车便是俨然不动，随着楚风眠心神的催动之下，速度便是变的更加迅速许多，直接撞了过去。
<br/>　　“砰！”
<br/>　　挡在最前方的几尊武者，在这战龙宝车的撞击之下，顿时纷纷身躯爆炸开来，直接化为了一团血雾，全部被活活撞死。
<br/>　　而周围的那些武者，都在在这冲击之下，纷纷推后，不少人都是口中吐出了无数鲜血，受了不小的伤势。
<br/>　　“走！”
<br/>　　所有的武者，都被镇开，楚风眠面前的道路，便是再次畅通无阻，他动用战龙宝车，直接向着山峰上冲了过去。
<br/>　　“砰！”
<br/>　　楚风眠的战龙宝车，冲破人群，直接带这他跟寒月漓，一下子进入到了山峰之上，在他的面前，便是一条宽阔无比的道路。
<br/>　　但是踏上这道路的一刻，楚风眠便是感觉到了一种恐怖的压力，这一股股的压力，好像是要将楚风眠直接赶下这道路，赶下这山峰一样。
