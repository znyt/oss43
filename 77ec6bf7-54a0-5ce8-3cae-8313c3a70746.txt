　　想远了……叶少阳思绪又回到翠云讲述的事情上，说道：“后来呢？”
<br/>　　“后来，这对浙江夫妻打算几年后要走，想把店盘给我来做，当时我是很想跟我男人一起经营的，不过那时候我男人检查出肺痨病，为了治病，把积蓄都花光了，我要照顾他，也没时间再在饭馆里帮厨，回家去了，后来我男人死了，钱也花光了，再说我一个乡下女人，没个男人帮衬着，啥事也做不了的，这件事也就不想了……”
<br/>　　说到这里，翠云的目光也暗淡下去。
<br/>　　叶少阳听他说完心酸往事，在她手上拍了拍，安慰了一番，说道：“姐，如果现在有机会给你开饭馆，你还想不想经营，开一个比这还大的，我给你雇人帮工，你自己做菜。”
<br/>　　翠云一下子就怔住了，随即想到了叶少阳的意图，急忙摇头说道：“那不成，那是你的钱，是有来给你娶媳妇的。我可不能花。”
<br/>　　“我的钱，也有你的一半，再说我娶媳妇能用的了这么多钱吗。”叶少阳看出她其实是想要开个饭馆的，只是有些顾虑，于是又鼓励了一番。
<br/>　　翠云被他说的心中那个曾经的愿望又活了，认真想了想，还是摇头，说道：“姐说过，要给你管家的，除非你到时候不要姐了，不然我不放心你的。”
<br/>　　叶少阳笑了笑，内心更是坚定了自己的想法：自己毕竟不会久留在这个时代，如果将来走了，留下翠云一个人孤寡地生活，自己也是不放心的，如果她有自己的事业的话，那自己也可以放心地走了。不过现在还早，他没有将自己的想法告诉翠云，只是在心中默默记下了。
<br/>　　等了一会，陈三掀开包间的帘子走进来，身后跟着一个中年人，穿改良后的外形像长衫的道袍——这种道袍兼有道袍的功能，但是外形看上去更像长衫，走在街上不会太引人注目，以免暴露身份，是一般经常行走江湖的道士所穿的，就算是在百年之后，依然是道门的传统。
<br/>　　这道士很瘦，个头小，但是腰杆挺直，剑眉星目，鼻若悬胆，天圆地方，前庭开阔，一脸英气，天生福相，上唇还留着一抹小胡子，看上去透着几分诙谐。
<br/>　　居然是道门子弟，叶少阳立刻站起来，捏了子午诀，对方也很客气地还礼，然后陈三给双方介绍，先介绍了叶少阳，然后介绍了这个中年道士：“这位是天道宗的毛道长，正是他将那人形煞的主人废掉丹田，驱逐出了法术界……具体经过听毛道长给你讲吧，不过我们先吃，来尝尝这火锅。”
<br/>　　这时候水已经开了，两人坐下之后，陈三要了一壶酒，涮菜吃起来。
<br/>　　叶少阳品尝了一下，感觉跟百年后的火锅味道不太像，好像料子没有那么多，比较清淡，但是味道还是不错的。
<br/>　　终于吃到熟悉的火锅，叶少阳也大快朵颐，一边听这个毛道长讲述事情的经过。
<br/>　　毛道长简单讲了一遍，原来他有个入魔的师弟，被逐出师门之后，修炼邪术，在法术界劣迹斑斑，毛道长这次下山，就是为了给师门清理门户，多方打听之下，总算找到了这个师弟，当时这个师弟为了修炼鬼仆，到处寻找阴命的女子，找到一个富人家的小妾，勾搭她在一起，然后杀了，炼制成了人形煞……
<br/>　　毛道长找到这个师弟的时候，他刚带着这具刚炼成的人形煞，准备藏到深山里去继续炼制，结果被毛道长追赶，又不舍得这人形煞，灵机一动，找到义庄在县城里的钩子，将人形煞封印，想让他们赶到平南县来。天道宗的法术，多半是以煞气定位，没有了人形煞，毛道长便不好追踪他，而且他行走起来也快的多，他本意是摆脱了毛道长之后，到这平南县来，等着义庄的赶尸匠把尸体赶过来，自己再带着尸体进山修炼，一举两得。
<br/>　　不过这家伙想的是好，但是毛道长还是用了别的法术，追踪到了他的气息，一路追到平南县来，而义庄因为一时间没找到人赶尸，耽误了几天，这位邪修的道士又放不下人形煞，迟迟不走，这才被毛道长追上，斗法一场，被毛道长打碎了丹田和气海，从此不能再修炼任何法术……”
<br/>　　毛道长说完了整件事，对叶少阳拱了拱手拜道：“我处理了我那师弟之后，问清楚了人形煞的去向，因为不知道你们赶尸走哪条路，只好在你们约定好的这里等着，直到见到了陈爷，说了路上的事，当时也是吃了一惊，没有想到叶天师小小年纪，居然有这等实力和担当，这是惊人，只是不知道后来叶天师去了土匪窝之后，又发生了什么事，何以能够住诛杀人形煞，安全赶来？”
<br/>　　叶少阳于是把自己在土匪窝里所经历的事情简单讲了一遍，毛道长听完之后，惊得说不出话来，对叶少阳的尊敬也是更加深了一层。
<br/>　　大伙闲话了一会，叶少阳突然想起一件事，对毛道长拱手说道：“还没请教道友尊号名讳？”
<br/>　　毛道长拱手道：“不才姓毛名小方，道号洞虚。”
<br/>　　“姓毛名小方，哦，那就是毛小平毛师兄。”
<br/>　　“不敢不敢。”毛小方很客气。
<br/>　　叶少阳突然皱起了眉头，总觉得哪里不对劲，仔细一想，注意力放在了对方的名字上，毛小方，毛小方……
<br/>　　猛然间浑身一震，嘴里嚼的半块肉也一下子卡在了喉咙里。
<br/>　　“毛小方！！你是毛小方！！”
<br/>　　毛小方见他如此吃惊，吓了一跳，喃喃道：“正是在下，叶道兄怎么了，莫非是听过贫道的名字？”
<br/>　　“听过，听过。”叶少阳喃喃自语，心中真是说不出是什么情绪，毛小方……这世上真的有毛小方道长……叶少阳立刻想起了一个名字：林正英。
<br/>　　在大名鼎鼎的《僵尸道长》里，林正英扮演的那个擅长抓僵尸的道士就是毛小方。
