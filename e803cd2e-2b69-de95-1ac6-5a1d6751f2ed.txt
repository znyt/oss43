　　第131章
<br/>　　“道德三皇五帝，功名夏后商周。
<br/>　　五霸七雄闹春秋，秦汉兴亡过手。”
<br/>　　正在单飞和马家父子闹得剑拔弩张之时，忽然远处一声音响起。众人扭头看去，只见西北方向有两个道士过来。越走越近，老道士口念一歌。
<br/>　　渐渐走近风雨亭，那小道士肩上扛着一杆白幡，四个字在空中晃呀晃的：看破人间。
<br/>　　“是他来了！”
<br/>　　“这老混蛋原来还没死！”
<br/>　　单飞皱起眉头：“算命的，你有什么话说？”
<br/>　　“诸位江湖同道，今日风雨亭一会，来的都是绿林道上的弟兄。了儿，行走江湖什么最重要？
<br/>　　“和气！”少年斩钉截铁。
<br/>　　“然。”老者笑道：“大家总不能伤了弟兄间的和气，今日斗胆，就让小老儿做个公道，帮大家排排位置好不好？”
<br/>　　“算命的你读书多，懂的事情也多，你倒是说说，我们谁该坐第二张椅子？”
<br/>　　“二位稍安勿躁。”
<br/>　　老者含笑抚须，眼前这群草莽魔头，向来都是天老大老子第二的脾性，让他们承认不如别人谈何容易。
<br/>　　稍有不顺心便要拔刀相向，顺便把这老道也砍了。可面对这棘手的问题，老者似乎一点不觉得头疼。
<br/>　　“这第一张椅子，自然是伏虎山熊老大来坐。”
<br/>　　老道说罢，本来乱哄哄的风雨亭竟突然安静几分，连旷野中呼呼的风啸恍惚也静下来。
<br/>　　过了片刻，单飞冷哼一声道：“大家都知道的事，就不用你废话了。”
<br/>　　马五道：“我们也没要和熊老大争，你直接说第二张？”
<br/>　　“既然二位要争，咱们不如先搁下。刚才老道已经看过，这里共有八十一张椅子，我们干脆从后往前说起。”
<br/>　　风雨亭前乱糟糟一片，人头穿行不息，其实有多少张椅子一时也不容易数清，而对于在场更多人而言，让他们从一数到十都困难。却是没料到这老道只是粗略扫一眼，便可精确的报出数目。
<br/>　　他从后向前安排众人的座次，在场众人竟然没有不服的。有些人虽然表面上说一句：我那里不如他了。可心中却暗想：这老道的话确实有几分道理。
<br/>　　他站在那里口若悬河，侃侃而谈，众人的来历、本事、做过的事情被他一一言中。有些事情本来极少有人知道，此刻却被这老道说破。
<br/>　　单飞把玩着掌心的匕首，盯着老道的后脑勺，眼中闪过一丝狠戾。
<br/>　　风雨亭乱糟糟一片，没人注意到，在距离此地极远处的土坡上，停着一乘小轿。
<br/>　　杨龙停负手立在土坡上，目光看着风雨亭方向。
<br/>　　“一帮乌合之众，几张椅子有什么好争的？”齐德龙道。
<br/>　　杨龙停笑笑：“向来文无第一，武无第二，一群莽夫聚在一起，自然要论个高低，你以为他们争的是椅子，他们争的是地位。”
<br/>　　“主公，用不用我去管一管，再闹下去，我怕是会出什么乱子？”军师李善言道。
<br/>　　“不必。由他们去争，由他们去闹，今天便是死上几个人，也无关系，我也想看看，到底谁是我辖地的匪类头目。”
<br/>　　杨龙停不会在今天风雨亭会露面，他的身份绝不至于沦落到和一帮山贼土匪推杯换盏，不过，来瞧个热闹的兴趣，杨龙停却还是有的。
<br/>　　“虽是绿林草莽，却也有奇人异士，可惜，终不能为我所用呐。”杨龙停摆摆手，轻轻叹了口气。
<br/>　　风雨亭下，依旧乱哄哄一片，各种各样的声音都有。
<br/>　　单飞早已等得不耐烦了，突然道：“算命的，你也别废话，直接说这椅子该谁坐吧，说得不对，爷爷一刀割了你。”
<br/>　　“依老道看，二位都不配坐，能在幽州绿林道行二的另有其人。”
<br/>　　“谁！”马五。
<br/>　　“什么！”
<br/>　　众贼又是一片议论纷纷，各种声音噪杂混在一起。
<br/>　　正在这个时候，有一辆马拉雪橇从北边过来，车上窝着几个人。等快到风雨亭下，驾车的人才将马勒住。
<br/>　　『马夫』升得好相貌，八尺身躯，一双虎目炯炯有神。从车上又下来三人，一留着山羊须的猥琐老头，一裹着大袍子的年轻人，一……
<br/>　　“原来是高寨主！”
<br/>　　“上次去你飞虎寨做客，你我的酒可是还没喝够。”
<br/>　　“高寨主，我正想与你合伙做一单生意，不知你有没有兴趣？”
<br/>　　此时已经有人将高飞虎认了出来，人群将高飞虎围住，高飞虎也忙和大家打招呼。
<br/>　　程大雷裹着大袍子往前走，今天风大，他不是很喜欢这样的天气。抬头看见亭中摆着几张椅子，也无人坐，便想进去避避风。
<br/>　　他信步走进亭子，见马五和杨龙停用奇怪的眼神看着他。程大雷摸摸脸，难道今天没洗脸被看出来啦。算了，反正不认得，不去管他。
<br/>　　他看看左边的椅子，又看看右边的椅子，然后在左边的太师椅上坐了。
<br/>　　最怕空气突然安静……
<br/>　　单飞愣了好久没回过神来，突然一步跳到程大雷面前：“小子，找死么？”
<br/>　　程大雷抬起头，一脸无辜。
<br/>　　“这位置是你配坐的？”单飞用食指指着程大雷鼻子：“滚。”
<br/>　　程大雷看向徐神机，徐神机看向秦蛮，秦蛮看向单飞。
<br/>　　秦蛮出手，抓向单飞的手腕，来势汹汹。单飞身子一矮避过秦蛮，匕首向上撩去扎秦蛮的小腹。秦蛮拔出腰上短刀去斩单飞，铛，空气中发出闷响，二人连拆数招，快到看不清出手。
<br/>　　单飞手长脚长，身形快到令人发指。大家只感觉眼前一闪，一道灰影窜向亭顶。模糊中，一只大手伸出来抓住他的脚脖子，狠狠一摔，直接砸到地上。
<br/>　　单飞喉头一甜，然后不顾一切向前冲去。虎一般的身形从亭中冲出，二人一追一跑，秦蛮直追出二里地远，方才擒主单飞，直接把他扛了回来。
<br/>　　咚！
<br/>　　秦蛮将单飞狠狠的摔在地上，紧跟着一刀砍了他的左脚。
<br/>　　“啊！”
<br/>　　单飞抱着腿，发出一声杀猪般的惨叫。
<br/>　　迅雷不及掩耳，有人甚至根本没看清怎么回事，目光纷纷向亭内看来。
<br/>　　程大雷歪在太师椅上，手中袖筒伸出来，托腮看着众人。
<br/>　　“这里今天好吵喔。”
<br/>　　
<br/>(本章完)
