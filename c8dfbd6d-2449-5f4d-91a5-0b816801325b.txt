　　兑换图册上那琳琅满目的宝贝，看得萧逸直眼馋。
<br/>　　四品天材地宝，火龙果，可增强火属性武魂拥有着数倍以上的控火能力。
<br/>　　五品天材地宝，幽蓝碧果，可让破玄境以下武者，任意突破三重修为。
<br/>　　四品丹药，破空丹，先天武者服之，一天内短暂获得飞行能力，乃逃生保命之首选。
<br/>　　五品丹药，月华丹，吸收日月精华炼制而成，武者服之，短暂大幅度提升修为，三个时辰内拥有超越自身一个大境界的战力。
<br/>　　玄阶巅峰武技，雷鸣掌。
<br/>　　地阶低级武技，噬妖三刀。
<br/>　　下品灵器，猎妖刃。
<br/>　　上品灵器，三千雷火棍。
<br/>　　“咕”，萧逸不禁咽了口口水，眼馋之极。
<br/>　　“这么多宝贝，也不怕被人抢了。”萧逸微微一笑，而后看了眼殿内某处，那里正有一群黑袍武者。
<br/>　　那是猎妖殿的执法队，个个都战力极强，且萧逸看不透他们的修为。
<br/>　　证明这些人中，最弱的都是先天武者，甚至更强。
<br/>　　萧逸再次看了几眼兑换图册便放下了。
<br/>　　兑换图册上的物品并不完整，丹药、天材地宝、兵器、武技，每类列举一些出来罢了。
<br/>　　猎妖师需要兑换物品时，自然会有完整的图册观看。
<br/>　　传闻，在炎武王国王都的主殿中，连极品灵器、天阶武技，甚至传说中的宝贝都拥有。
<br/>　　萧逸微微摇头，收起心中杂念。
<br/>　　那些强大的宝物，暂时不是自己能够拥有的，还是先得到自己的培源朱果再说。
<br/>　　培源朱果在三级天材地宝行列，但属于稀有行列，在外面难以买到。
<br/>　　而在猎妖殿的兑换点数，则是100点。
<br/>　　萧逸不禁皱起了眉头，“100任务点，难度倒不大，只是时间上可能来不及。”
<br/>　　他还要去裂天剑派。
<br/>　　“啧啧。”
<br/>　　这时，萧逸身旁出现了一声声惊疑赞叹的声音。
<br/>　　萧逸转过头看了眼，不知何时起，书架前正有一个约莫15、6岁的少女。
<br/>　　少女正认真地看着书籍，而且看得津津有味，不时发出一声‘啧啧’。
<br/>　　萧逸瞥了一眼，发现少女看的是猎妖殿的过往历史和英雄事迹。
<br/>　　萧逸不禁哑然失笑。
<br/>　　“你笑什么？”少女忽然转过身来，气鼓鼓地盯着萧逸，很是不满。
<br/>　　“呵呵。”萧逸笑笑，刚想说没什么时，但看到少女的正面时，却忽然愣住，直直地盯着少女。
<br/>　　少女一身淡青色长衫，长得很美，气鼓鼓的样子煞是可爱。
<br/>　　最重要的是，她的年龄和身高都和依依很是相似。
<br/>　　有那么一霎那，萧逸错认为是依依。
<br/>　　“喂，我俩素不相识，你这样看着我很不礼貌诶，信不信我揍你？”少女跺跺脚，一副要打人的样子。
<br/>　　萧逸摇摇头，收回了目光。
<br/>　　除了样貌不同外，少女和依依有很多相似的地方。
<br/>　　但，依依乖巧可人。
<br/>　　而少女，一看就是那种喜欢调皮捣蛋的家伙。
<br/>　　萧逸收回目光，道了声抱歉，而后自顾地往咨询处走去。
<br/>　　咨询处，仍旧是那个年轻女子。
<br/>　　“易霄，可是有什么不懂的地方？”女子微笑着问道。
<br/>　　如今萧逸已经是取了令牌的猎妖师，出于尊重，女子便没有再称呼为少年，而是以名字称呼。
<br/>　　萧逸点点头，道，“我想问问，兑换图册上的物品，能否直接用钱买？”
<br/>　　萧逸补充一句，道，“我可以出比市面更高的价格买下来，两倍甚至三倍。”
<br/>　　女子微笑着摇摇头，道，“不行，你出再高的价格都不行。猎妖殿有猎妖殿的规矩，猎妖师是一门光荣的职业，只有完成任务，得到你相应的荣耀，方能有资格得到猎妖殿的宝贝。”
<br/>　　萧逸无奈，他现在不缺钱，毕竟临走前，长老们送了他不少银两，加上之前战斗所得，十万八万银两是拿得出手的。
<br/>　　但培源朱果在外面的商行又买不到，当然，不是说买不到，只能说要靠机遇。
<br/>　　摇摇头，萧逸看向了任务处。
<br/>　　任务处上有一任务栏，共四列，分别是一级任务、二级任务.、三级任务、四级任务。
<br/>　　这密密麻麻被分布好的任务，已经经过评级，猎妖师可查看任务的具体要求和难度。
<br/>　　萧逸大致地看了眼。
<br/>　　一级任务，大多是非常简单的事，例如到城外将凶猛的野兽清理掉，到某个商行搬货几个星期等等。
<br/>　　野兽比妖兽弱多了，一般的凡境武者都能轻松完成，但任务点只有1点。
<br/>　　二级任务，难度稍大，大多是些护送任务，护送商队到相邻的城池。
<br/>　　城外野道不比城内，不时会遇到山贼流寇，乃至妖兽袭击。
<br/>　　起码要凡境九重以上，乃至后天境武者才有把握完成，任务点有5点。
<br/>　　三级任务....10点。
<br/>　　四级任务....30点。
<br/>　　等级越高的任务，难度越大，相应的任务点也越多。
<br/>　　“完成十次三级任务，才攒得够兑换培源朱果的任务点。”
<br/>　　“猎妖殿每次从任务中抽取的一成悬赏金，十次加起来，抵得上一颗培源朱果的价格了，甚至还有余。”
<br/>　　一般的三品药材也就数百两价格，培源朱果就算因为太稀有，有钱难买，但最多不过一千两的价格。
<br/>　　培源朱果的效果本身不大，只不过萧逸刚好要用来炼制金脉丹罢了。若是其他炼药师，可能就对它没多大兴趣。
<br/>　　悬赏任务，由个人或者家族、势力来猎妖殿发布，给予不菲的酬劳。三级任务，悬赏金普遍在二千两以上。
<br/>　　就算猎妖殿只抽取一成悬赏金，十次加起来，得到的钱也比培源朱果本身的价格多得多了。
<br/>　　“这猎妖殿倒是精明，不仅不亏，还有赚，还能让天下猎妖师甘愿卖命。”
<br/>　　“啧啧，作为最古老的势力之一，果然不是省油的灯。”
<br/>　　心中腹诽一阵，萧逸也没有办法，谁叫人家底蕴深厚，什么珍稀宝贝都有呢。
<br/>　　萧逸收起心绪，喃喃道，“算了，先完成十个三级任务再说吧。”
<br/>　　正当萧逸准备走向任务处时。
<br/>　　“慢着。”女子忽然叫住了萧逸。
<br/>　　“易霄，你没认真看注意事项吧？”女子笑道，“你现在只是一级猎妖师，不能跨级接任务的。”
<br/>　　“什么意思？”萧逸一愣，道，“我只能接一级任务？”
<br/>　　“是的。”女子道，“这也是为了猎妖师们的安全着想，若实力不足，硬接高难度的任务，丢了性命不说，还白白让任务搁置，浪费其他猎妖师的时间。”
<br/>　　“这...”萧逸顿时气急，一级任务只有1点任务点，岂不是要完成100个任务？自己根本没有那么多时间耗在这里。
<br/>　　女子不语，但仍旧是满脸微笑。
<br/>　　忽然，萧逸眼尖，看到任务栏上，有些任务没有评级，但却都给出了10点以上的任务点。
<br/>　　女子看了一眼，解释道，“那是招募任务，由猎妖队发布，招募队员一同完成任务。你是否能接这个任务，由猎妖队本身决定，所以我们猎妖殿便不作出评级了。”
<br/>　　要整整一支猎妖队去完成的任务，想来不简单，难怪给出了10点以上的任务点。
<br/>　　“我也能接这任务？”萧逸问道。
<br/>　　女子点点头道，“可以，但前提是别人瞧得上你。”
<br/>　　.....
<br/>　　猎妖殿外，萧逸腰间挂着紫云精铁剑，俨然一副行走天下的猎妖师模样。
<br/>　　其实，早在萧逸的控火兽武魂还是黄色时，凝聚的火剑就已经比紫云精铁剑厉害了，如今更是绿色，威力就更强了。
<br/>　　但毕竟是三长老准备的，萧逸也不好不收。
<br/>　　反正，这是自己身上最不值钱的东西了，拿来装装经验丰富的武者也好。
<br/>　　本来，萧逸信心满满。
<br/>　　但，在他问了一个又一个猎妖队，且都被拒绝后，差点没被气死。
<br/>　　“小鬼，年纪轻轻就有后天一重的修为，倒是不错，但在我们猎妖队里，你还是太弱了，问问其他人愿不愿意收你吧。”
<br/>　　“才后天一重？免了，我们是要找队友完成任务，可不想带个累赘。”
<br/>　　“小鬼，滚一边去。”
<br/>　　“....”
<br/>　　相同的话，几乎不断重复着。
<br/>　　几乎所有猎妖队的队长，在看到萧逸只是个少年，不像是有经验的猎妖师，又感知到只有后天一重的修为时，都毫不犹豫地拒绝了萧逸。
<br/>　　“唉。”萧逸叹了口气，“这个世界，没有实力，还真是处处碰壁阿。第一次当猎妖师，竟是这份待遇。”
<br/>　　萧逸想起前世自己刚出道当杀手时，已是一名任务率高达百分百的传说杀手，何其风光。
<br/>　　虽然猎妖师跟杀手不同，但跟前世雇佣兵的性质差不多，同样是看钱办事。
<br/>　　言归正传。
<br/>　　萧逸之前在击杀慕容墨时，体内八条主脉受损，如今还未恢复。
<br/>　　主脉受损，要么靠丹药恢复，要么靠体内真气慢慢温养。
<br/>　　恢复经脉，二品丹药中有一些能起效果。可关键是，需要的材料都比较偏门，萧逸暂时没有这些材料。
<br/>　　像培源朱果，这就是能用来炼制恢复经脉的丹药的。只是，这是用来炼制金脉丹的，萧逸不舍得用。
<br/>　　赶路的这些天，萧逸单靠体内真气温养，仅仅恢复了一条主脉，自然便只散发出后天一重的修为。
<br/>　　猎妖队的队长们感知他的修为，自然也只有后天一重。
<br/>　　想起当初冰鸾剑反噬的锥心疼痛，那仿佛能冰封灵魂的冷冻，萧逸微微后怕。
<br/>　　“算了。”萧逸摇摇头，撇去杂念，呢喃道，“再找一个猎妖队，不行就算了，休养一番，直接去裂天剑派。”
