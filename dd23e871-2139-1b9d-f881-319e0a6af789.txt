        她的左耳因为那晚傅夜沉的失手，让她撞到了置物架的尖角，耳朵受到外力重击，造成内外血出血感染，损伤了感音神经，又因为没有及时去治疗，交叉感染牵连到右耳也受到了感染，影响到了听力。
<br/>
<br/>        医生说，像她这种创伤性耳聋可能无法痊愈。
<br/>
<br/>        先打针吃药，做消炎止血后，听力若是还是无法恢复的话，她以后就得戴助听器了。
<br/>
<br/>        得知这个结果时，童筱颖并未感到太过悲伤，只是心如止水地接受医生的治疗方案。
<br/>
<br/>        而同天，傅夜沉收到了红十字会的领导亲自送来的荣誉证书。
<br/>
<br/>        因为落以柔的大肆宣传，傅氏集团大厦里上上下下，都知道了他们的顶级大Boss是个超级富有爱心的男人。
<br/>
<br/>        紧接着，落以柔命令公关部，将这件事情积极向上的一面，扩散到了网络上去，买新闻头条为傅夜沉和集团公司做宣传，再次扩大了他们的知名度。
<br/>
<br/>        一个亿诶！
<br/>
<br/>        这可是不是一笔小数目，就这样免费捐出去了。
<br/>
<br/>        而且，傅大Boss捐的还不是公司的资金，而是他的个人收入。
<br/>
<br/>        其实，以前傅氏集团也经常做慈善，不过都是以公司的名义去做的。
<br/>
<br/>        只是没想到，童筱颖替傅夜沉做的这个决定，在落以柔带领团队的渲染和操作下，竟然还带动了傅氏集团旗下几个子公司的股份。
<br/>
<br/>        特别是旗下在云城娱乐影视业的两个子公司，因为网络效应，股票大涨。
<br/>
<br/>        集团今后的收益会呈现爬坡式上涨趋势，但这并未给傅夜沉带来一丝欢乐。
<br/>
<br/>        他孤独地站在总裁办的落地窗前，可以俯瞰整座城市的繁华，却看不透自己所爱的那个女人的心。
<br/>
<br/>        一个星期后……
<br/>
<br/>        童筱颖出院了，同时两只耳朵都戴上了助听器。
<br/>
<br/>        从此，她在路人的目光中，看到了他们眼中对她流露出来的一种同情与怜惜。
<br/>
<br/>        就连医院的医生都主动给她开了残疾等级评定书，让她去户口当地的残联办理残疾人证。
<br/>
<br/>        如果大哥知道这事了，他会不会也对她有所怜惜呢？
<br/>
<br/>        或许会……
<br/>
<br/>        又或许不会……
<br/>
<br/>        现在因为自身条件的缺陷，童筱颖新买的手机都可以不用买手机卡了，只需要用手机接收电子邮件信息即可，同时她又给自己新增了一个“技能”，去夜校进行唇语培训。
<br/>
<br/>        培训班是夏凌焕帮她找的，里面的培训老师最这市里最好的唇语老师。
<br/>
<br/>        此外，她还从夏凌焕的口中得知，夏凌焰快要结婚了，和市长家的千金大小姐在一起了。
<br/>
<br/>        而婚礼的前一个星期，夏凌焰也主动给她童筱颖发了请柬。
<br/>
<br/>        婚礼这天，童筱颖本想托夏凌焕给自己捎了个红包去，然后帮她带包喜糖回来吃就行了。结果，夏凌焕不依，非得拉着她童筱颖一起去参加夏凌焰的婚礼。
<br/>
<br/>        到了婚礼现场入口，童筱颖看到门边的海报才知道，夏凌焰的新娘居然是傅夜沉之前的那个未婚妻李忆莎。
<br/>
<br/>        她之前和李忆莎在宠物店里发生过摩擦，一想到那件事情，心里就膈应着，于是，她上了红包礼数，领了喜糖，不跟夏凌焕打招呼，就偷偷撤人了。
<br/>
<br/>        因为酒店里人声鼎沸，吵得童筱颖有些耳鸣目眩，她索性将两只耳朵上的助听器给取了下来收入了口袋中。
<br/>
<br/>        耳根子里清静后，两耳上也没有了压迫感，她感觉舒服多了，于是一边朝那边的电梯走去，一边低头打开喜糖袋从里面捏出一颗水果糖剥开来准吃的时候，忽然有人伸手将她揽入了怀中。
<br/>
<br/>        童筱颖还未来得及反应过来，视野里一个黑影覆上，她刚把糖果纸展开将糖果送入嘴中，一个温热的唇，隔着这片糖果纸，压在了她的唇上。
<br/>
<br/>        当她瞪大眼睛看清楚这个强吻自己的人时，吓得猛地推开了他。
<br/>
<br/>        傅昼景却顺势搂住她的肩头，将刚刚那个压在她唇上的吻，转移到了她的额头上，如蜻蜓点水般轻轻地啄了一下。
<br/>
<br/>        “大哥，好巧啊！你也来参加李忆莎的婚礼了！”
<br/>
<br/>        直到这一刻，童筱颖才看到正从电梯里出来的傅夜沉。
<br/>
<br/>        而傅夜沉的俊脸上波澜不惊，甚至看都未看他们一眼，便在一群人的簇拥之下，唯我独尊地往那边的婚礼礼堂入口大门走去。
<br/>
<br/>        看着傅夜沉的背影渐行渐远，童筱颖回过神来，扬起手，直接掴了傅昼景一巴掌。
<br/>
<br/>        “啪”地一声清脆的巴掌声，在这电梯前室里响起。
<br/>
<br/>        童筱颖趁着电梯门还未关上，二话不说，疾步往电梯里走去。
