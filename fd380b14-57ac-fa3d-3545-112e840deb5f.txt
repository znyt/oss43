　　云儿道：“我已经忍受了三年，早就习惯，再有初来的姑娘，又如我当初一般……三年之后，还有别人，我愿以自己一人代替她们接受这命运，若主子留我下来，我必定百倍温柔服侍，恳请主子不要再染指别的姑娘……”
<br/>　　五通神大笑：“你倒是好心肠，哈哈，似乎该在庙里供奉的是你，不是我。你是不想我再去糟蹋别的姑娘？”
<br/>　　云儿垂下头，不敢说话。
<br/>　　五通神道：“好，既然你要留下来，本王我当真了，留你下来，一生不许下山！”
<br/>　　云儿一惊，道：“主子答应了？”
<br/>　　“你自己要求，我自然答应，不过……别的姑娘，我却也是要的！至于你……等本王我玩腻了再说，哈哈。你刚不是说糟蹋吗，那我今天晚上就好好糟蹋你！”
<br/>　　说完直接扑了上去……
<br/>　　“不！”云儿惨叫着，却被五通神压在了身下。
<br/>　　叶少阳强忍着冲天的愤怒，飘身离去，他不忍心看这一幕，但是自己现在只是一抹元神，什么都做不了，只能先下山……
<br/>　　元神回到自己身体之后，叶少阳睁开眼睛，从地上坐了起来。
<br/>　　“你醒了，怎么样？”毛小方急忙问。
<br/>　　“王八蛋！畜生！”叶少阳大声叫骂，立刻朝山上奔去。
<br/>　　毛小方愣了一下，立刻追赶上去，一把抱住叶少阳，问道：“出什么事了，少阳子，你先说清楚！”
<br/>　　叶少阳定睛看着他，沉声说道：“我要去屠神，你跟我一起吗？”
<br/>　　“屠……神？”
<br/>　　“来不及跟你说这么多，你要是不去，我就自己去。”
<br/>　　叶少阳撇开他，继续往山上狂奔。他现在脑子里只有一个念头，就是灭了五通神！
<br/>　　毛小方好不容易追上他，苦苦劝道：“少阳子，你不要冲动，不管发生什么事，你总先和我说一声，不然反而容易坏事。”
<br/>　　在毛小方苦苦劝说之下，叶少阳总算稍微冷静下来，将山上发生的事情简单说了一下。
<br/>　　毛小方听完，当场就怔住了，喃喃道：“那五通神……居然干出这种勾当？”
<br/>　　“亲眼所见！”叶少阳道。
<br/>　　“这……少阳子，我知道你想要打抱不平，你想清楚，这五通神是法师上表、阴司封赐的正神，即便能做到，你我可没有资格杀他。你也说了，当年大明朝时斩杀的五通神，还是请的文曲星方孝孺，你我一介人间法师，能把他怎么样？”
<br/>　　叶少阳怔怔地看着他，长出了一口气，道：“知道了，我自己去。”
<br/>　　说完一路狂奔，跑了一段，突然听见身后有脚步声，转头一看是毛小方跟了上来，停步说道：“你不要再劝我了，你不去，我也理解，我自己去就是了。”
<br/>　　毛小方道：“以你现在的实力，能杀了五通神？”
<br/>　　叶少阳道：“书生一怒，血溅五步！那姑娘正在遭受摧残，我要是不去，愧对道心。抱歉，我这人天生当不了缩头乌龟。”
<br/>　　毛小方听他这么说，有些动容，叹口气说道：“少阳子，你不是普通人。我毛小方虽然不是什么宗师，一向也是率性而为，我也不怕得罪五通神，这件事，我陪着你一起。”
<br/>　　叶少阳微微一笑，“我就知道，大名鼎鼎的僵尸道长，肯定不是怕事的人。”
<br/>　　毛小方道：“不怕事，但也不能贸然前去，我们这么上去，恐怕不是五通神的对手，根本就救不了那姑娘，反而打草惊蛇……从长计议，我们好好商量对策。”
<br/>　　叶少阳道：“可是那姑娘正在……”
<br/>　　毛小方道：“她已忍受三年，也不差这一晚，我知道这话不对。但是如果我们这就上去，反而救不了她，可能反而害了她。我们好好商量，把她救出来，免她一生受苦。反而对她是好事，你说呢？”
<br/>　　叶少阳沉吟了一下，道：“虽然你说的是对的，可我还是不忍心，我倒是想到个办法……”
<br/>　　“什么？”
<br/>　　“放火。”
<br/>　　十几分钟后，两人各自捧着一大把从山上找到柴草，来到庙堂侧面的上风口，把火点燃，火很快就把庙堂外面的树木烧着了。
<br/>　　叶少阳捡了几根着火的树枝，一个劲地往庙堂里扔，同时吹了一声口哨，大声说道：“大丈夫行不更名，我们是二龙山的绺子，杀富济贫，打倒一切牛鬼蛇神！今天先给你们点添头，三天之内，凑齐一万银元，等着爷们来取……”
<br/>　　说完一招手，两人迈开步子，顺着山路狂奔下去。
<br/>　　叶少阳相信，经过他们这么一闹，五通神肯定没心思干那事了，不过最后这段冒充山贼土匪的话，也是临时想到，他知道五通神不可能信，也就是随口一说，找个借口。
<br/>　　屋子里全是烟，五通神还哪有心思啪啪，从云儿身上爬下来，朝门外冲去，到了门口，转身看到云儿还坐在床上抽泣，冷笑一声：“你想被火烧死？可惜本王还被玩够！”
<br/>　　冲过去一把抱在怀里，带了出去。
<br/>　　叶少阳的一个火把正好扔进了柴房，点燃了柴草，陈老汉正在打水救火，看到五通神抱着云儿出来，也不惊讶，磕头说道：“大神，刚有土匪来砸窑了。”
<br/>　　“我神庙在这里几十年，谁不知灵验，从没有土匪敢惹，不可能是土匪。”
<br/>　　陈老汉道：“也可能是流寇，知道我们这香火鼎盛，却认不得真神……”
<br/>　　“我出去看看，看到了她。”五通神放下云儿，冲出了庙门。
<br/>　　陈老汉自己去打水救火，也不管云儿，他知道云儿不会跑，事实上，她也不是没跑过，不过五通神交合之时，已经吸了她的魂气，不管她跑到她，都会被抓回来，然后虐打一番，就算自杀也没用，投井，上吊，她都试过，五通神都能帮她还魂，真正的求生不得求死不能。
<br/>　　云儿呆呆地坐在地上，双手捂脸，无声地哭泣着。
<br/>　　叶少阳和毛小方早就下了山，躲了起来，商量对付五通神的办法。
<br/>　　按照毛小方的意思，对付五通神，只能上表阴司，撤了他的神位，然后当成邪物来对付。
