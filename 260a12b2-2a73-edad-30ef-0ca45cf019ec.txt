　　“不要紧！”
<br/>　　云熙打掉罗昊伸过来的手，眼眶一红，眼泪就要往外掉，显得非常委屈。
<br/>　　这下子罗昊彻底急了，赶紧跳下床来询问云熙具体情况；可云熙却还是执意不说。
<br/>　　罗昊一把将云熙抱起放在床上，伸手就要去脱云熙的裤子，可云熙却死死地抓着裤腰！
<br/>　　“松手！”
<br/>　　罗昊瞪了云熙一眼。
<br/>　　“全都是你做得好事，你现在还吼我！”
<br/>　　云熙一脸的委屈。
<br/>　　看着云熙的样子，罗昊知道自己昨天晚上的确是做得过分了，连忙安慰了几句，好说歹说，才说服云熙让自己帮她查看伤口。
<br/>　　这不看不知道，一看还真把罗昊吓了一跳，明明已经过了生理期，可是云熙内裤上却垫了卫生棉，而且上面还带着血迹！看到眼前一幕，罗昊才知道昨天晚上的自己，到底是有多粗鲁。
<br/>　　“我去拿药膏！”
<br/>　　罗昊赶紧起身，下楼去找药箱，并且很快抱了一个药箱回到了卧室。
<br/>　　尽管结婚多年，可那里毕竟是女人最私密的地方，让罗昊上药还是让云熙觉得不好意思，想要自己去厕所涂药膏，可是却被罗昊给拒绝了。
<br/>　　凉凉的药膏涂在伤口处，让伤口的疼痛感缓解了不少，云熙看着埋头给自己上药的罗昊并没有告诉他，其实昨天晚上在他睡着之后，自己偷偷起身去了厕所，已经换过一次卫生棉了。
<br/>　　罗昊收起药箱，对云熙说道：“目前来看血是止住了应该没什么问题，如果伤口再出血可能就要去医院了。”
<br/>　　云熙点了点头，说道：“你去给悠儿和小战弄点吃的，他们该饿了。”
<br/>　　罗昊俯身在云熙额头上浅吻了一下，道了一声“对不起”。
<br/>　　刚才下楼的时候急着找药箱没有看到云悠留在茶几上的纸条，现在下楼放药箱，罗昊注意到茶几上的纸条，拿起一看，顿时乐了，“还真饿不了这姐弟俩。”
<br/>　　罗昊刚把纸条放下，家里的电话就响了起来，是唐磊打来的电话。
<br/>　　“昊子，你家的心肝和宝贝在我这儿呢。”唐磊对着罗昊打趣道：“姐弟俩对于你霸占了他们的妈妈，让他们饿肚子意见很大，小罗战都说你是个假爸爸。”
<br/>　　罗昊脸色一黑，这混小子居然敢这么说自己，看回到龙隐部队自己怎么训他！
<br/>　　“他们在你那儿我就放心了，那就让他们现在你那儿待着吧，我得去伺候我家的女王大人了。”罗昊对着电话那头的唐磊说道。
<br/>　　“我算是发现了，小罗战说得果然没错，你丫的就是个假爸爸。”唐磊说道：“行了，你就开启你的宠妻模式吧，你家的心肝和宝贝我先帮你照看着，他们都好久没来找我玩了，我正想他们呢，等晚点我再让他们回来。”说完，唐磊就直接挂断了电话。
<br/>　　罗昊笑了笑放下电话，进了厨房，给自己和云熙各做了一碗炸酱面，端着面碗上了楼。
<br/>　　刚进房间，云熙就似笑非笑的看着自己，说道：“假爸爸，你现在你儿子和女儿心中的印象可是一落千丈啊。”
<br/>　　别墅里的电话除了客厅以外，主卧里也有分机，刚才唐磊电话打进来的时候，云熙也接起了电话，所以把自己跟唐磊之间的对话听得一清二楚。
<br/>　　罗昊尴尬的笑了笑，把比较少的那碗面递给云熙。
<br/>　　云熙接过面碗，对罗昊说道：“以后在家还是注意点，给孩子知道了不好。”
<br/>　　罗昊唏律着面条，说道：“这姐弟俩聪明的很，他们都知道。他们俩，长大了肯定比我们还优秀。”
<br/>　　“我这女儿和儿子本来多乖啊，你一回来，他们就学坏了。”云熙白了罗昊一眼，说道。
<br/>　　“这么说都怪我咯？”
<br/>　　“不怪你难道还怪我？”
<br/>　　“……”

<br/>　　杭山市国际机场，1号航站楼。
<br/>　　慕容飞很郁闷，并且是非常郁闷。本来他订的是昨天傍晚的机票，为了能尽早赶回去，还刻意多花了钱订了时间最早一班的航班，可是谁知道杭山市这边遇到了一场罕见的雷暴雨，全国所有飞往杭山市的航班，以及从杭山市出发的航班全都停飞了。

<br/>　　最后，那几个闹事的大妈非但没有能提前登机，还被警察叔叔带走去“吃早餐”了，可谓是得不偿失。
<br/>　　杭山市国际机场1号航站楼中，昨天晚上因为雷暴雨而被滞留在这里的旅客还没有全部得到解决，各个航空公司的柜台前，带着行李的旅客排起长队等着办理登记手续和托运行李。

<br/>　　慕容飞拎着背包折身返回了航站楼去找地铁站入口，站在地铁售票机前，慕容飞犹豫着是先回家还是去佟千雪那儿。
<br/>　　最后慕容飞在心中对自己老爸老妈说了声对不起，还是按下了距离佟千雪所住小区最近的地铁站，他要给佟千雪一个惊喜。
