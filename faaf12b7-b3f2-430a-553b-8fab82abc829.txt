        &lt;h3&gt;第三卷第二百二十一章&lt;/h3&gt;
<br/>
<br/>        郝景文将陈思宇送回家，陈爸开着车带着郝景文和陈思宇在扬州走了一个遍，过年了，走亲访友是中国的传统习俗，前几天陈思宇没有在家，他们确实是去了陈思宇的姥姥家，只是吃了一顿年夜饭就回家了，他们就陈思宇一个女儿，女儿一走感觉家里空荡荡的，非常的不习惯。
<br/>
<br/>        “你说你是不是神经？郝家那孩子确实不错。并没有向你家提亲，你就让人家带着你女儿回去，这思宇一走，家里就空了。”陈思宇的妈妈贾凤瑶说。
<br/>
<br/>        “孩子迟早是要走的，早一天迟一天有什么区别呢？郝家那小子确实不错，思宇嫁过去不会受罪，所以我就想撮合他们。”陈正刚叹了口气。
<br/>
<br/>        “你对郝家那小子又不怎么了解，就那么放心你女儿嫁过去？”贾凤瑶对她老公的想法无语。
<br/>
<br/>        “我了解他们的感情，郝家那小子的眼里都是思宇，你们女人不懂，反正我能看出来。就像是我的眼里都是你一样的。”陈正刚笑眯眯的看着贾凤瑶。
<br/>
<br/>        “说孩子呢，扯到我们干啥，思宇说今天回来呢，看情况吧，感觉有希望了就带着他们去走走亲戚，让大家也认识认识。”
<br/>
<br/>        “行，听你的，你说怎么办就怎么办。那我去机场接他们，你在家里准备饭菜，一定要准备丰盛些。”陈正刚拿起车钥匙就出门。
<br/>
<br/>        ……
<br/>
<br/>        陈思宇看着郝景文：“感觉是回娘家一样，还拿着这么多东西。”
<br/>
<br/>        郝景文：“难道不是回娘家吗？”
<br/>
<br/>        陈思宇：“不理你了。”
<br/>
<br/>        郝景文笑了：“你要是不理我，我真得在马路上吹吹风，你看，这可是在你的地盘。”
<br/>
<br/>        风，微微的吹着，温润而潮湿，俩个人站在马路边等陈正刚来接。
<br/>
<br/>        “思宇。”陈正刚将车停在俩个人的旁边，看着自家的女儿，一瞬间有一种想哭的感觉。
<br/>
<br/>        “爸爸，叔叔。”郝景文和陈思宇同时喊着。
<br/>
<br/>        “回来了就好，回来了就好。”陈正刚有些激动，女儿也是经常在外面上班，这一次回来心里总是有一种酸涩的感觉。
<br/>
<br/>        女儿大了，总归是要嫁人的，想着女儿会离他而去，心里不是滋味，那个时候经常催着女儿相亲，可是真的找到男朋友的时候心里是万般不舍。谁说生女孩比生男孩省心?生女儿怕她被人欺负，护在手心当宝，等到长大了，要嫁人了，又怕遇人不淑。好不容易遇到一个心仪的又患得患失。
<br/>
<br/>        郝景文将带回来的东西放进了行李箱，和陈思宇坐到了后排，陈正刚从后视镜看到俩个年轻人十指紧扣，心里微微松了口气。
<br/>
<br/>        扬州新年的气息还是很浓的，风夹杂着淡淡的微凉在穿街走巷，似乎想要将扬州古老的记忆一股脑的塞给这个叫郝景文的新女婿。
<br/>
<br/>        陈正刚有很多的话想要问陈思宇。比喻说，郝景文家里情况怎么样啊，去了郝家，父母亲怎么样啊，对她好不好，有没有受到委屈，过去生活习不习惯，什么时候会结婚等等这些问题，但是，郝景文还在车上，什么都不能问，不过看情况，思宇是一点委屈都没有受。
<br/>
<br/>        郝景文当然会明白未来岳父的心思，然后和陈思宇在车上聊天，故意的将陈思宇的思路带起来。
<br/>
<br/>        “思宇，在那些亲戚中你印象最深的时谁？”郝景文问。
<br/>
<br/>        “哎呀，你知道我这人有个毛病，记不住人，感觉大家都好，每一家都好热情，每天去好几家，我记不住了。”陈思宇实话实说，她是搞财务的，她的记忆力也是特别好也很细心，但是在于辨识人物印象上她天生短板。
<br/>
<br/>        “那爸爸妈妈你该记得住吧？”
<br/>
<br/>        “文仔，你傻呀，我要是连爸爸妈妈都记不住了那我不就真成了白痴了？说真的，我去你家，真的让我好感动，尤其是叔叔阿姨对我特别特别好，比我爸爸妈妈对我都好。”陈思宇直肠子，都没有考虑到她爸爸在前排开着车。
<br/>
<br/>        虽然陈思宇说了郝景文父母亲比他们好，但是陈正刚一点也不生气，反而心里一松，女儿的幸福有着落了，他不用那么揪心了。
<br/>
<br/>        “肯定是比不上叔叔阿姨，但是我爸爸妈妈一直想要个女儿，你这是又当女儿又当媳妇，我现在都已经靠边站。”
<br/>
<br/>        陈正刚欢喜的笑了。
<br/>
<br/>        乡镇企业家陈正刚带着郝景文参观了他的工厂，带着郝景文走亲访友，大家对郝景文的颜值和举止都暗暗赞叹。
<br/>
<br/>        ......
<br/>
<br/>        日子如流水，郝景文和陈思宇都要投入到紧张的工作，今年的工作力度比往年都大，陈子昂的耗材推广已经定上了日常，全国联动的计划没有改变，陈子昂坚信要一炮打响，陈思宇和凤盛夏是财务副总监和总监，俩个人也倍感责任重大。
<br/>
<br/>        上市计划早就起动，就等挂牌的那一天，大家心里都憋着劲，陈子昂的公司里每个人都干劲十足，他们感觉在这里上班就是一种荣耀，他们的公司虽然年轻，但是他们公司的文化氛围和企业精神是一般的人望尘莫及的。
<br/>
<br/>        上班之后，郝景文才发现他们几个人的群非常安静，安静的有些诡异，除了新年大家互相祝福之外，这一段时间基本很少联系。
<br/>
<br/>        他最近也是有着陈思宇陪着就不去群里聊天了，况且这个群就是他们几个的工作群。
<br/>
<br/>        帅得掉渣的文文：“哈喽，大哥二哥，三哥，四哥大家好。我回来了！”
<br/>
<br/>        帅得掉渣的轩轩：“二哥？四哥？”
<br/>
<br/>        帅得掉渣的文文：“陈疯子，文仔。”
<br/>
<br/>        帅得掉渣的轩轩：“哈哈哈哈，好吧！不过陈疯子比我小。”
<br/>
<br/>        帅得掉渣的文文：“你曾经被陈疯子领导，就这样吧。”
<br/>
<br/>        帅得掉渣的轩轩：“上班了？”
<br/>
<br/>        帅得掉渣的文文：“是啊，向大哥们汇报工作情况。”
<br/>
<br/>        帅得掉渣的轩轩：“大哥好像去了美洲，陈疯子估计有事情在忙。我在西城已经上班好几天了，你自己看着办就好。”
<br/>
<br/>        帅得掉渣的文文：“那我下午还是去看一下陈疯子，和她商量一下后期计划，另外看看软件怎么样了，这几年的旅游业越来越兴旺了，得加快脚步。”
<br/>
<br/>        帅得掉渣的轩轩：“不单单是工作吧？一定还有别的事情。”
<br/>
<br/>        帅得掉渣的文文：“没有什么别的事情，我和思宇过年在一起，刚刚分开，唉，不过刚分开也是想念的紧，你说的对，我还想去见见思宇。”
<br/>
<br/>        铺面而来的狗粮从屏幕中喷了江俞轩一脸。
<br/>
<br/>        江俞轩满心酸涩，他曾经也到处撒狗粮，从过年相互的问候到现在他和张倩楠竟然没有联系过。
<br/>
<br/>        陈子昂回去的时候也没有和他联系，也不知道陈子昂如何了，最近他害怕见到她们。
