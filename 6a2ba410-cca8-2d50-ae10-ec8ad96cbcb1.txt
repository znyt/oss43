        童筱颖这才想起，自己光顾着自己吃去了，大哥好像一口都还没吃到。

<br/>        “可是，已经没了噢！”童筱颖瞥了一眼面前那盘只剩下汤汁的空盘子，顿了顿后，接着说道，“要不，大哥，你再做一份？”
<br/>
<br/>        “你还想吃啊？”傅夜沉忍俊不禁地问。
<br/>
<br/>        他这算不算是被自己的老婆给调教了？
<br/>
<br/>        从一个厨艺白痴，到厨艺高手，其实只要是为了“爱”，再困难的事情都好像变得很简单了。

<br/>        童筱颖手中拿着叉子，眨巴着眼睛看着傅夜沉，很认真地点了点头。

<br/>        “睡觉前吃太饱，对身体不好。”傅夜沉从中岛台前起身，走到童筱颖的跟前，微微俯身，温柔地哄道。

<br/>        他的一张俊脸，瞬间在她的瞳孔里放大。
<br/>
<br/>        童筱颖怔愣地眨了眨眼睛，完全不知道他接下来的意图是什么。
<br/>
<br/>        就在她出神之际，他的大手倏然抬起她的下巴，一吻落下。
<br/>
<br/>        “唔……”她刚吃完鸡翅都还没擦嘴巴的啊！
<br/>
<br/>        肯定是一嘴的油……

<br/>        童筱颖试图推开傅夜沉，但他却迅速伸出另一只手，一把捉住她的双腕，紧紧地扣在他的胸膛上让她动弹不得。

<br/>        唇齿间有可乐鸡翅的味道，在两人的味蕾上交换。

<br/>        这些天，他跟她在一起一直在禁欲。

<br/>
<br/>        她的双唇被他吻得微肿，唇色潋滟的样子，格外迷人。

<br/>        童筱颖喘息之际，只觉身子突然腾空上升。

<br/>        傅夜沉已经轻而易举地将她打横抱起，他抱着她大步流星地离开了厨房。

<br/>        但这一夜，并没有一室旖旎。
<br/>
<br/>        不过，他难得一次没有再跟她分房睡。
<br/>
<br/>        签证办下来后，傅夜沉没有食言，从西班牙到希腊，只要是她童筱颖想去的地方，他都奉陪到底。

<br/>        和傅夜沉在一起的这段日子里，童筱颖的脑海里总是隐隐约约地闪过一些与他有关的画面。
<br/>
<br/>        起初，她会觉得，那些画面是另一个女人与他的故事，到后来，慢慢地她把那个女人当成了自己。
<br/>
<br/>        有些事情，她没有跟傅夜沉挑明，但是旅途结束后，在回国时的VIP机舱里，童筱颖正挽着傅夜沉的手臂，谈笑风生的时候，一个与她长得一模一样的女人，突然迎上前来，跟他们打了声招呼。

<br/>        “夜沉，好久不见。”苏漫雪摘掉了脸上的墨镜，微笑着向傅夜沉挥了挥手。

<br/>        这里是VIP特等舱，除了苏漫雪的经纪人，就只剩下傅夜沉和童筱颖了。
<br/>
<br/>        童筱颖看到这个女人时，回想起那天在慈善拍卖会上，她和傅昼景的事情，便一脸狐疑地瞪向傅夜沉。
