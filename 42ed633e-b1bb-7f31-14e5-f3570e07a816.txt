　　第416章
<br/>　　一道黑色洪流从城门涌入，他们身后都绣着蛤蟆，这种丑陋的生物，此刻在帝国军心中俨然已成为恐怖的象征。帝国军从两侧涌过来，想要将敌人堵回城外，两道洪流恶狠狠的碰撞，溅起红色的血浪。
<br/>　　李乐天身先士卒，出现在队伍最前方，他握紧剑，高高扬起，又重重挥下。
<br/>　　“杀，与我杀！”
<br/>　　“殿下的安全要紧，我们掩护殿下撤退。”负责保护李乐天的亲卫道。
<br/>　　李乐天挥剑砍掉一名起义军的脑袋，血溅了他一脸，他伸手抹去，瞳孔已布满猩红的血丝。
<br/>　　“杀，再敢有言退者，与此人同命！”
<br/>　　士兵掀起一波小高峰，李乐天的悍勇刺激了诸人，他们握紧武器厮杀，然后迎来更大的牺牲。
<br/>　　城门内外，皆是杀机，尸体压着尸体，残肢叠着残肢，李乐天也在厮杀的最中央。
<br/>　　左斜劈，右横斩，中平刺……
<br/>　　他双手握剑，最基础的剑招，很难说此刻的战斗凭的是大脑，还是单纯的肌肉记忆。
<br/>　　战斗从不是一件好玩的事情，用最大的力气捅穿敌人的心脏，杀死和自己同样构成的生命，然而也随时在等待被敌人杀死。
<br/>　　一柄流矢射在李乐天的肩头，他吃痛跪倒，随即昂然站起，挥剑捅进一个敌人的小腹。
<br/>　　这种状况已分不清流矢是来源于哪里，正义教或者是帝国军。血流不止，因力竭和失血，李乐天已有些头晕。
<br/>　　“殿下，撤退吧。”亲兵队长道。
<br/>　　“不可退，与我杀！”李乐天用自己最大的力气嘶吼，却是越来越无力。
<br/>　　亲兵队长狠狠的一咬牙，将李乐天扛在背上，挥剑劈开身周的敌人，往城西逃去。
<br/>　　“不可退，不可退……”李乐天奄奄一息，想要用最后的力气做最后的努力阻止败势。
<br/>　　亲兵队长护送李乐天离开，对李乐天来说是最好的选择，然而对于战斗来说却是未必。
<br/>　　将为兵之胆，李乐天的溃逃带走了帝国军最后的士气，也带走了胜利的唯一可能。一支失去士气的队伍，同待宰的猪狗相比没有任何区别。
<br/>　　当然也不能说完全没有区别啦，至少猪狗更难对付一些。
<br/>　　李乐天明白这一点，他甚至做好与长安城共存亡，死在战场上的准备。然而他终于什么都没能做，在亲兵的带领下一路向西。
<br/>　　此时此刻，向外逃的不仅李乐天一个，城中的商贾、平民、高官……像是被火燎着的蚂蚱一般疯狂的涌向城外。
<br/>　　西门更加拥挤，有男人的咒骂、孩子的啼哭、女人的抽泣。这些人大多都是长安城乃至整个帝国的大人物，他们都是体面人。
<br/>　　但今日，这些人的体面跌得干干净净。
<br/>　　“大王子！”
<br/>　　一名老将候在城门，似乎一直在等待李乐天。李乐天用最后的力气推开亲兵队长，步履踉跄的靠近。
<br/>　　这员老将是帝国将军尉迟离，他手下豢养的门客多有高手，如果此时纠集人马，杀回东门，此战的胜负还未确定。
<br/>　　“尉迟将军……”
<br/>　　尉迟离重重咳嗽，他已年迈，不是那名战场上厮杀的猛虎。
<br/>　　“大殿下，快快逃命吧，老臣护送大殿下离开！”
<br/>　　李乐天一怔，心中恍惚有什么东西被击破。然后，他又想起了更多的事情。尉迟离位高权重，帝国一多半的名将由他亲手调教。但在这次战役中，他的表现算不上愚蠢，但也仅仅是平庸而已。
<br/>　　恍惚，是他也在等待贼人破城。
<br/>　　这位一直被父王视为帝国栋梁的人，当真对帝国忠心耿耿么。
<br/>　　忠不忠，奸不奸，这便是如今的帝国。李乐天想起明帝曾对自己说过的一句话：养士如养鹰，饱则遁去，饥则噬主。
<br/>　　似乎此刻也无法责怪他什么，驭下无力，当责怪的似乎只要明帝。即便想要责怪什么，李乐天确实也没有这力气。
<br/>　　长安城破之日，李乐天中箭负伤，在尉迟离的掩护下，带着败军从西门逃出长安城。
<br/>　　长安城破，扯掉了帝国最后一块遮羞布。但此时若说正义教取得这场战斗的胜利，尚为时过早。
<br/>　　帝国军不缺逃兵，但也不缺悍将，这些人带领残余兵马，在长安城内与正义教的匪兵展开巷战，杀得依旧十分惨烈。
<br/>　　而此时，林少羽带着手下精锐，直扑帝国皇宫。
<br/>　　在这里，他遭遇了帝国最精锐的一支队伍，鱼龙卫。
<br/>　　帝国军是帝国的兵，但未必是李家的兵，他们大多在各个诸侯手中掌握，但鱼龙卫则是明帝亲手调教，亲手提拔，实实在在是明帝手中握着的队伍。
<br/>　　战斗打得很艰难，据皇宫坚守，拒正义教的匪兵在门外。
<br/>　　“陛下，长安城已破，老奴护送陛下离开吧。”刘阿吉跪在明帝面前。
<br/>　　明帝坐在金殿上，文武百官一无所踪，眼前只有一个去势的太监。在他面前，摆着一杆铁枪，明帝目有困惑，似乎思索着什么，对于刘阿吉的话置若罔闻。
<br/>　　“陛下……”刘阿吉又唤了一声。
<br/>　　明帝恍然回过神来，抬起头道：“什么，你刚才说什么？”
<br/>　　“贼兵已破了长安城，正与鱼龙卫激战，为陛下龙体着想，由老奴护送陛下离开吧。”
<br/>　　“离开，又要逃啊？”明帝声音有些恍惚。
<br/>　　刘阿吉不敢回答，怎么能说逃呢，要说东巡、西猎、北游、南狩，怎么可以说逃呢。
<br/>　　明帝摇摇头，目光由浑浊变得清澈，他长身而起，龙游袍衣。
<br/>　　“我逃了一次又一次，不想再逃了。”
<br/>　　“陛下。”刘阿吉声音有些沙哑。
<br/>　　“阿吉，你说日后的人，会怎么评价我们这些人呢？”明帝问。
<br/>　　“陛下英明神武，做过的事情当是有人记得的。”刘阿吉眼眶泛红。
<br/>　　“戎族犯边，民不聊生，诸侯林立，豪强逞凶。”明帝苦笑着摇摇头：“我们李家管这些事，的确是做的不好呐。不破不立，只希望后来人，能做得比我好。”
<br/>　　“陛下，日后未必不能东山再起，教匪猖狂，却必然不得长久。”
<br/>　　“不，我做这龙椅一辈子不体面。”明帝长身而立：“现在，朕想有个体面的死法。”
