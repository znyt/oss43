　　“训练继续！”
<br/>　　杨天的忽然出声，打断了这批新兵的好奇心，也让他们毫无任何心理准备，就这么在接下来的一个多月时间里，被刘玉定的各式处罚折腾的死去活来！
<br/>　　“体能极限汇报。”杨天又说出了一句没头没脑的话。
<br/>　　在场的五千新兵都没明白，而分配到每个小队的警戒队成员，则是立即依次出声。
<br/>　　“第一小队汇报，新人类两人，普通士兵三人，五名菜鸟！”
<br/>　　“第二小队汇报，新人类三人，普通士兵三人，四名菜鸟！”
<br/>　　“第三小队……”
<br/>　　五百个小队一个个报过去，旁边的几名记录人员不断在纸上刷刷写着，杨天站在旁边认真听着，五千士兵则是一个个呆滞的站在原地，根本不知道发生了什么。
<br/>　　新人类他们听得懂，普通士兵也听得懂，但是菜鸟是啥意思？
<br/>　　他们不知道这是狼牙训练新兵时的正常流程，每一个普通人的初始体能都不同，有些人体能较好则是能列入普通士兵之列，这些人已经接近了新人类的门槛，只差一步就能越过去，这时候再配合狼牙研究部的逆天发明新人类觉醒剂，就有着近乎百分百的可能成为新人类！
<br/>　　而菜鸟实在没什么好解释，就是体能弱到爆的最普通人，而这些人大多数都来自段氏和南署基地的居民，没有长期作战和训练的情况下，想要步入新人类的可能性微乎其微。
<br/>　　此时哪怕有新人类觉醒剂，想要成为新人类也有着一定的风险性。
<br/>　　在狼牙一次又一次的实验，以及不断培养而出的新人类过程中，研究部已经对新人类觉醒的条件了如指掌。
<br/>　　就像现在，只是一次九十分钟的俯卧撑体罚，警戒队成员就牢记了小队里每一名成员的情况，什么时候坚持不住，坚持到了几分钟，坚持了多少个俯卧撑，这些数据结合起来形成了一个大致门槛，没有达成的则被称之为菜鸟。
<br/>　　菜鸟不能服用新人类觉醒剂，因为体能的不足让他们的觉醒存在风险，必须体能到达普通士兵的级别，才可以尝试。
<br/>　　正是因为这种严谨态度，所以在新人类觉醒剂发明而出，并且一次又一次的改良过程中，狼牙战团的士兵们哪怕全体步入了新人类，也极少出现意外情况。
<br/>　　楚涵说过，他不会让任何一名精心培养出来的狼牙战士用生命冒险！
<br/>　　所以这次刘玉定来此的突然处罚，当即就达到了一箭双雕的目的，异心者第一时间逐出大部队，也让新兵们尝试了自己的体能极限，让警戒队的小队长们能够更好的了解队员情况，以便在接下来的训练以及觉醒剂服用中，有着更好的配合与作用。
<br/>　　新兵们当然不知道这一点，他们只清楚貌似只是一次九十分钟的俯卧撑，那些警戒队队长们就将每一个人在团队中所处的位置大致分配完毕。
<br/>　　这些五万新兵被全体划进了团队作战的配合训练中，所以在这仅有两个月的时间里，狼牙的军官也压根没想过将他们以战力高低分开，而是从一开始就安排了新人类、老兵和新人的组合，让所有人的能力在战斗中发挥出最大效果。
<br/>　　第一批五千人大队的训练继续，而刘玉定则是已经来到了下一个五千人大队面前，将第一大队发生过的一切来了个重复。
<br/>　　这时候所有未见过刘玉定本人的这批新兵们，都已经知道了一小时前第一大队那里发生了什么事，于是在口口相传的消息中很多新兵都心生恐惧，当刘玉定真的出现时，甚至有人从头到尾都在颤抖。
<br/>　　集合拖延的情况不再发生，所有人都用了最快速度来到刘玉定面前，并且尽可能的表现出他们的绝无二心。
<br/>　　只不过虽然情况有所好转，新兵们想要达到刘玉定苛刻的要求实在太难了，于是在短短一天的时间内，这五万新兵的十个大队全体成员，上到中将杨天，下到最弱的菜鸟，全部都没能躲过狼牙军法处长刘玉定的恐怖体罚。
<br/>　　同时也正是这次体罚，让狼牙警戒队头疼许久的问题，一次性得到了解决。
<br/>　　这五万人的训练，终于能够进入正轨！
<br/>　　当天夜里整个军营中都四处响起新兵们的哀嚎声，白天那俯卧撑让他们虚脱之后，警戒队的那些队长竟然不给任何休息的机会，继续让他们训练，直到晚上到了指定时间才停下。
<br/>　　新兵们从未这么累过，一个个浑身酸痛的躺在营地中无法动弹，大多数警戒队成员的情况同样如此，因为在做俯卧撑的时候他们用了全力，速度快到惊人，哪怕是体罚也绝不偷懒，而紧接着的高强度训练，也让他们倍感疲惫。
<br/>　　有意思的是，狼牙警戒队队长，中将杨天竟然也与大多数人相差无几。
<br/>　　刘玉定待在中将帐篷里，面无表情的看着躺在那不停哼唧的杨天：“装什么装？那么点强度的体罚，至于么你？”
<br/>　　杨天愤怒的抬起头：“我靠！我已经很久没有受到体罚过了，别说体罚，训练也很少啊，我没装，我是真浑身都酸！”
<br/>　　听着这话，刘玉定无奈的上前给杨天捏了捏肌肉：“要不要这么弱，你可是八阶新人类啊！”
<br/>　　杨天苦不堪言：“警戒队每天那么多琐事要处理，再加上倾巢大战到现在，狼牙地界大事小事不断发生，我一个人无暇分身，哪有时间训练啊，很忙的好吗？！”
<br/>　　刘玉定点点头：“这么说来还真是，辛苦你了。”
<br/>　　“这就完了？”杨天惊讶中带着不满意：“我为了配合你，连老命都豁出去了啊！”
<br/>　　刘玉定笑着拍了拍他的肩膀：“那也是你太弱，这么点破事好几天没解决，现在时间有多紧迫，这五万人的训练竟然才刚刚开始，要不是老大未雨绸缪提前让我过来，还不知道要被你浪费多少时间。”
<br/>　　杨天沉默了下，忽然说了句没头没脑的话：“看在我比你帅的份上，我就不计较你这次公报私仇了。”
<br/>　　刘玉定的手一顿：“你啥意思？”
<br/>　　杨天从床·上一跃而起，哪有半点浑身酸痛的样子，冲着刘玉定举了举拳头威胁道：“再有下次看我揍不死你！”
