<br/>　　片刻后，衣扣一颗颗被解开，红裙褪下，旋即一具宛如上天杰作的完美玉体，便是呈现在了眼中。

<br/>　　往下，两对硕大雪白的峰峦暴露在空气中，雪峰顶部嫣红的小樱桃，显得有些娇羞。

<br/>　　纤腰下面，是两条圆润修长的长腿。
<br/>　　“咕噜。”
<br/>　　慕清霄忍不住用手轻轻抚摸，光滑柔嫩的触感，令他吞咽起口水，仅仅是这双腿，他就能玩一辈子。

<br/>　　旋即，山洞中就响起一阵令人面红耳赤的妩媚呻吟......
<br/>　　整整一个多时辰过去，山洞中的呻吟方才停止。
<br/>　　此时，美杜莎身无寸缕，软软的趴在慕清霄胸膛上。

<br/>　　彻底征服怀中的女王，慕清霄心情前所未有的舒畅。

<br/>　　闻言，美杜莎痴痴的点了点头，满口答应。
<br/>　　在这个强者为尊的世界中，名字并没有任何意义，美杜莎这个凶名，她本就不是太在意，换了也好。
<br/>　　而且，她觉得慕清霄说的不无道理。
<br/>　　见她言听计从，慕清霄坏笑一声，手掌把玩着圣女峰，道：“彩鳞，先前，你打算跟我说什么？”

<br/>　　都怪这混蛋，一点都不怜香惜玉！
<br/>　　收回目光，彩鳞视线落在山洞中的一处角落。

<br/>　　此时，慕清霄也取出一件崭新的白袍穿上，撤去雪白纳戒上的封印。
<br/>　　他的心可没有那么大，自己的女人，怎么可能让别的人看，哪怕只是一具灵魂。
