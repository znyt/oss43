　　　　道玄真人望着擂台上的两人，欣慰的点了点头，同时心中也升起一阵担忧，两人都是青云门精英中的精英，往后必然是门内的顶梁柱，现在决出一个胜负，也不知道是好是坏。
<br/>　　“哐当！”
<br/>　　就在这时，宣布比试开始的钟鼎声响起，擂台四周的议论声纷纷停止，所有视线都不约而同的汇聚在这两名身影上。
<br/>　　随着比试的开始，其余擂台上的弟子都已经祭出法宝，散发出缤纷的色彩，显然已经开始对决，都准备速战速决，结束后前来观战。
<br/>　　而中央擂台上，慕清霄负手而立，面露风轻云淡的笑容，丝毫没有先出手的意思。
<br/>　　齐昊眉头皱了皱，不明白他用意何在，思索片刻，脸色微沉，暗道：“这小子真是狂妄至极，居然想让自己先动手，既然如此，我便如你所愿。”
<br/>　　齐昊深深的看了慕清霄一眼，知道此子狡诈多端，刚拜入青云门五年，修为便如此深厚，互相恩怨已结，绝不能放任他继续成长下去！
<br/>　　否则，后果不堪设想。
<br/>　　“慕师弟，你屡次给我下绊子，并且废我龙首峰两名弟子，身为龙首峰的师兄，我必然会替他们讨还一个公道，念在我们同门师兄弟一场，我便让你三招，三招过后，刀尖无眼，生死由天！”
<br/>　　说完，齐浩负手而立，语气平淡，面无表情，表现出一副玉树临风的模样。
<br/>　　高台上，各脉首座闻言，面色平淡，他们就是希望各脉弟子互相角逐，只有这样优秀的弟子才能脱颖而出。
<br/>　　至于齐昊耍的手段，或者一些小伎俩，只要不伤及性命，他们都是睁一只眼闭一只眼。
<br/>　　“齐昊师兄如此胸襟，不愧是龙首峰的大师兄......”
<br/>　　“齐昊师兄，你一定要替孙师兄讨还一个公道啊！”
<br/>　　“是啊，是啊，让他领教一下我们龙首峰的本事。”
<br/>　　慕清霄嘴角瞥了瞥，真是好口才，短短几句话超将自己的所作所为抛的干干净净。
<br/>　　明明就是齐昊贪图田灵儿美色，被自己破坏后，在七脉会武暗中做手脚，肆意报复自己，甚至打算用武力废了自己，真是个道貌岸然的伪君子。
<br/>　　慕清霄冷笑一声道：“不必如此，齐师兄想要赢我，修为恐怕远远不够啊！”
<br/>　　狂妄！
<br/>　　此话一出，周围顿时响起一片倒吸凉气的声音，几乎所有青云门弟子看向慕清霄的眼神，犹如在看一位白痴。
<br/>　　高台上的几位首座面面相觑，目光全都落在田不易身上，貌似再说：“这么好饿的苗子交给你，你就把他教育成这样，哪怕他是先天道灵体，也只修炼了几年，如此狂妄可不是好事。”
<br/>　　田不易察觉到各脉首座的目光，冷哼一声，道：“比试尚未开始，谁胜谁负还尚未知晓呢。”
<br/>　　其实，田不易也觉得很奇怪，在他的眼中，慕清霄一直都比较低调，在众弟子中，也未曾争过什么，只是今日为何如此反常？
<br/>　　陆雪琪对周围的辱骂声仿若未觉，美眸凝视着慕清霄，充满了自信，只有她知道，后者并非狂妄自大，确实有藐视齐昊的实力。
<br/>　　想当初，她倾尽全力拔出‘天琊’，若非慕清霄没有抵挡，恐怕难以伤他分毫。
<br/>　　水月察觉到陆雪琪的目光，望着台上狂妄的神俊青年，心中隐隐期待起来。
<br/>　　台下......
<br/>　　“这大竹峰的弟子真是不识好歹，齐昊师兄礼让是给他天大的面子......”
<br/>　　“喂，你小声点，祸从口出啊，田师叔在上面坐着呢，你不要命了！”
<br/>　　“他是白痴吗？刚进青云门几年便觉得自己能上天，这种狂妄自大之人我见多了，但现在都已经死了。”
<br/>　　齐昊听闻四周传来的谩骂声，心中颇为畅快，眼底闪过一丝狰狞与得逞之色，面色平淡道：“既然如此，慕师弟可要小心了。”
<br/>　　话语刚落，齐昊低喝一声，祭出仙剑，一柄冰蓝仙剑出现在他的手中。
<br/>　　寒冰剑！
<br/>　　正是齐昊采万载冰晶修炼修炼而成，剑体银白，威力绝伦，与他的寒冰仙术配合相得益彰，可凝冰成墙，也是龙首峰流行修炼寒冰仙术和法宝的起因。
<br/>　　手握寒冰剑，只见齐昊手捏口诀，口中暗咏法决，顷刻间，寒冰剑寒气四溢，脚下偌大的擂台都覆盖上一层冰霜。
<br/>　　“咔嚓，咔咔......”
<br/>　　伴随着清脆的冰冻声，冰霜向擂台下蔓延，直至将整个擂台包裹，方才停止。
<br/>　　齐昊者一手，周围弟子都感受到一股寒意扑面而来，精英弟子骤然色变，他们都能看出，齐昊的修为比起上一届七脉会武强上不少！
<br/>　　这一招真是精妙绝伦，整个擂台结出一层冰霜，齐昊在战斗过程中，占尽地利与先机。
<br/>　　“恩......非常不错，苍松师弟真是培养出一名好弟子啊。”
<br/>　　道玄真人目光看着擂台，眼中满是赞赏，道：“齐师侄这一招将精湛的修为展现的淋漓精致，道行之深已经至玉清境九重，恐怕用不了多久便能真正的入道，年纪轻轻就有如此修为，前途不可限量啊。”
<br/>　　闻言，苍松道人面露微笑，眼中尽是得意，瞥了眼田不易，道：“齐昊这孩子平时修行刻苦，只是比我想象的还要出色。”
<br/>　　田不易面色难看极了，齐昊修为果然比上一届七脉会武精近许多，居然已经将太极玄清道修炼到了玉清境九重，真是大大出乎人的意料。
<br/>　　此时，田不易心中隐隐产生动摇，慕清霄先天道灵体的天资无人能比，只需要给他时间，早晚甩开齐昊。
<br/>　　现在想想，让他参加本次七脉会武，确实有些想当然了。
<br/>　　田不易目光凝重的盯着擂台，两人纠葛貌似不小，齐昊平淡的面色下暗藏杀机，必须时刻警惕。
<br/>　　若是慕清霄有什么三长两短，必然是整个青云门的损失，更是他大竹峰的损失。
