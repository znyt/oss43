　　行动前，罗昊已经跟杭山市警方做好了沟通，行动结束后，李霄力由罗昊负责带走；而李霄力名下的那些酒吧、健身中心、“碧海潮生”夜总会都交给杭山市警方去处理。
<br/>　　不管李霄力做事有多谨慎、有多狡猾，只要他做过就一定会留下蛛丝马迹，以前是没有机会，现在有机会了，必定肯定从那些地方找到证据，至于起不起诉李霄力其实已经不重要了，李霄力已经被军方接管，跟杭山市警方不再有关系。
<br/>　　李霄力左小腿上挨了一枪，伤口不致命，但罗昊还是让人把他送去了杭山市的武警医院接受治疗，全程由杭山市的特警支队和武警机动大队负责押解、看管，就算是李霄力长了翅膀也不可能跑得掉。
<br/>　　当天晚上，杭山市警方就组织警力对李霄力名下的五家酒吧、三家健身中心和“碧海潮生”夜总会展开了突击行动。
<br/>　　即便是周宋捷、刘炯、宋冬飞这些干了几十年的老警察，在看到从李霄力名下酒吧中查获到的东西，在看到李霄力那五家酒吧中用来关押“皮箱”的地下室时，还是感到深深的震惊。周宋捷不知道到底有多少人在这些地下室中，被李霄力逼着强行吞毒，然后被作为藏毒工具被送去境外，周宋捷只知道以后这些罪恶之窝将不复存在！
<br/>　　事后，媒体将杭山市警方这次突击行动评价为近十年来杭山市最成功的“打黑扫黄扫毒”突击行动。
<br/>　　罗昊原本预计是当天行动结束后，就把李霄力押解回京，但是因为李霄力在行动中受伤的关系，返回时间被拖延到了第二天早上。罗昊给了慕容飞两个小时的时间，让他回去看佟千雪，然后直接去特警支队跟他们汇合。
<br/>　　慕容飞是开着特警支队的车去佟千雪所居住小区的，早晨路上的人还不多，慕容飞并没有花费太长的时间就赶到了佟千雪的小区，原本还想着把车开进小区，可是刚打方向就看到佟千雪和佟母结伴朝着小区门口的方向走来。
<br/>　　慕容飞把车靠边停下，推门下车，对着佟千雪和佟母喊道；“千雪，妈！”
<br/>　　佟千雪听到有人在喊自己的名字，先是一愣，随后循着声音看去，当看到站在警车边的慕容飞时，脸上露出不可置信的神色，佟千雪怎么都没有想到，已经返回部队的慕容飞会突然出现在这里。
<br/>　　“你怎么来了？”
<br/>　　佟千雪走到慕容飞身边，打量着慕容飞。
<br/>　　佟母这个时候也是走了过来，对于慕容飞这个女婿，她和佟千雪父亲都是很满意的，不说他帮助自己家解决了一直都没解决的难题，关键是能让自己女儿满意并且愿意交往，这就比什么都重要了。
<br/>　　“本来我还带着千雪出去走走，顺便看看有没有新鲜的蔬菜，现在你来了，就陪千雪聊聊吧。”佟母微笑着对佟千雪说道：“你陪慕容吧，我自己去就行了。”
<br/>　　佟母也知道慕容飞从部队回来一趟不容易，也就打扰小两口，自己一个人往大卖场的方向走去。
<br/>　　“上去坐坐？”佟千雪对慕容飞问道。
<br/>　　“不了。”慕容飞摇了摇头，说道：“我一会儿就要走，上午就要回部队去。”
<br/>　　“这么快啊。”佟千雪略显得有些失望，看着自己面前一身迷彩服的慕容飞，说道：“没看出来，你穿迷彩服的样子还蛮帅的。”
<br/>　　佟千雪第一次跟慕容飞在欧洲相遇，战术中队是隐瞒了身份在欧洲行动的，所以慕容飞穿得也不是迷彩服；后面见面，也都是慕容飞休假的时候，穿得也都是便服，这还真实佟千雪第一次看到慕容飞穿着迷彩服出现在自己面前。
<br/>　　“那是必须的必。”慕容飞一点也不谦虚的说道：“天生的衣架子，穿什么都帅。”
<br/>　　“臭美！”
<br/>　　佟千雪在慕容飞身上拍了一下，问道：“你不是回部队了吗？怎么又突然回来了？”
<br/>　　“有行动，跟杭山市警方的联合行动就又回来了。”慕容飞说道：“想你了，就请了两个小时的假来看你，还好赶上了，如果我晚一点，估计你跟妈就出去了，我就走空了。”
<br/>　　慕容飞没有告诉佟千雪自己是因为担心她的安全，才特意赶回来看她，她现在正怀着孕呢，慕容飞担心自己说多了，会影响到佟千雪；慕容飞没有说，但是佟千雪在心中也有了一个简单的猜测，她猜不到具体的原因，但是她能猜到，慕容飞来看自己绝对不是他表面说的理由。
<br/>　　看到了佟千雪没有什么意外后，慕容飞也算是放下了心来，又忍不住对着佟千雪唠叨叮嘱了几句后，就向她告辞，准备离开。
<br/>　　“等我从T国回来，我再给你的‘研究所’打电话行了吧？”佟千雪白了慕容飞说道：“我又不是小孩子了，又不是第一次出国旅游，你不需要什么都要叮嘱一遍啦。”
<br/>　　“你不是小孩子，但是你肚子里还有个小的呢。”慕容飞捏了捏佟千雪的鼻子，说道：“我走了，自己注意安全，有事情就去大使馆，在国外，还是自己国家的大使馆更可信。”
<br/>　　“嗯，我知道了。”佟千雪点头应道。
<br/>　　同样的叮嘱，虽然慕容飞说了好几次，但佟千雪还是乖乖的应了下来，因为佟千雪知道，如果自己不答应，慕容飞肯定还会对着自己唠叨。
<br/>　　慕容飞驾驶着汽车掉头离开，佟千雪就这么站在小区门口，一直看着慕容飞驾驶的那辆警车完全从自己视野中消失，在转身返回了小区。轻轻叹了一口气，还有差不多一年的时间，慕容飞应该就到强制退出作战部队的年限了吧？佟千雪幽幽地想着。
<br/>　　……
<br/>　　这次“打黑扫黄扫毒”行动取得了圆满的成功，参加行动的警员都得到了嘉奖，而主要嫌疑人李霄力杭山市警方没有做更多的透露，因为他已经被龙隐部队带去了京城，龙隐部队需要从他身上挖出关于玛丽科娃的消息。
