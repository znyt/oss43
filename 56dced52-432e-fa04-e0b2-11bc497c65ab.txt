　　这会儿就在他们尝试破阵而入的时候，里面那些大佬们仍然好端端坐着，头也没有抬一下。
<br/>　　“他们大概是算好了，他们祭祀的时间，一定比我们破阵的时间短，因此，这阵法不具备任何反噬之力，只是消耗，维持足够的时间，等仪式结束，东岳大帝降临，我们再也不能怎么样了。”
<br/>　　林三生目光灼灼地看着叶少阳，沉声说道：“我们要集中全力，撕开封印！”
<br/>　　叶少阳环顾大伙，“那也没什么好说的了，上吧。”
<br/>　　碧清已经看到苏沫了，眼中闪着凶光，立刻将炼尸缸抛起，悬在结界顶端，自己也化作一朵莲花，进入炼尸缸里里，然后就看到猩红色的煞气，从炼尸缸里一圈圈飞出来，落在结界上，接触的地方立刻泛起一层波纹，与煞气互相消耗着。
<br/>　　“你们也去帮忙！把修为集中在我的剑上！”叶少阳说完，拔出七星龙泉剑，念了咒语，将宝剑掷出去，直接插进了结界里，只留下剑柄在外面。
<br/>　　小青小白美华瓜瓜包子炭头等等一起冲上去，围着七星龙泉剑，各自的修为尽数喷吐出来，借着叶少阳在剑锋上布置的咒法，全灌入了龙泉剑上。
<br/>　　龙泉剑不断深入，几乎要穿透结界，但是越到深处，遇到的压力也就越大，龙泉剑很快停滞不前了。
<br/>　　“七星龙泉剑，还说你不是叶少阳！”
<br/>　　苏沫一眼看到七星龙泉剑，辨认出来，冲叶少阳冷笑。
<br/>　　“哼，小丫头，你又看看我是谁！”
<br/>　　碧清的声音从炼尸缸里传出，与苏沫的声音一模一样。苏沫之前并没有看到她，此刻听到她的声音，一张脸变得煞白，咬了咬牙，对左右那些人道：“大家随我一起作法，支撑结界，千万不要放他们进来！”
<br/>　　说完自己冲到结界边上，结了个法印，作法将自身真气灌入漂浮在围墙上那些红灯笼中的一个里，红灯笼中间火苗立刻旺起来，其余那些三个门派的弟子也都加入进去，一人操控一个灯笼，或者两人一个，一时间所有灯笼都明亮起来，绕着结界四周旋转，连成了某种阵法。
<br/>　　正在操控七星龙泉剑的叶少阳，顿时感觉到结界的强度增加了，七星龙泉剑僵在结界中间，无法再往前挺近。
<br/>　　“叶少阳，你个大逆不道的叛徒，居然来骚扰祭拜帝君的大礼！你可真是越来越堕入魔道了！”
<br/>　　慈心师太冷冷地望过来，嘴脸让人作呕。
<br/>　　叶少阳也不开口，跟大伙一起继续作法，消耗着结界的力量，这是眼前唯一的办法。老郭的那些装置，面对结界没什么用处，只能在一边干站着看。
<br/>　　小九和林三生因为担心被认出来，一直站在边上观察局面，没有妄动。
<br/>　　看帝君庙里，通玄道人的气息越来越微弱，只剩下哼哼了，帝君的神像却是不断明亮起来，身上好像镀了一层金，金光忽明忽暗地。
<br/>　　叶少阳虽然不了解仪式过程，但也隐约感觉到差不多了，心中暗自着急。
<br/>　　小九看在眼里，对他说声：“我来！”也顾不上隐藏身份了，当下现出本尊，一下跳到结界上，九条尾巴一起插进七星龙泉剑撑开的裂缝里，仰面朝天，轻吟一声，体内强大的妖力一瞬间释放出来，缺口四周那不断涌过来的结界之力，朝四周翻滚出去。
<br/>　　缺口越来越大。
<br/>　　叶少阳抓住时机，将更多真气倾注在七星龙泉剑上，用力搅动起来，小青小白等一干人也齐心协力，僵持半晌，只听的轰的一声，结界被打穿了一个窟窿。
<br/>　　“少阳，快！”小九大喊一声，到这时，她一口气也泄尽了，需要缓一下，稍微一放松，四周的气息不断用来，窟窿眼看着就要填上了。
<br/>　　叶少阳飞身上前，抓住七星龙泉剑，从裂缝中纵身穿过，一下子钻了进去，在地上滚了一圈，起身回头看去，窟窿已经合并了。
<br/>　　结界毕竟是以黎山老母三个一流强者布置出的，力量之强，在没有叶少阳的七星龙泉剑的帮助下，就算有小九当主力，一时间也是没办法再次强行穿透，只能按照常规办法继续消耗结界的力量。
<br/>　　操，就我一个进来了？
<br/>　　面对虎视眈眈的苏沫等人，叶少阳微微有点怂。
<br/>　　“少阳，那些灯笼能加强结界，打碎他们！”小九在外面喊叫。
<br/>　　叶少阳回过神来，把七星龙泉剑归鞘，解下勾魂索，朝最近的一盏灯笼冲过去。
<br/>　　几个大佬还在作法，进行祭祀仪式，没法移动，连话都不能说，于是苏沫等人带头来阻击叶少阳。
<br/>　　叶少阳刚挑起来，朝一个灯笼挑过去，突然横空飞来一支拂尘，抓住了勾魂索，一瞬间，叶少阳感觉到了强大的力量，将勾魂索往那个方向拽过去，转头一看，是苏沫。
<br/>　　身边还跟着很多人，有些是空界的法师，有些是人间弟子，叶少阳法器被困，左手一抬，打出了三张暗金神符，口中念咒，神符在空中激活，化作六道天雷，在众人身前炸开，将所有人逼退。
<br/>　　“叶少阳，好久不见啊！”苏沫咬着牙，冷冷说道。
<br/>　　“你比之前厉害多了。”叶少阳感知着柳条上的劲道，感慨说道，“但你不是碧清对手，你当心吧。”
<br/>　　苏沫哼了一声：“我而今已是梨山弟子，纵然是芙窕仙子，又能如何。”
<br/>　　这结界能挡住生灵，但是挡不住声音，碧清在外面将他们的对话听得一清二楚，冷冷道：“大言不惭，今日我就要炼了你！”
<br/>　　苏沫也回骂道：“大言不惭，你只是我前身，却是远古之人，不该出现在这里，而今我才是我！”
<br/>　　说着单手结印，朝叶少阳拍过来，两人法器交缠在一起，一只手斗起法来。
<br/>　　碧清一边用煞气腐蚀结界，一边怒道：“你居然改换门楣，有什么资格做我！”
<br/>　　叶少阳听着两人的对话，有种很奇怪的感觉，就像一个人对着镜子自说自话，他突然想到一个问题：如果李浩然在这里，他会帮谁？
