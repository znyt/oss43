　　紫色，无边的紫色，除了紫色，再无他物。
<br/>　　这是一个充满紫气的世界，这个世界并没有一个生物，只剩下了空荡荡的紫色，看上去是那么高贵，那么幽深，那么无涯。
<br/>　　无限紫气形成一个浩瀚无边的世界——鸿蒙界！
<br/>　　这就是万界之始，大道之源的鸿蒙界。
<br/>　　一点光耀突然出现在这无尽紫色风暴之中，这光耀陡然间放射出耀眼的紫光，当这点亮光无比强烈之时。
<br/>　　“嘭……”
<br/>　　一阵猛烈的爆炸风暴，顿时无边紫气空间被这道光耀吞噬。
<br/>　　无限耀眼的空间照耀下顿时闪现出一道更加高贵的紫色线条，它显得那么幽远，绽放着阵阵紫光，仙音袅袅，玄妙无比，好像大道无涯。它就是道的化身。
<br/>　　这道神秘紫气正在缓缓吞噬着其他紫气，一点一点，越来越快，慢慢壮大……
<br/>　　无边紫色空间中不知道有多少这样的紫气，他们都在慢慢吞噬着对方。这就是一场争夺，不是你死，就是我亡，至于最后的结果是什么，谁也不知道。他们只知道吞噬对方会有巨大无比的好处。于是，他们借助那一点点本能，在疯狂吞噬。
<br/>　　岁月还是一如既往的不值钱，况且鸿蒙中也没有时间的概念。
<br/>　　不知过了多久，单调的紫色已经开始变少了，此刻，空间中却又多了一种灰茫茫的气体。
<br/>　　灰色气体渐渐多了起来，而紫色气体却慢慢少了。但，剩下的紫色气体，全都耀眼无比，比之前先得更加高贵，更加无涯，散发的气息也更加玄奥了。
<br/>　　直至……天地全部被无边灰色气体包围，吞噬，无边紫色只剩下仅仅三千零五十条粗大无比的金紫色气体。之后，他们再如何吞噬对方，这种数量都不会减少，好像他们注定要剩下这么多，一根不多，一根不少。
<br/>　　它们就是道的本源，它们就是道的化身，它们就是道。
<br/>　　突然，其中三千条紫气相互交织，相互纠缠，似相互吞噬，又似相互弥补。渐渐地，它们发现谁也奈何不了谁，于是，它们开始了合作，相互弥补对方。同样地，另外五十条紫气也开始了相同的蜕变，不过，在吞噬的时候，有一条紫气瞬间遁入虚空，消失不见。此为“一线生机”！
<br/>　　不知过了多久，也许是一瞬，也许是永恒，灰茫茫的空间之中出现了一大一小两个闪耀着无边紫金光芒的光源，瞬间就照亮了这无尽的灰色空间，让这空间不再黑暗。
<br/>　　突然，大的一团紫金光源中飞出三千份小的光源，三千份小的紫气光团中，充斥着种种法则的玄妙，并迅速向着混沌空间飞去，密布苍莽各处，静静地等待着什么！
<br/>　　分出小紫气光团的大紫金光源一瞬间好像萎靡了许多，虽然依旧充斥着威严、高贵、无涯的气息，但相较之前已经小了不少。
<br/>　　“嗖——”
<br/>　　突然，大的一团光源一闪即没，瞬间遁入虚空，消失不见了。
<br/>　　“嗖——”
<br/>　　又是一声，小的一团光源也随着遁入了虚空，眨眼间消失不见。
<br/>　　这一大一小两光源，一曰“鸿蒙大道”，一曰“混沌天道”。（以后为了方便，直接称大道与天道，不过，此天道并非后世洪荒天道。）
<br/>　　鸿蒙初判混沌现，大道天道遁虚空。
<br/>　　“轰——”
<br/>　　在这大道天道消失后的瞬间，原先还风平浪静的灰色空间，瞬间就变得汹涌澎湃起来，无数灰色风暴在空间中肆掠，无边灰色气流变得嚣张起来。同时，空间中响起九道大道之音，像是在悲鸣，又像是在庆祝。
<br/>　　它们这是在用自己的方式来哀叹鸿蒙的离去，欢庆混沌的初生。
<br/>　　……
<br/>　　混沌中不记年，不知过了多久，混沌中开始变得平静，虽然时时有混沌风暴刮起，但相较之前已经变得平静许多了。而就在这时，混沌中也开始陆陆续续出现了一个个大蛋，直至出现整整三千个巨大的混沌蛋，它才不再增加。
<br/>　　如果仔细看，就会发现这三千个大混沌蛋正是之前大道分出的三千大道法则所密布的位置。显然，这三千混沌巨蛋正是三千大道法则所化。
<br/>　　他们就是以后威震混沌的三千神魔。
<br/>　　在这浩荡的混沌之中，混沌依旧浩瀚无边，深邃而寂静，无边混沌气流在翻滚，密布在苍茫各处。
<br/>　　无边的混沌气流不断地汇聚在那三千大蛋周围，给予它们足够的养分！三千个巨大的蛋在缓缓吸收这四周的混沌之气，一天又一天，一年又一年，永不停息，无穷的岁月流淌，而蛋中的威压却变得越来越重。
<br/>　　在这些大蛋的周围，有着一种磅礴的大道法则之力以一种近乎实质的方式显现着，它们缠绕在大蛋之上，好像在描绘着什么，又好像在诉说着什么，玄之又玄！
<br/>　　混沌之中，混沌大蛋有大有小，在混沌的中心，一个巨大无比的混沌蛋屹立其中，周围充斥着霸道至极的力度，它是混沌中最大的一个蛋，在其下方，一座同样巨大无比的青莲，散发着种种玄妙无比的法则之力，正在滋养着其上方的大蛋。
<br/>　　这是四大混沌至宝之一的三十六品混沌青莲！
<br/>　　在大蛋的内部，一个巨大的巨人模样在缓缓成形，一柄霸道至极的斧头若隐若现。
<br/>　　这是另一个混沌至宝——混沌开天斧！
<br/>　　同为四大混沌至宝的另外两个是造化玉牒与混沌珠！
<br/>　　还有几个巨大的蛋，他们虽然没有中心的大蛋那般大小，但又比其他的蛋要大。他们上方的法则之力同样玄妙无比，玄之又玄。其周围的混沌气流十分浓郁，化作了液体，被其吸收着！
<br/>　　这些大蛋之中有一个，周围的混沌气流浓郁到近乎实质，在源源不断的注入到巨蛋之中，来多少，就收多少，其炼化速度较之其他几个要快上许多！
<br/>　　缠绕在他周身的，是一种诡异的法则之力，那是一种黑白两色细线，时而黑色，时而白色，时而黑白相接，时而若隐若现，好像在阐述着什么，又好像没有，他无时无刻不在，又好像无时无刻都在。有白就有黑，有黑必有白，玄之又玄！
<br/>　　这是因果法则！
<br/>　　然而，在没有任何存在知道的情况下，在那混沌色大蛋之中一块寸许大小的白玉色圆印正在缓缓旋转，吸收着外面狂涌而来的混沌之气。圆印之上，“元始”两字像是在阐述这某种大道，又像是在吸收着某种大道之力，它就像是众道之源。
<br/>　　而在其元始圆印空间内，一个微弱的意识正在缓缓吸收着经过元始印过滤的混沌之气，他正在缓缓壮大。
<br/>　　混沌不记年，岁月在这寂静的苍茫中缓缓流逝，混沌依旧深邃而苍茫，灰蒙蒙的一片。混沌就仿佛一个巨大无比的怪兽，它现在正在沉睡，但是，总有那么一天，它会苏醒过来，到时候，混沌就会变得无比热闹。
<br/>　　混沌之中寂静无比，它是在酝酿着什么？还是在等待着什么？
<br/>　　是酝酿着盛世，还是毁灭？等待的是希望，还是宿命？
<br/>　　谁也不知道。
