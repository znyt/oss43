　　昆仑山，玉虚宫。
<br/>　　道印天尊元始一袭白玉道袍，玄坐太空九色玄龙宝局之上，旋观胜妙，万天敬礼，龙腾虎掷，化现无论紫金光明圆象，至高无上。
<br/>　　在他身边，是那一袭白色道袍，鹤发童颜的太上道德天尊，另一边是一生青衣，手持青萍剑的通天教主。
<br/>　　他身前端坐的是那一袭彩色衣裙，满脸娇容，全身玄光绽放的女娲娘娘，当然，还有一位脸色淡然的玄天武祖。
<br/>　　高台之上的元始看了众人一眼，谁也猜不出他的心思，只听他淡淡道：
<br/>　　“天机已现，人族当兴，此乃天道机缘所至，不可违背。而天机显示，人族欲要大兴，还需要拟定三皇五帝，制定人族纲常礼仪，镇压人族气运。三皇五帝归位之际，人族自然大兴，从此气运不衰。”
<br/>　　说到此处，他顿了顿，看了众人一眼，却见众人都面无表情，显然是早已知道此事。也是，身为圣人，知道这些天机也算正常。他不停，继续说道：
<br/>　　“然而，这里的三皇五帝乃是天定，吾等不可争夺，不过，人皇出世之后，却是需要有人引导才行。”
<br/>　　说到这里，元始停了下来，便不再多说。其他人也在心中微微思量起来，算计着此番教导人皇，赚取功德一事。虽然三皇五帝他们不能够亲自出手，但圣皇之师还是可以谋取的，功德是个好东西，谁也不嫌多。
<br/>　　“大师兄，不知我兄长伏羲可否为一位人皇？”沉默一会儿后，女娲娘娘率先开口道。
<br/>　　“三皇五帝乃是从人族之中选出，伏羲大圣昔日为妖族羲皇，屠杀人族，按理来说，不能够成为人族圣皇。但，他却偏偏有着女娲师妹人族圣母的兄长这一重身份，但就算如此，想要做人族圣皇，依旧不够。”元始淡淡道。
<br/>　　“为何？”女娲有点怒了。
<br/>　　“这事你就要问玄天小友了。”元始微微一笑道，毫不在意女娲的态度。
<br/>　　女娲将目光投向玄天，就连其他几圣也将目光投了过来。被几大圣人注视着，武祖玄天心态很好，不急不慢，缓缓说道：
<br/>　　“三皇五帝乃是人族之事，伏羲为妖族大圣，自是不能成为我人族圣皇。想要成为人族圣皇，最基本的一个要求便是，他要是一个人。就这点，伏羲就做不到。”
<br/>　　女娲沉思片刻，脸上闪过一丝痛苦，像是下定某种决心，接着她沉声道：“我可以让我兄长投胎为人族。”
<br/>　　“若真是如此，那伏羲却是要选择抛弃妖族的身份，以后他只能有一个身份，那便是人族。”玄天坚定道。
<br/>　　“可以。”女娲脸色有些苍白道。
<br/>　　她身为圣人，其实早就想到了这个问题，伏羲若是抛弃了妖族的身份，那也便是抛弃了前世的种种，也就是说，以后的伏羲，只能是人族。而人族，却是女娲的子女，换句话说，伏羲今后的身份，注定他不能成为女娲的哥哥了，只能尊称女娲为圣母。
<br/>　　这也是女娲之前一直顾及的地方，但她也不得不如此，否则伏羲的身份就得不到人族的承认，若是人族不能承认他的身份，那么，圣皇之事，也就成了无稽之谈。是以，伏羲想要去争夺那圣皇之位，前世的身份，是必须抛弃的，所以，女娲才会如此痛苦。
<br/>　　然而，女娲却又不得不如此，自己的兄长为自己已经做过了太多了，她不能眼睁睁的看着自己的兄长一直如此。
<br/>　　“就算如此，想要争夺人族圣皇之位，还得看他日后的表现，日后他能否带领人族走向大兴，能否为人族做出贡献？是否真心为人族？这些都是圣皇必备的。”玄天不急不缓道。
<br/>　　“这是自然，那些不劳玄天道友费心了。”
<br/>　　女娲坚定道。她何尝不知道这些，她在意的是人族对伏羲的承认，至于之后的事情，那就是圣人之师的事情了，她自信可以搞定。
<br/>　　玄天思考一番，说道：
<br/>　　“好，既然如此，那么看在女娲造化我人族，还顶着一个人族圣母的头衔，我可以做主让伏羲具备争夺圣皇的主角，能不能成功，那就要看他日后的表现了。”
<br/>　　玄天自是知道伏羲乃是天定的人族天皇，这是大势，他只是一个混元都还不到的人物，自是没有资格更改。但他知道，女娲可不知道啊，他现在所说的，只不过是为了表演给女娲看，让女娲欠下自己人情，或者是说，让自己了去与女娲的那一份因果。
<br/>　　毕竟，自己说到底还是占了女娲的便宜才来到这个世上，自己这就欠下了女娲的因果，不是一般人族母子间的那般的因果，而是霸占人家儿子身份的这一层因果。
<br/>　　当然，因为种种原因，女娲是不知道这一层因果的。
<br/>　　“如此，那就谢过玄天道友了。”
<br/>　　女娲还是很懂礼数的，当面向着玄天抱拳谢道。
<br/>　　“道友不客气！”
<br/>　　玄天连忙闪身躲过女娲的礼数，笑话，他还想多活几年呢。女娲的两重身份，一为人族圣母，二为天道圣人，不管是那一重，对于玄天这个人族来说，都是非常尊贵的，天道圣人乃是天底下最为高贵的人，人族圣母更是相当于人族母亲这样的角色，若真是承受了这一礼，以他的实力与命格，肯定要折损许多，说不定一个反噬，直接让他吐血。
<br/>　　洪荒之中，除了道祖鸿钧，轮回平心，或者同等级的人物，还没有谁能够有资格让圣人行礼的。
<br/>　　不为别的，只因为他们太高贵了，高贵到别人根本承受不起圣人的一礼。准圣以下者，圣人只要微微一拜，便可直接被反噬得吐血身亡，只因为，他们的命比圣人低贱。就算是准圣，一般都不敢让圣人行礼，否则轻则重伤，重则气运有损，命格反噬。
<br/>　　屠圣更是一个笑话，若是你没有比圣人高贵的气运或者命格，屠圣的结果只有一个，那就是与圣人同归于尽！因为圣人是承载着洪荒亿万生灵的因果气运，行天之道，执天而行。一旦圣人死亡，那么，圣人身上的无尽气运或者因果、天道便会反噬过去，没有人能够承受得住那种反噬，就算是同等级的圣人，也可能跌落神坛。
<br/>　　当然，若是有着化解这种反噬的方法之人，那就说不定了。
<br/>　　看见玄天的动作，女娲微微一愣，这可不是玄天平时的性格啊？她心念微动，便知道了原因，不由得有些哭笑不得。好在她也并没有在此时上多加纠缠，而是将目光投向太上道主，问道：
<br/>　　“太上师兄，你以人族立人教，欠下我一分因果，可否了结？”
<br/>　　“师妹所言极是，贫道看这份因果就在伏羲天皇一事上了结，如何？”
<br/>　　太上也不想自己一直欠着一份因果，要知道，越是修为高深之辈，就越是怕因果一事，因为因果未了，会在修道路上造成各种阻碍，最后的超脱，更是要化去全部因果。如今既然女娲提出来了，太上自然也愿意了去这一段因果。
<br/>　　伏羲为圣皇，他也没有多少意见。
<br/>　　“大善！”
<br/>　　女娲听后，之前一直阴沉着的俏脸之上终于露出了一丝微笑。顿时，犹如雨过天晴，春风拂面。
