　　丸手斋本来毫不在意的坐在台上，听完刘哔的话语之后，顿时惊讶的从椅子上站了起来。
<br/>　　大战在即，他需要有关于‘枭’更多的资料，虽然给他提供资料的只是一个三等新人搜查官。
<br/>　　“恩，是在二十区的一家名叫古董的咖啡店里，那里面几乎都是我无法辨认出来等阶的高阶喰种，据说他们的店长，和‘枭’有点关系。”看了一眼丸手斋，刘哔恭敬的回答道。
<br/>　　重新坐了下来，丸手用手指轻轻敲打着银色的手提箱，似乎再考虑什么，久久没有说话。
<br/>　　他有些惋惜，如果提前几天知道这个消息，他可以做更多的事情，现在这个时间，已经有些晚了。
<br/>　　不过，这也是没办法的事情，CCG一直在追查‘枭’的踪迹，却一直没能找到什么线索，一个刚从学校毕业的新人，能做到这一步已经让他非常意外了。
<br/>　　看了一眼刘哔，丸手眼神有些赞赏。
<br/>　　“好了，这件事情非同小可，我需要禀报上去，你们继续开会，原定的计划暂时不变。”交代了一句，丸手将话筒递给主持人，自己从旁边的侧门急匆匆的走了出去。
<br/>　　刘哔也重新坐了回去，原定的计划达成，还是让他心中有些高兴的。
<br/>　　会议继续开始，主持人继续在上面絮絮叨叨的讲着一些注意事项，还发放了一份保密合同，让众人一一签署了。
<br/>　　等到主持人宣布会议结束的时候，时间已经到了正午了，众人都是松了口气，提着自己的箱子从会议室中走了出去。
<br/>　　木嶋班也收拾好了东西，跟着人流走出了大门，刘哔和旧二多福走在最后面，却突然被拍了一下肩膀。
<br/>　　身体稍稍一僵，刘哔嗅到了一股熟悉的清香味，转过头去，看见铃屋什造略有些病态的妆容。
<br/>　　“嘿，你是说，上次我们一起击杀的那个喰种吗，原来它竟然和枭有关！”什造身后跟着几个黑西装，似乎是她的组员，却表现出一副以她马首是瞻的样子。
<br/>　　可她明明只是个新人，甚至前几天和刘哔第一次外出搜查的时候，都还没有自己的库克因。
<br/>　　“并不是那次，话说，这是那家伙制造出来的库克因吗？”忍不住捏了一下什造精致的鼻头，刘哔的视线落在她手中的银色密码箱上。
<br/>　　虽然只是一个低级喰种制造出的库克因，但落在什造手上，也能发挥出不小的威力。
<br/>　　这可是个让他不开心的事情，什造的实力越强，他就越不容易对她下手了。
<br/>　　“是啊，威力还很不错呢，你要不要看一下？”皱了皱鼻子，一提及自己的库克因，什造便一脸兴奋。
<br/>　　心中一动，刘哔点了点头，他觉的这倒是个不错的机会。
<br/>　　既然是展示库克因，那自然要到空旷一点的地方去吧，至少也是要离开CCG的总部大楼，到时候他或许就能找到机会吞噬掉什造。
<br/>　　“哈，那可要看好了哟！”令他没有想到的是，什造根本没有什么顾忌，直接在人来人往的过道中打开了库克因。
<br/>　　一道长长的鞭子出现在她手上，鞭身上长着倒刺，微微有些暗红的颜色，反射着一股妖异的光芒。
<br/>　　过往的人群顿时狼狈的四散躲开，这东西可不是开玩笑的，万一剐蹭到人身上，难免会被伤到。
<br/>　　四周的搜查官都向什造投来恼怒的目光，但她像是丝毫没有察觉一般，任性的拿着鞭子，甚至还凌空挥舞了几下，引起了一连串的音爆声。
<br/>　　“这可是山田先生花费了很长时间才制作出来的，怎么样，厉不厉害？”有些痴迷的看着自己手中的库克因，什造的一双眼睛开心的弯了起来。
<br/>　　无语的看了一眼什造，刘哔露出一抹温柔的笑意，随手揉了揉什造的头发，没多说什么。
<br/>　　“好了，时间不多了，我们还得回去准备接下来的任务。”木嶋式走了过来，冲什造咧了咧嘴，算是笑过，随后向刘哔提醒道。
<br/>　　应了一声，刘哔和什造告了声别，跟随木嶋式一路回到宿舍中。
<br/>　　一个下午，都是在无聊中度过的，木嶋式没有回家，而是留在这里帮他们做着一些准备。
<br/>　　如果抛开面向不谈，木嶋确实是一个十分称职的班长，最起码，他对自己班内队员的关心作不了假。
<br/>　　有过几次任务经验的立智和阳牧还好，虽然依旧有些紧张，但也知道他们应该做些什么。
<br/>　　而旧二多福却表现的有些不堪了，整整一个下午的时间，他都是在惶恐不安中度过的，甚至写遗书的时候，还非常不争气的哭了起来。
<br/>　　无奈之下，木嶋也只能待在他的身边，循循善诱的开导着他，开导的同时，也在担心刘哔的心理状况。
<br/>　　毕竟刘哔和多福都是同一批进入CCG的新人，论经验和能力两人都是半斤八两。
<br/>　　但他自从回到宿舍之后，就把自己关进了房间，木嶋就算再担心，也没有什么办法。
<br/>　　倒不是说谁懦弱，没有真正经历过战争的人，在面临生死危机的时候大都是一副不堪的样子，当年的木嶋式，也是从这种阶段过来的。
<br/>　　只是他却不曾想到，之所以一回来就把自己关进房间，可不是因为恐惧，而是刘哔需要一些私人空间，来为接下来的行动做一份详细的计划。
<br/>　　此时他正坐在桌子前，用笔在一副城市地图上标注着什么。
<br/>　　……
<br/>　　大概晚上6点多钟的时候，敲门声响起，刘哔将已经制作完成的地图收入空间戒指中，开门走了出去。
<br/>　　“时间已经差不多了，赶紧吃点东西，我们要去总部集合了。”站在门外，木嶋的表情满是关切。
<br/>　　点头应了一声，刘哔从屋子里走了出来，和往日不同，餐桌上的菜肴很是丰盛。
<br/>　　鸡，鸭，鱼，肉，水果，蔬菜，应有尽有，木嶋班的众人围绕在餐桌前，脸上的表情皆是有些深沉。
<br/>　　面对着诱人的菜肴，众人都没什么食欲，因为他们知道，这也许是他们生命中的最后一顿晚餐。
