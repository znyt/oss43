        连华生见童筱颖和傅昼景渐行渐远，便与傅夜沉并肩齐行。
<br/>
<br/>        他一边跨步入了大门门槛，一边不忘跟傅夜沉感慨道：“阿沉，你真打算对童筱颖放手吗？你真打算对你弟弟婚内出轨这件事，无视于睹？再怎么说，你也得狠心点，把童筱颖从你弟弟手中夺回来啊！我觉得，不如这样，把真相告诉童筱颖。让童筱颖看清楚你弟弟的真面目！”
<br/>
<br/>        “……”傅夜沉静默地听着连华生的问话，一声不吭地往前走。
<br/>
<br/>        连华生说着说着，就没话可说了，因为，他跟他说，就跟自己在自言自语一样，但他心里还是忍不住替他傅夜沉感到郁郁寡欢。
<br/>
<br/>        傅夜沉此刻到底是什么想法，连华生一点都捉摸不透。
<br/>
<br/>        倘若换位思考，如果换做是他连华生遇上这档子的事情，他又该怎么做？
<br/>
<br/>        自己心爱的女人不爱自己，爱的是自己的弟弟，而弟弟又婚内出轨了，自己心爱的女人又不知道。
<br/>
<br/>        连华生忽然觉得，他应该要为自己这出生入死的兄弟做些什么。
<br/>
<br/>        今晚，傅家大庄园的“御厨”做了一桌子丰盛的晚餐。
<br/>
<br/>        围在一起吃饭的人，少了一个苏漫雪，却多了他连华生。
<br/>
<br/>        童筱颖见连华生看起来似乎跟傅老太爷傅炎廷的关系很好，忽然间也能理解，那次连华生为什么要帮着傅老太爷傅炎廷一起“试探”她和苏漫雪了。
<br/>
<br/>        “华生的父亲和小沉的父亲是生死之交，而华生和小沉也是生死之交，所以，华生对于我们傅家来说，就跟亲人一样。”傅夜沉的母亲冷太太微笑着向童筱颖和傅昼景介绍道。
<br/>
<br/>        童筱颖还发现，冷太太似乎并未因为苏漫雪未到而影响到心情。
<br/>
<br/>        苏漫雪的事情，傅老太爷傅炎廷已经告知了自己的儿媳妇。
<br/>
<br/>        冷太太也感到诧异，完全没料到苏漫雪是这种女人，也感叹自己竟然也有看走眼的时候。
<br/>
<br/>        而童筱颖作为她儿子真正的救命恩人这件事情，冷太太也只能在心里感激她了。
<br/>
<br/>        苏漫雪似乎对于他们傅家的人来说，就像是一场过客，不，不能说是过客，应该比喻为“妃子”更为妥当。
<br/>
<br/>        受宠的时候，可以让她无法无天，不受宠的时候，就只能打入冷宫了。
<br/>
<br/>        苏漫雪自作自受，把自己作进了监狱。
<br/>
<br/>        但是，童筱颖对苏漫雪被关进监狱里的这件事情毫不知情。
<br/>
<br/>        傅夜沉不打算告诉她，傅昼景就更加不会再提及了。
<br/>
<br/>        “林贵，去把二少爷和二少奶奶的澳洲PR（永久居民）卡拿过来。”傅老太爷傅炎廷命令道。
<br/>
<br/>        林贵立即将一个用塑料文件袋装好的资料，递给了傅昼景。
<br/>
<br/>        傅昼景接过文件袋后，很客气地谢道：“谢谢爷爷。”
<br/>
<br/>        而童筱颖再看林贵的时候，心里怎么着都不是滋味。
<br/>
<br/>        因为，他林贵的女儿林若琴，是劈腿她前男友的小三，而且还是杀害她朋友马智杰的凶手。
<br/>
<br/>        “小景，到了澳洲后，和筱颖好好过日子。”傅老太爷傅炎廷语重心长地说道。
<br/>
<br/>        傅昼景点了点头，下意识地拉住童筱颖的手，与童筱颖深情地对视。
<br/>
<br/>        童筱颖看傅昼景的眼神里，总是透着一丝淡淡的爱慕之情。她的眼神柔情似水，又有明月般的微光。
<br/>
<br/>        一个女人爱一个男人，是什么样子，大致看到她童筱颖就能明白。
<br/>
<br/>        坐在傅老太爷傅炎廷身旁的傅夜沉，正好是坐在童筱颖和傅昼景的对面，所以，他们两那“恩爱”的小动作，他尽收眼底。
<br/>
<br/>        他以为自己的心早已痛到麻木，但是看到她和他的亲密无间，他的心还是会隐隐作痛。
<br/>
<br/>        傅夜沉垂下眼帘，不再去看童筱颖和傅昼景。
<br/>
<br/>        开席后，傅昼景把鱼肉的刺剔除后，夹到了童筱颖的碗里，并温柔地说道：“筱颍，这是你最爱吃的鱼。”
<br/>
<br/>        倘若，他没记错的话，小时候，她很喜欢去河里抓鱼，然后就在岸边生火烤着吃。
<br/>
<br/>        其实，那个穿着他妈妈送的粉色连衣裙的小女孩，随着他妈妈的去世，便一直与他妈妈一起尘封在了他的心底。
<br/>
<br/>        更是他不曾跟任何人提及的一抹白月光。
<br/>
<br/>        “你还记得！”童筱颖微微偏头，眉开眼笑地对上了傅昼景的目光，心底一阵欣喜。
<br/>
<br/>        她以为，他像她一样，差不多快要忘记了小时候在一起的那段美好时光。
<br/>
<br/>        傅昼景总是把最好吃的菜，往童筱颖的碗里夹。
<br/>
<br/>        连华生坐在一旁，自顾自地吃饭，却在心里坏坏地暗想：“待会，看看你两还能不能‘秀恩爱’，有好戏看了。”
