夏夕绾大概知道为什么妈咪会挑中李文清做徒弟了，李文清心里博爱，是一个真正的医者，他的性情又简单而洒脱，十分适合延年益寿的针法。
<br/>
<br/>        “院长，你师傅是不是曾经在这个枢密研究院里留下了什么东西？”
<br/>
<br/>        夏夕绾直奔主题了，她一直在等李文清院长回来，因为林婶说了妈咪留了东西给她。
<br/>
<br/>        李文清院长面色一变，“夕绾，你是怎么知道这件事的？”
<br/>
<br/>        “有人告诉我的，院长，你可以将这个东西交给我吗？”
<br/>
<br/>        李文清院长沉思了片刻，“当年我师傅失踪的时候曾经说过，将来会有人来取这个箱子，虽然我不知道你是怎么知道的，但你跟我来吧。”
<br/>
<br/>        这么容易？
<br/>
<br/>        夏夕绾真的没有想到她会如此轻松容易的拿到了妈咪留下来的箱子。
<br/>
<br/>        ……
<br/>
<br/>        院长办公室。
<br/>
<br/>        李长清院长打开了保险柜，将一个小箱子拿了出来，“夕绾，我师傅留在研究院的就是这个箱子，不过箱子有锁，如果你能打开锁，我就让你把箱子带走，如果不能，那不好意思，我还要收回去。”
<br/>
<br/>        还有锁？
<br/>
<br/>        夏夕绾就知道这件事没有那么容易的，林婶醒来的时候可没有告诉过她关于锁的事情。
<br/>
<br/>        夏夕绾看了一下箱子的锁扣，她瞳仁一缩，这不是一般的锁，锁芯直接镶嵌在了盒子里面，里面应该有精细而复杂的机械轴承，根本就不是用钥匙或者是密码打开的，更类似上古时期的八卦图。
<br/>
<br/>        夏夕绾研究了片刻，然后拿出了随身携带的银针，她将细长的银针推进了锁眼里。
<br/>
<br/>        李文清院长赞许的点头，“不错，这个箱子是要用银针打开的，打开的方式就是一套针法，我师傅是神仙一样的人物，她留下来的东西都是价值连城的珍宝，不过能不能拿走就要看你的本事了。”
<br/>
<br/>        夏夕绾屏住了呼吸，她插入第一针的时候指尖已经感觉到了里面的开关在运转，她指尖轻挑，将针法化为机械轴承的运转轨迹，咔嚓一声，第一关打开了。
<br/>
<br/>        李长清院长鼓掌，“不愧是我看中的徒弟。”
<br/>
<br/>        夏夕绾拿起第二针，不久后，又咔嚓一声，第二关打开了。
<br/>
<br/>        “漂亮！”李长清院长双眼发亮。
<br/>
<br/>        夏夕绾已经出了一头的汗，随着闯关等级的增加，其中蕴含的针法越发的博大玄妙，她感觉到了吃力。
<br/>
<br/>        咔嚓一声，第三关又打开了。
<br/>
<br/>        李长清院长竖起大拇指，“厉害！”
<br/>
<br/>        夏夕绾只觉得喉头腥甜，胸膛里好像有一股气血要涌出来了，她学医这么久，现在就感觉自己置身在了一片汪洋大海里，而她只是一个漂流的浮萍，因为实力不够而强行打开宝箱，她现在气血翻涌。
<br/>
<br/>        夏夕绾瘫坐在了椅上，强行压下喉头的腥甜，她摆了摆手，“我不行了，下一关我肯定过不了，我放弃。”
<br/>
<br/>        李长清院长将宝箱放回了保险柜里，他笑眯眯道，“夕绾，你能连闯三关还真是惊到我了，没关系，你可以下次再战！”
