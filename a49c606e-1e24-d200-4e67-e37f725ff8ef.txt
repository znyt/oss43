<br/>				见她抱着孩子，轩辕墨泽便走到飞船前头去，他往远处看着，良久，仿佛想起了什么一般，回头道：“对了，原本陌尘是要跟我们一起来的，不过我拒绝了，但我想，他应该是会先去蓬莱仙岛那边。”<br/>　　<br/>　　“陌尘啊！”凤九点了点头，倒也没觉得奇怪：“由着他吧！以他的实力与心性，那登仙梯他应该也是可以走得上去的。”<br/>　　<br/>　　她抱着孩子走到他的身边，道：“我们去看下你父亲，再送凤夜他们回去，然后就直接往蓬莱仙岛去吧！”<br/>　　<br/>　　“嗯。”轩辕墨泽点了下头。<br/>　　<br/>　　后面的凤夜走了上来，有些不解的问着：“小九九，那登仙梯是什么样的地方？登仙梯上面又有什么？你们都要去那里做什么？”<br/>　　<br/>　　凤九微微一笑：“那是一个可以通到另一片地方的一条路，但只有实力以及心性很强的人才能走得上去，你回去后好好修炼，将来也可以去那里一试。”<br/>　　<br/>　　“那冷华他们也都上得去吗？”他好奇的回头看了一眼罗宇等人。<br/>　　<br/>　　凤九抿唇一笑：“他们上不去，但我有办法带他们过去。”看着他欲言又止的神情，凤九就知道他想说什么了，于是摸了摸他的头，道：“你还小，好好在这边修炼，不愁没机会。”<br/>　　<br/>　　“可是灏儿更小。”他小声的说着。<br/>　　<br/>　　“灏儿不一样啊！正因为太小，我才得将他带在身边。”她轻笑着，道：“好了，你看，阳阳自己在那边盘膝修炼，你也不能偷懒，要不然以后你的实力比阳阳还要弱，总要他保护你的话，那是很丢脸的。”<br/>　　<br/>　　“我才不会比阳阳弱，我会比阳阳厉害的。”他连忙说着，也跑了回去，来到阳阳的身边坐下，也跟着一起修炼起来。<br/>　　<br/>　　飞船在云层中穿梭着，掠过了一个又一个的地方，在这其中，凤九一人都只在飞船上休息，修炼，以及吃着储存着的东西。<br/>　　<br/>　　直到，他们来到轩辕帝国，飞船直接停落在皇宫的皇城门口。<br/>　　<br/>　　皇城的门卫一看到那从飞船上下来的一黑一红两抹身影时，不由一惊，也顾不得行礼，拔腿就往里面跑去，迅速去通传。<br/>　　<br/>　　当轩辕国主听底下的人禀报，说是轩辕墨泽和凤九来了时，他不由微怔了下。<br/>　　<br/>　　他们回来了？不由的有些欢喜，有些紧张。这两人去了上游大陆那么久，总算是知道回来看看了。<br/>　　<br/>　　轩辕墨泽牵着凤九来到里面宫殿，两人朝主位的人行了一礼：“见过父亲。”<br/>　　<br/>　　“国主。”凤九唤着，也行了一礼，只不过她怀中抱着孩子，也就只是稍微点了下头而已。<br/>　　<br/>　　轩辕国主看到凤九手里抱着的孩子时，脸上不由的露出激动的笑容来：“这、这是你们俩的孩子？我的孙子？快快，快抱过来我看看。”<br/>　　<br/>　　听到这话，两人相视一眼笑了，凤九抱着孩子走上前去，看着他兴奋的接过左看右看后，道：“怎么不像你们两人？”<br/>　　<br/>　　闻言，两人这才一笑，说道：“这是我们捡回来的孩子。”<br/>　　<br/>　　：。：<br/>
