　　按照鬼魂的想法，既然秋镇长知道是她作怪，如果对付不了，很可能会举家搬走，躲避起来，她费了很大力气才找到阿兰，自然不可能让她就这么逃掉。
<br/>　　两人躲在角落处蹲着，也不说话，借着树丛完好地掩护住了自己。
<br/>　　叶少阳听见秋镇长剧烈的呼吸，转头看去，秋镇长一脸的紧张，于是劝他进屋，反正有没有他在场事情都是照办。
<br/>　　不过秋镇长关心阿兰的安危，坚持不肯离去。
<br/>　　好在这个世界才是阴历四月，没到夏天，草丛里也没蚊虫，待在草丛里也不算难受，为了消磨时间，叶少阳拿出了一沓符纸，开始画符，各种常用的灵符，每样画了几张，叠好后装进腰带里。
<br/>　　等了有一个小时左右，院子里还是没什么动静，秋镇长有些焦急了，问叶少阳：“她今天不会不来了吧？”
<br/>　　“有可能不来，不来就明天，再等等看。”
<br/>　　叶少阳话刚落音，从中院的门外走进来一个胖胖的大妈，一边走一边左顾右盼，问道：“怎么没个人啊。”
<br/>　　“是陈妈，这老货！”秋镇长骂了一声，问叶少阳，“怎么办？”
<br/>　　“你去看看怎么回事。”
<br/>　　秋镇长于是走了过去，见面就训斥道：“什么事！我不是嘱咐过晚上任何人都不许进来吗！”
<br/>　　陈妈道：“老爷息怒啊，是夫人让我来的，夫人煮了一锅鸡汤，让我送一点给姨太太补补身子。”
<br/>　　“行了，跟我来吧。”秋镇长不疑有他，带着陈妈往阿兰的房间走去。
<br/>　　刚到门口，身后突然想起一声：“等等！”
<br/>　　两人站住，转头看去。
<br/>　　是叶少阳走了过来，走到陈妈面前，盯着她手中的食盒说道：“陈妈，这里是鸡汤？”
<br/>　　陈妈怔怔地点点头。“夫人亲自炖的乌鸡汤，给姨太太补身子的。”
<br/>　　“好吧。我在房间里布置了东西，免得你无意中撞到，我领你进去吧。”
<br/>　　叶少阳说着上前去抓陈妈的胳膊。
<br/>　　陈妈却一反手，将食盒对着叶少阳砸了过来，鸡汤洒落，叶少阳已经轻松避开了。
<br/>　　“陈妈你！”秋镇长只喊了半句话，就看见了惊人的一幕：
<br/>　　叶少阳绕到陈妈后面，桃木剑已经拿在手中，对着陈妈后心刺了过来。陈妈就地一个翻滚避过，好像羊癫疯似的，口中大喊大叫，浑身哆嗦着，张着两手朝叶少阳扑过去。
<br/>　　陈妈很胖，肚子很大，像个水缸，她能做出就地翻滚这么灵活的动作，秋镇长是打死都不信的。
<br/>　　秋镇长怔怔地看着死命扑杀叶少阳的陈妈，脑海中想起了三个字：鬼附身！
<br/>　　陈妈的确是被鬼附身了，这一点叶少阳虽然有些感到意外，但也在情理之中：腊梅一个鬼魂，下午刚被自己赶走，晚上敢来，肯定也会猜到叶少阳有过一番布置，如果是自己魂体前来，很容易中埋伏，但是附身在人身上就不一样了，说白了就是找个人的肉身当挡箭牌，这样一来，那些针对鬼魂的法器和法药就会失效。
<br/>　　眼看着两人缠斗，秋镇长深吸一口气，平复了一下情绪，然后从口袋里摸出了一把盒子枪，瞄准陈妈，口中说道：“叶先生，我听你指令！”
<br/>　　叶少阳瞥了他一眼，道：“收起来，用不着！”
<br/>　　如果为了抓一个鬼，反而牺牲另一个人的性命，那叶少阳的法师也是不用当了。
<br/>　　叶少阳当下抖擞精神，施展出天罡步，与陈妈周旋，寻找破绽。腊梅附在陈妈身上，虽然避过了一些法器，不过也简直了鬼魂灵动的特点。叶少阳虽然只有一成法力，不过步法和身手还是有的，对付一个幽魂附身的人那是绰绰有余，瞅准机会，一脚将陈妈绊倒，立刻扑上去。
<br/>　　陈妈倒地之后，双手对着叶少阳直插过来。被鬼附身的人力道极大，叶少阳不跟她正面硬刚，躲过之后，从背包里抓出了墨斗，抓住线头，用力一甩，涂抹过朱砂的红线弹在了陈妈手臂上。陈妈“啊”的一声叫，一只手抽了回去。
<br/>　　叶少阳不退反进，故意后撤，陈妈立刻坐起来，叶少阳就地一滚，已经绕到她身后去，同时左手抓住了她的头发，右手中的墨斗对着印堂又弹了一下，陈妈哆嗦了一下，倒在地上。
<br/>　　叶少阳仍然不急着进攻，纵身又回到正面，陈妈再度坐起来。
<br/>　　叶少阳左手早抓了一把香灰在手里，摊开手掌，大喊一声：“三清开眼！”
<br/>　　用力吹了一口气，将香灰尽数吹在了陈妈脸上，大部分糊住了眼睛。
<br/>　　“啊！”陈妈双手去揉眼，叶少阳等的就是这个时机，墨斗在两手中飞快流转，在陈妈两条腿上缠了几道，同时一张灵符贴在了陈妈的额头上，这才退后，双手结印，灵符燃烧起来，陈妈在地上翻滚起来。
<br/>　　不一会工夫，一道人影从陈妈身上飞出，想要逃走，叶少阳手一抖，一道渔网一样的东西在空中展开，将人影兜在中间，用力一拉，收紧网口，坠落在地上。
<br/>　　叶少阳长出了一口气，用手抹了一把额头上的细密的汗珠，很装逼地摇头，叹了口气：“费劲。”
<br/>　　从与陈妈缠斗开始，叶少阳一直游刃有余，掌控着斗法的节奏，最终困住了陈妈的肉身，用地火符炙烤她躯体内的腊梅的鬼魂，让腊梅耐不住煎熬，只能放弃附身，刚要逃走，被叶少阳用血渔网抓了个结结实实。
<br/>　　实际上，叶少阳本来是有更加省事的法子，但他怕把腊梅逼急了，以陈妈的魂魄作为要挟，万一横下心来同归于尽，那真的是“一尸两命”。因此叶少阳循序渐进，每一步都让她觉得还有退路，不至于拼命，最后成功将她从陈妈身体里逼出来，然后再用老郭做的血渔网成功将她擒住……
<br/>　　这一套行云流水的进攻，看似浑然天成，实际上每一步都充满着变数，需要在短时间内给出精准的判断，这恰恰是叶少阳的长处，也是他作为一个法师，立命安身的根本。
