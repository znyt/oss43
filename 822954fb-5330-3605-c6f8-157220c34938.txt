　　“震旦！”
<br/>　　鲜血魔尊看向那震旦魔尊，眼神都是极为忌惮。
<br/>　　震旦魔尊，在九魔城弟子之中，单凭实力，只在那锁魂魔尊之下，论起力量来说，甚至还要超越锁魂魔尊。
<br/>　　鲜血魔尊单打独斗之下，跟这震旦魔尊交手，都是没有一成胜算。
<br/>　　“小子，你的阴谋，在小魔主的面前，根本便是如同孩儿般的玩耍罢了，现在你主动来到这里，正好是落入到了我们的手中。”
<br/>　　又一尊魔尊阴冷的开口道。
<br/>　　“杀了此人，也是为了小魔主做事，日后等小魔主继位，我们都是功臣。”
<br/>　　“还废话那么多干嘛，赶紧杀了此人，将他的头颅带给小魔主，还有他手中的祖龙至尊剑，这等宝物，也只有小魔主才配拥有！”
<br/>　　又是一尊魔尊，充满杀意的开口道。
<br/>　　“一群小魔头，如今倒是全部聚集在一起来了，也省的我在一个个找了。”
<br/>　　楚风眠面对这眼前无数魔尊，脸色依然不变，嘴角甚至都是露出了一抹笑意，平静的开口道。
<br/>　　“将你们都杀了，在这天羽神域中，应该也就没有其他九魔城弟子了吧。”
<br/>　　“将我们都杀了？小子看来你还是在做梦吧！死到临头还在嘴硬，我今天就先撕了你的嘴，看看你的嘴有没有那么硬！”
<br/>　　震旦魔尊冷喝一声，突然出手，他的灵器，乃是一对巨斧，双斧舞动，顿时冲着楚风眠斩杀而来。
<br/>　　“剑！”
<br/>　　楚风眠轻喝一声，祖龙至尊剑，突然出现在他的手中，剑光一闪，这一抹剑锋，就跟震旦魔尊的双斧，猛然碰撞。
<br/>　　“咔嚓！”
<br/>　　就在这碰撞的一瞬间，震旦魔尊的双斧，便是突然开裂，随后破碎，一招交手，他的灵器，都在楚风眠的剑下，被斩成碎片。
<br/>　　“好锋利的剑！”
<br/>　　“震旦魔尊的双斧，虽然并非是天级灵器，但是也是远古时代的地级极品灵器，在这小子的剑下，居然直接破碎！”
<br/>　　一位位魔尊，看到这样一幕，都是震惊无比，再度看向楚风眠手中灵剑一刻，皆是一脸贪婪。
<br/>　　“天级灵器！”
<br/>　　“祖龙至尊剑！”
<br/>　　能够做到这一切的，楚风眠手中的这剑，起码也是天级灵器，甚至只能是天级灵器中的佼佼者，唯有龙族至宝，祖龙至尊剑才能做到。
<br/>　　“祖龙至尊剑，果然在此子手中！”
<br/>　　“上，杀了他！将这祖龙至尊剑献给小魔主！”
<br/>　　那些魔尊，一个个都是显得极为兴奋，纷纷催动灵力，直接出手，这祖龙至尊剑，他们无法得到，只能是献给小魔主。
<br/>　　但是献给小魔主也同样是大功一件，小魔主的身份，便是代表着他是成为下一任九魔城的魔主。
<br/>　　现在讨好与他，将来有着无数的富贵。
<br/>　　杀了楚风眠，正好是他们最好建功的时候。
<br/>　　十八魔尊，几乎是刹那之间，同时出手，一时间整个宫殿之中，都是魔气翻腾，如同一方魔域一般。
<br/>　　这些魔尊，放在九域去，哪一位都是响当当的人物，其中大部分的人的实力，都丝毫不比剑苍穹要弱。
<br/>　　这样十八个人，同时出手，简直是惊世骇俗，让超越生死的强者前来，都要有些头皮发麻。
<br/>　　但是楚风眠，却是站在原地，脸色不变，沉稳到了极点。
<br/>　　在这十八魔尊，同时出手的一刻，楚风眠的身形也是动了，他手中长剑，突然舞动，顿时宫殿之中，便是漫天剑影显现。
<br/>　　楚风眠的剑道，彻底的展现出来。
<br/>　　这漫天剑意，比起一剑万重的境界，都要高得多，只怕是许多超越生死的人物，如今都不一定可以领悟的出来。
<br/>　　可是现在，那五十道无上剑气，令他的九域剑术如今都开始变的升华。
<br/>　　在楚风眠的身边，五十道无上剑气，全部都围绕在了楚风眠的身边，令楚风眠身上的剑意，如今变的更加锋利起来。
<br/>　　楚风眠出手，这漫天剑影，顿时便是将无数袭来的攻击，全部抵挡，甚至是将周围的魔气，都给斩灭。
<br/>　　令周围，仿佛是陷入到了剑之世界之中。
<br/>　　“这是什么宝物？无上剑气？这种宝物，不是只有那天剑宗的剑祖荒才拥有吗？怎么这小子也有！”
<br/>　　看到楚风眠身边的凌冽剑气，这种凌冽的剑锋，令不少魔尊一眼便是认了出来，在楚风眠身边的，正是无上剑气。
<br/>　　足足五十道无上剑气，在楚风眠的身边，已经是组成剑阵，令楚风眠的实力如今都在节节攀升。
<br/>　　“好宝物！无上剑气，乃是曾经远古上剑宗的镇宗之宝，这足足五十道无上剑气，如果我能得到，我的剑术将会直接大成！”
<br/>　　一尊魔尊，激动的喊道，这魔尊便是白裁魔尊，也是一尊剑修强者，以魔如剑，开创一番剑术。
<br/>　　这无上剑气，对他来说，简直是比起世间的一切珍宝，都要珍贵的多。
<br/>　　“魔剑戮天！”
<br/>　　白裁魔尊猛然出手，他一出手便是绝对的杀招，只见他拔剑一刻，便是一股刺鼻的血腥味出现。
<br/>　　凌空之中，一道剑光突然斩杀过来，这一道剑光，呈现出血红色，凝聚出了无上杀意。
<br/>　　这一道剑气，笼罩着楚风眠，似乎是要将楚风眠，直接斩杀。
<br/>　　“以魔入剑？有些门道。”
<br/>　　看到这白裁魔尊出手，楚风眠冷哼一声，直接持剑迎了上去，这一交手，楚风眠便是将这白裁魔尊的剑术，全部看在眼中，融入到了九域剑术之中去。
<br/>　　“不过在我面前动剑，你这是在自寻死路！”
<br/>　　剑术以得，这白裁魔尊对楚风眠来说，便是没有任何意义，他现在的唯一意义，就是死。
<br/>　　“嘶！”
<br/>　　楚风眠手中祖龙至尊剑，剑光闪动，一抹剑锋，简直快的惊人。
<br/>　　这剑锋出现一刻，更是一股圣光，爆发出来，将那剑气直接驱散，同时笼罩着白裁魔尊的身躯。
<br/>　　在下一刻，这白裁魔尊的身躯，便是就在楚风眠的剑气之下，被直接撕碎。
<br/>　　一剑之下，屠杀魔尊。
<br/>　　这白裁魔尊，生死境巅峰的人物，在楚风眠的一剑之下，都被直接斩杀。
